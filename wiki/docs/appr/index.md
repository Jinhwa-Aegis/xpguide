# Xp전자결재

> **메뉴 경로**: Xp전자결재

## 개요
Xp전자결재는 관리사무소의 각종 문서를 전자적으로 결재하는 시스템입니다. 기안, 승인, 반려 등의 결재 프로세스를 온라인으로 처리할 수 있습니다.

## 주요 기능
- 결재 문서 작성 (기안)
- 결재선 설정 및 승인/반려
- 결재 문서 조회 및 검색
- 결재 현황 모니터링

## 화면 구성
> 스크린샷 영역 (추후 추가 예정)

## 사용 방법
1. Xp전자결재 메뉴 진입
2. 새 문서 작성 또는 기존 문서 조회
3. 결재선 지정 후 상신
4. 결재자가 승인/반려 처리

## 관련 메뉴
- [Xp문서함](/docs-menu/) - 결재 완료 문서 보관

## 주의사항
- 결재선은 사전에 설정된 기준을 따릅니다
- 상신 후에는 수정이 제한됩니다
