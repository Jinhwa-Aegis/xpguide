# 단지관리

> **메뉴 경로**: 단지관리

## 개요
단지관리는 아파트 단지의 기본 정보, 환경설정, 조견표 등을 관리하는 메뉴입니다. 시스템 운영의 기초가 되는 설정을 관리합니다.

## 하위 메뉴

### [단지정보](/dang/dang-info)
| 소메뉴 | 설명 |
|--------|------|
| 단지정보등록 | 단지 기본 정보 등록/수정 |
| 동호생성 | 동/호수 구성 생성 |
| 공동동호관리 | 공동 동호 관리 |
| 입주자단체등록 | 입주자대표회의 등 단체 등록 |
| 세대검침환경등록 | 세대별 검침 환경 설정 |
| 표준업무등록 | 표준 업무 코드 등록 |
| 결재라인등록 | 전자결재 결재선 등록 |
| 연말정산기본정보등록 | 연말정산 기초 정보 |
| 특이사항 | 세대별 특이사항 기록 |
| 과거데이터요청 | 이전 시스템 데이터 요청 |
| 모바일사용자 | 모바일 사용자 관리 |
| 소방시설세대점검 | 세대별 소방시설 점검 |
| 대피계획 수립 | 대피 계획 수립 |
| 사용자승인등록 | 시스템 사용자 승인 |
