# 환경설정

> **메뉴 경로**: 단지관리 > 환경설정

## 개요
XpERP 시스템의 운영 환경을 설정합니다. 검침, 부과, 출력 등 각 업무 모듈별 기본값과 처리 방식을 설정할 수 있습니다.

## 소메뉴 상세

### 검침환경등록
![검침환경등록 화면](/screenshots/DANG0050.png)

각 검침항목의 자리수 및 조견표 등의 기본 설정을 하는 메뉴입니다.

**전기 검침환경 설정 주요 항목**
- 원하는 검침항목(예: 전기)을 클릭하여 전기검침 환경 설정을 확인합니다
- **조건표 설정이 필요한 경우**: 계약구분의 종합단지 단일 설정, 연약의율이 있는 경우 고지서에 설정합니다
- **검침 자리수 및 업무 순서 비고지기준을 설정합니다**:
  - 검침 선택으로 조회가 되면 일 검침 집결량을 확인할 수 있습니다
  - 동 단위 데이터를 관리하며 사용하지 않는 부분은 해제합니다
  - 자료작성은 선택하거나 구분별로 조회합니다

**수도 검침환경 설정 주요 항목**
- 수도등점은 매월(기 2개월) 반환경을 설정 및 등록합니다
- 수도환경 구성: 수도 + 온수 + 오수 설정 경우 사용됩니다
  - 수도 설정 시 단수적용물을 선택합니다
  - 오수 설정 시 오수적용물을 선택합니다
- 항목 업무 분류를 선택할 수 있으며, 지역할인은 지역서비스가 됩니다

### 관리비부과환경등록
![관리비부과환경등록 화면](/screenshots/DANG0070.png)

관리비 부과에 관련한 기본 환경설정을 하는 메뉴입니다.

**부과 각입금 설정**
- 관리비의 기본 마감일을 설정합니다
- **가수금처리방법**: 기수금 분할공제액(1기 지급관리) 금액을 대상일 기수금원에서 가수금 납부처리합니다
  - 가수금납부액수: 기본 가수금 대상 납부액을 설정합니다
  - 미납 가수금 대상 납부 형태를 지정합니다
  - 미납·가수금이면 반전관리에서 납부 가수금처리를 할 수 있습니다
  - 납부된 가수금처리 방법도 설정합니다

**카드 및 PG 처리 설정**
- 카드 및 PG처리에 대한 수납형태를 관리소통장 입금처리 합니다
  - 자동이체인경우: 세대가 선택한 카드 및 삼사카드 수납처리
  - 정신운정관리소 공과통합을 수납처리
  - 자동이체 정신정보 발행번수를 수납처리

## 사용 방법

### 검침환경 설정 순서
1. 단지관리 > 환경설정 > 검침환경등록 메뉴 진입
2. 상단에서 검침항목(전기, 수도 등)을 선택하여 해당 항목 환경 확인
3. 조견표 설정이 필요한 경우 계약구분, 연약의율 등을 설정합니다
4. 검침 자리수, 업무 순서, 고지기준 등을 설정하고 저장합니다

### 관리비부과환경 설정 순서
1. 단지관리 > 환경설정 > 관리비부과환경등록 메뉴 진입
2. 부과 마감일, 가수금처리방법, 납부 형태 등 항목별 설정값 확인
3. 카드/PG 처리 방식, 자동이체 설정 등 수납 환경 입력
4. 저장 후 해당 월 부과 작업 전에 반드시 설정 완료 확인

## 주의사항
- 환경설정 변경은 전체 시스템에 영향을 미칠 수 있습니다
- 변경 전 현재 설정값을 기록해 두는 것을 권장합니다
- 부과/검침 환경은 해당 월 업무 시작 전에 설정을 완료해야 합니다
- 검침환경에서 자리수 설정이 잘못된 경우 검침 데이터 오류가 발생할 수 있습니다
- 수도 환경에서 수도, 온수, 오수 구분 설정은 실제 단지 계량기 구성에 맞게 설정해야 합니다
- 가수금처리방법 변경 시 기존 처리된 내역과 불일치가 생길 수 있으므로 주의합니다
