# 단지정보

> **메뉴 경로**: 단지관리 > 단지정보

## 개요
단지의 기본 정보를 등록하고 관리합니다. 단지명, 주소, 동/호수 구성, 면적 정보 등 시스템 운영의 기초 데이터를 설정합니다. 단지정보는 모든 업무의 기초가 되므로 최초 설정 시 정확하게 입력해야 합니다.

## 소메뉴 상세

### 단지정보등록
![단지정보등록 화면](/screenshots/DANG0010.png)
- 단지(아파트)와 프로그램 고유번호인 코드, 단지의 기본 정보를 등록·확인할 수 있습니다
- 단지코드는 수정 불가하므로, 단지를 재입력할 경우 관련 정보 일체를 다시 등록해야 합니다
- **관리소 정보**: 관리소의 사업자등록 및 고용주 증증 정보를 입력하고 수정할 수 있습니다
- **수납 처리 수수료 우선 순위**: 기본으로 설정할 수 있으며, 다음 항목을 포함합니다
  - 부과/납부/입주자 정보
  - 도, 로, 토, 관계기관의 자료를 바탕으로 활용
  - 관리비 고지서 관련 기본 설정

### 동호생성
![동호생성 화면](/screenshots/DANG0030.png)
- 동호 정보를 확인 및 추가, 사재, 면적 등 정보를 수정할 수 있습니다
- **단지 보기 버튼**: 클릭하면 단지 내에 포함된 전체 동을 확인할 수 있습니다
- **행추가 버튼**: 클릭하면 새 동 정보를 추가합니다
- **동별 면적합 수정**: 수정 후 저장하면 변경 사항이 반영됩니다
- 현재 입주(전입)된 세대가 있는 경우, 전출처리를 완료한 후 동을 삭제할 수 있습니다
- **동라인 생성 탭**: 동 순서에 맞게 승강기 라인 및 별도의 순서를 지정하여 인쇄에 반영할 수 있습니다
  - 동라인생성 버튼을 클릭하여 동 내 라인 내용을 확인합니다
  - 행추가 버튼을 클릭하면 라인별 동수, 시작호, 종료호, 층수 등의 정보를 등록할 수 있습니다
  - 세대학용요금이 사용되는 경우 해당 라인 정보도 함께 설정합니다

### 공동동호관리
![공동동호관리 화면](/screenshots/DANG0036.png)
- 일반적 사용 동호 외에 임의의 동호를 등록하는 경우 사용합니다
- 조회 후 신규 버튼을 클릭하여 추가할 동호를 등록합니다
- 고지서에 출력할 동 항목을 필요에 따라 포함하거나 제외할 수 있습니다
- 일반 동호를 공동 동호로 변경하면 세대(입주자)별 정보를 공동으로 변경 처리하여 개별 사용이 불가합니다

### 입주자단체등록
![입주자단체등록 화면](/screenshots/DANG0020.png)
- 아파트 내 입주자대표회의, 선거관리위원회 등의 단체를 등록하여 관리하는 메뉴입니다
- 입주자단체등록 버튼을 클릭하여 아파트 내 단체를 모두 등록합니다
- 단체등록 후 이메일 등 정보를 모두 등록하면 해당 단체에 공지/발송이 가능합니다
- **아파트아이 앱 연동**:
  - 아파트아이 앱에서 입주자대표가 대표자 등록 요청 시 XpERP에서 단체등록을 처리할 수 있습니다
  - 기존 단체등록으로 빠른 메뉴도 연동됩니다
  - 앱에서 직책/기간을 선택 → 시작일 선택·자동기간 설정 → 임기기간 기입 → 등록 요청 완료 순으로 진행합니다
- **아파트아이 연동 단체등록 처리 절차**:
  1. 아파트아이 앱을 통해 대표 등록 요청이 접수되면, XpERP에서 확인 버튼을 클릭하여 입주자 단체등록 메뉴로 이동합니다
  2. **단체 탭**: 입주자단체(회의) API가 자동으로 생성됩니다
  3. **전체 입주자 탭**: 직책과 성명 등록 화면에서 조건을 수정한 후 승인 처리합니다
     - 승인처리: 입주자 대표 승인 후 요청 상태가 변경됩니다
     - 승인 후 XpERP 전자기안 연계 처리에 전산화됩니다
  4. **임기**: 임기기간(시작~종료)을 설정하면 임기기간에 따라 관련 대표자가 조합됩니다
- **가입요청 발송**:
  1. 가입요청 발송 버튼을 클릭합니다
  2. 행추가 버튼을 클릭하여 대표자 목록을 추가합니다
  3. 전송 후 체크표 발송 버튼을 클릭하여 대표자 연락처로 가입요청을 발송합니다
- **일정관련 발송**:
  1. 발령 버튼을 클릭합니다
  2. 행추가 버튼으로 대표자 선택, 날짜, 시간, 장소 등을 입력합니다
  3. 발령 미리 보기를 클릭하여 발령 내용을 날짜별로 확인합니다
  4. 발령 공지 항목 재검증 버튼을 클릭하여 발송을 처리합니다

### 세대검침환경등록
![세대검침환경등록 화면](/screenshots/DANG0035.png)
- 검침항목의 입력 여부 및 조건표 등을 세대별로 설정할 수 있습니다
- 세대별로 변환할 수 있으며, 항목에 내용이 없는 세대는 각 검침항목에서 조회하여 누락을 확인합니다
- 검침항목에 내용이 없는 세대는 검침 항목 연대에 나타나지 않습니다
- Y 표시가 있으면 해당 검침항목을 사용하는 것이며, V 표시가 없으면 검침항목 A로 표기됩니다. B를 기입하면 해당 검침 설정이 적용됩니다
- **하위 스크롤**을 이용하면 각 검침항목 계량기 자리수를 변경할 수 있습니다
  - 검침환경의 동환경에 기본 자리수로 설정됩니다
  - 자리수가 없으면 새로운 검침기 자리수를 직접 등록합니다

### 결재라인등록
![결재라인등록 화면](/screenshots/DANG0051.png)
- XpERP 인쇄물에서 결재판을 지정하여 인쇄할 수 있습니다
- 장표종류를 선택 후 해당 양식에 맞는 결재라인을 등록·저장하여 사용합니다
- 결재라인 유형(드롭다운 선택): 관리전문, 공지관리, 발전건설, 입수화재, 발전가구, 입출자동, 입주시설사무 등 업무 유형별로 구분됩니다

### 연말정산기본정보등록
![연말정산기본정보등록 화면](/screenshots/DANG0170.png)
- 급여 및 부가세 사용단지의 급여 관련 공제 정보를 등록하는 메뉴입니다
- 정산내용을 등록 후 조회하여 급여관련 공과정보를 확인하고 이동 처리합니다
- 연말정산 시 근로소득 세무처리방법에 관련한 사항을 등록하고 처리합니다
- 해당되는 근로세금 정보를 정확하게 기입해야 합니다
- **전년도 정보 활용**: 전년도에 입력된 정보를 참조하여 해당 연도 세사항에 변경·반영할 수 있습니다

### 모바일 발송
![모바일 발송 화면](/screenshots/TEST0009.png)

관리비 고지서를 카카오 또는 문자 발송할 수 있습니다.

**카카오톡 발송 (단건발송)**
- 관리비고지서를 카카오로 또는 문자 발송할 수 있습니다
- 고지서 발송 대상 세대를 선택한 후 지정된 카카오로 발송합니다
- 1회 발송 제한이 있으므로 발송 전에 입력 항목을 반드시 확인합니다
- **미리보기 버튼**: 발송 전 카카오 발송 내용을 확인할 수 있습니다
- 발송 후 취소가 어려울 수 있으므로 발송 전 내용을 꼼꼼히 확인합니다

**SMS 문자 발송**
- SMS(문자) 발송 방법은 카카오 발송 방법과 동일합니다
- SMS(문자) 발송 선택 후 발송 시 문자 내용을 추가하여 발송이 가능합니다
- 문자 발송을 위해 사용 서비스에 신청 후 추가 서비스가 활성화됩니다
- **유선번호 인증 요청** 절차 (SMS 발신번호 사용 전 필수 처리):
  1. 관리소에서 사용하는 전화번호를 가입한 통신업체에 통신 가입 증명서 발급 요청
  2. XpERP 게시판 또는 모바일발송 페이지에서 발신번호 대리인 신청서를 작성 및 다운받기
  3. 발신번호 대리인 신청서, 통신 가입 증명서, 신청 시 사용한 인감증명서 또는 신분증 사본을 전산업체에 발송
  4. 전산업체 접수 후 발신번호 등록 완료 후 사용

**고지서 인쇄**
- 모바일 발송 화면에서 고지인쇄를 선택하면 고지서를 인쇄할 수 있습니다
- 인쇄할 동을 조회한 후 조회기간 내 인쇄 기준에 따라 인쇄합니다
- 입대의 대부 이후 복합 항목별 금액을 합쳐 인쇄할 수 있습니다

**시점발인세**
- 부과고지서 및 현재시점 정보를 사용하여 부과고지 작업 선행 후 시점 기준으로 고지서를 발인쇄할 수 있습니다
- 출력이아(아감시)는 업데이트 이후 부과고지작업 결과를 기준으로 목록을 조회하여 인쇄합니다

**미납독촉장**
- 연재개월을 변환하면 조회 항목을 제어할 수 있습니다
- 독촉장 발송 세대를 선택 후 선택된 세대의 발송 정보를 확인합니다
- 연체료가 설정된 경우 연체료로 처리됩니다
- 이런보기 화면의 발송 내용이 수정될 경우 내용을 확인한 후 업데이트합니다

**고지서 발령목록 (대량)**
- 관리비고지서를 카카오톡을 통해 일괄 발송할 수 있습니다
- 전송 체크의 조대로번호가 등록된 세대에는 체크하면 발송이 가능하며, 체대발번호는 입주등록 탭에서 조회합니다

**발송이력**
- 고지서 구분 및 일자등록 입력 후 조회가 가능합니다
- 조회 내역 중 필요한 정보를 선택하면 각 필드 내용을 확인합니다

### 소방시설세대점검
- 소방시설법 개정에 따라 공동주택 입주자는 2년마다 1회 이상 세대 내 설치된 소방시설을 직접 점검해야 합니다
- 관리사무소 업무를 지원하기 위해 입주자가 직접 아파트아이 앱을 통해 소방 세대점검을 실시할 수 있도록 XpERP 기능이 업데이트되었습니다
- **아파트아이 앱**: 입주자가 앱을 통해 소방 세대점검을 직접 실시합니다
- **XpERP 시스템**: 관리사무소는 XpERP 시스템을 통해 우리아파트 소방세대 점검현황(점검완료/미점검)을 확인하고 관리할 수 있습니다
- **세대별현황보기 탭**: XpERP 소방 세대점검 페이지에서 전체 점검 현황을 조회합니다
  - 소방 세대점검 메뉴는 관리사무소와 입주민 영역으로 구분됩니다
  - 세대별전형보기를 클릭하면 전체 소방 세대점검 현황 페이지로 이동합니다

## 사용 방법

### 단지정보 최초 설정 순서
1. 단지정보등록에서 단지 기본 정보(단지명, 주소, 관리사무소 정보 등) 입력
2. 동호생성에서 동별 라인 및 호수 구조 생성
3. 공동동호관리에서 필요한 경우 공동 사용 동호 추가 등록
4. 세대검침환경등록에서 세대별 검침항목 설정
5. 결재라인등록에서 인쇄물별 결재선 설정
6. 연말정산기본정보등록에서 급여 관련 공제 정보 등록 (해당 단지)
7. 모바일발송 설정에서 발신번호 인증 후 카카오/SMS 발송 환경 구성
8. 입주자단체등록에서 입주자대표회의 및 관련 단체 등록

## 주의사항
- 단지코드는 최초 등록 후 수정이 불가하므로 신중하게 입력합니다
- 동/호수 변경은 입주자 정보와 연관되므로 현재 입주 세대가 있는 경우 전출처리 후에만 삭제할 수 있습니다
- 세대검침환경등록에서 검침항목 누락 시 해당 세대에 검침 항목이 표시되지 않습니다
- 모바일 발송을 위한 유선번호 인증은 사전에 통신업체를 통해 증명서를 발급받아야 합니다
- 연말정산기본정보는 해당 월 업무 시작 전에 등록을 완료해야 합니다
- 단지 기본 정보 변경 시 관련 업무 전반에 영향을 줄 수 있으므로 관련 부서와 협의 후 진행하세요
