# 조견표

> **메뉴 경로**: 단지관리 > 조견표

## 개요
관리비 부과에 사용되는 각종 조견표(기준표)를 관리합니다. 검침 단가, 부과 기준 등을 설정합니다.

## 소메뉴 상세

### 조견표등록
![조견표등록 화면](/screenshots/DANG0150.png)

검침 조건을 설정하는 메뉴입니다.

- **조견표에서 조건을 등록하고 저장하면 검침항목을 설정**할 수 있습니다
- **수도조견표 등록 시 유의사항**:
  - 사용요금 중 구간이 연리기를 설정하면 다 구간에 넘어가는 기본단가의 변화를 확인하시기 바랍니다
  - 사용요금 금액은 구간별 단계별로 누적 적용됩니다
  - 사용요금 금액은 [사용요금 중 구간 연리기를 정렬] 선택에서 이 금액 대상이 되며, 각 사용 단계 초과분에 대해 이전 금액 대비 변환이 되는지 확인하시기 바랍니다
- **조견표 계산 예시**:
  - 사용도 1~10일 때 사용요금: 100원
  - 11~30일 때 사용요금: 390원
  - 사용량 12인 경우: 12 = (10×100) + (30×2) = 1,780원
- 점검이 이미 등록된 경우 조견표 등록 및 변경 시 공제예산에 따라 계산이 재반영됩니다

### 주차조견표등록

주차비 조건을 등록하는 메뉴입니다.

- **등록필부과·자차등록필 부과**: 부과 책정 및 해제 시 필수 설정 항목입니다
  - 해지필부과: 자차등록 해제 시 부과를 취하거나 폐지할 경우 발생 처리
  - 부과재외필수: 차량이 등록된 경우 등록 차량에 부과하며, 일수기산제 없을 경우 등록 차량에 부과합니다
- **동별 입주 주차 기준으로 입주자를 등록합니다**:
  - 주차필 부과 경우: 세대(입주자)별로 차량이 등록되어 처리됩니다
  - 차량대수 형태는 차량 등록 수와 차량대수 합산 시 처리됩니다
- **부과제외필수 설정**: 주차 기준으로 차량이 등록된 경우 일수기산제 제외 설정 적용 가능합니다
- **동별 입주 등록 세대의 경우**: 주차 기준으로 등록된 세대 및 차량을 조회하여 부과 여부를 결정합니다

## 사용 방법

### 조견표 등록 순서
1. 단지관리 > 조견표 > 조견표등록 메뉴 진입
2. 상단에서 조견표 항목(수도, 전기, 가스 등) 선택
3. 구간별 사용량 범위와 단가를 입력합니다
4. 저장 후 검침환경등록에서 해당 조견표가 적용되는지 확인합니다

### 주차조견표 등록 순서
1. 단지관리 > 조견표 > 주차조견표등록 메뉴 진입
2. 등록필부과/해제필부과/부과재외필수 여부 설정
3. 동별 주차 기준으로 입주자 차량 정보 등록
4. 차량대수 기준(1대, 2대 이상 등) 요금 구간 설정 후 저장

## 주의사항
- 단가 변경 시 부과 금액에 직접 영향을 미칩니다
- 조견표 구간 설정 시 누적 계산 방식인지 구간별 개별 계산 방식인지 반드시 확인합니다
- 수도 조견표에서 구간 경계의 단가 변화(연리기 설정)를 반드시 검토합니다
- 이미 부과 처리된 데이터가 있는 경우 조견표 변경 시 공제예산이 재계산되므로 주의합니다
- 단가 변경은 입주자대표회의 의결이 필요할 수 있습니다
- 변경 이력을 반드시 기록하고 확인하세요
