# 정수검침

> **메뉴 경로**: 검침 > 정수검침

## 개요

세대별 정수기 사용량을 검침합니다. 단지에 정수 시설이 있는 경우 사용하며, 정수 단가를 등록하여 요금을 계산할 수 있습니다.

> 공통 검침 기능(동호찾기, 전월지침수신, 메모등록, 파일붙여넣기 등)은 [공통검침](/insp/common) 문서를 참조하세요.

---

## 소메뉴 상세

### 정수검침

![정수검침 화면](/screenshots/INSP4110.png)

**메뉴 경로**: 검침 > 정수검침 > 정수검침

매달 검침일에 세대 정수 계량기를 확인하여 입력 수치를 입력하고, 정수요금을 계산하는 화면입니다.

#### 화면 구성

검침 화면은 전기·수도검침과 동일한 레이아웃을 사용합니다.

**조회 영역 주요 필드:**
- **년월 필드**: 검침할 년월을 입력합니다.
- **동 선택**: 검침할 동을 선택합니다.
- **조회 버튼**: 선택한 조건으로 세대 목록을 불러옵니다.

**주요 컬럼:**
- **전월지침**: 전월에 검침한 지침 값
- **당월지침**: 금월 검침한 지침 값 (직접 입력)
- **사용량**: 당월지침 - 전월지침으로 자동 계산
- **요금**: 등록된 단가 기준으로 계산된 정수요금

**주요 버튼:**
- **전월지침**: 전월 지침값을 재수신합니다.
- **요금계산**: 정수요금을 계산합니다.
- **저장**: 검침 데이터를 저장합니다.
- **인쇄**: 검침 내역을 인쇄합니다.

#### 정수검침 입력 절차

1. **년월 입력** 후 조회 버튼을 클릭합니다.
2. **전월지침 수신**: 전월지침 버튼을 클릭하여 전월 데이터를 수신합니다.
3. **동 선택**: 검침할 동을 선택합니다.
4. **지침값 입력**: 각 세대의 금월 정수 지침값을 입력합니다.
5. **에러 확인**: 사용량이 과다하거나 이상이 있는 세대(에러 표시)를 확인하고 재검침합니다.
6. **요금계산**: 요금계산 버튼을 클릭하여 정수요금을 산출합니다.
7. **저장**: 저장 버튼을 클릭합니다.

### 정수사용량조회

![정수사용량조회 화면](/screenshots/INSP4210.png)

**메뉴 경로**: 검침 > 정수검침 > 정수사용량조회

정수 사용량을 상세 조회합니다.

**주요 기능:**
- 기간 및 동호를 선택하여 정수 사용량을 조회합니다.
- 전기, 수도, 정수 등 검침 유형별로 사용량을 비교하여 확인할 수 있습니다.
- 이상 사용 세대를 라인으로 선택하여 확인합니다.
- 조회 결과를 인쇄할 수 있습니다.

### 정수세대조회

![정수세대조회 화면](/screenshots/INSP4220.png)

**메뉴 경로**: 검침 > 정수검침 > 정수세대조회

세대별 정수 사용량 및 요금을 상세 조회합니다.

**주요 기능:**
- 기간 및 동호를 입력하여 특정 세대의 정수 이력을 조회합니다.
- 단지/수신단말에 관련된 정수 검침 데이터를 선택하여 조회합니다.
- 전월 대비 사용량 변화를 확인할 수 있습니다.

### 정수동별사용량조회

![정수동별사용량조회 화면](/screenshots/INSP4230.png)

**메뉴 경로**: 검침 > 정수검침 > 정수동별사용량조회

동별 정수 사용량 집계를 조회합니다.

**주요 기능:**
- 년월과 동을 입력하여 동별 정수 사용량을 조회합니다.
- 전체 동의 사용량 집계를 한눈에 확인합니다.
- 조회 결과를 인쇄하거나 데이터를 내보낼 수 있습니다.

## 사용 방법 (종합)

1. **정수검침 메뉴 진입**: 검침 > 정수검침 > 정수검침
2. **검침 년월 선택**: 상단 년월 필드에 해당 월 입력
3. **전월지침 수신**: 전월지침 버튼으로 전월 데이터 수신
4. **동 선택 및 검침값 입력**: 동 선택 후 각 세대의 금월 정수 지침값 입력
5. **에러 확인**: 이상 사용량(에러 표시 세대) 재확인
6. **요금계산**: 요금계산 버튼으로 정수요금 산출
7. **저장**: 저장 버튼 클릭
8. **결과 확인**: 정수사용량조회 메뉴에서 확인

## 주의사항

- 정수 설비가 없는 단지에서는 정수검침 메뉴를 사용하지 않습니다
- 정수 단가 변경 시 환경설정에서 단가를 먼저 수정 후 요금계산을 진행하세요
- 사용량이 0인 세대가 다수 발생하면 정수 공급 상태를 점검하세요
