# 전기검침

> **메뉴 경로**: 검침 > 전기검침

## 개요

세대별 전기 계량기의 수치를 입력하여 전기 사용량 및 요금을 산정합니다. 전기검침은 한전 수신 데이터와 연동되며, TV수신료, 복지할인, 감면 등의 요금 요소도 함께 처리합니다.

> 공통 검침 기능(동호찾기, 전월지침수신, 메모등록, 파일붙여넣기 등)은 [공통검침](/insp/common) 문서를 참조하세요.

---

## 소메뉴 상세

### 전기검침

![전기검침 화면](/screenshots/INSP1120.png)

**메뉴 경로**: 검침 > 전기검침 > 전기검침

매달 검침일에 세대 계량기를 확인하여 전기 지침값을 입력하고 요금을 계산하는 화면입니다.

#### 전기검침 화면 설명 (p20~p22)

**상단 조회 영역:**
- **입력 필드**: 년월 및 동호를 입력 후 조회 버튼을 클릭합니다.
- **한전서버 자동수신**: 전자검침기기가 있는 경우 한전 서버에서 데이터를 자동으로 수신합니다.
- **조회 화면 조회**: 데이터가 없는 경우 데이터를 수집 후 수신 상태를 확인합니다.

**주요 컬럼:**
- **전월지침**: 전월 검침 지침값
- **당월지침**: 금월 검침 지침값 (직접 입력)
- **사용량**: 당월지침 - 전월지침으로 자동 계산
- **전기요금**: 계산된 전기요금
- **TV수신료**: TV수신료 (면제 세대 제외)
- **복지구분**: 복지할인 구분 (기초생활, 국가유공, 복지시설 등)

**추가 열람 정보 (p21):**
- **한전자동 출력**: 한전 서버 전기요금 데이터를 조회합니다.
- **조회 버튼에 높였을 때**: 선택 여부에 따라 해당 데이터를 전체 또는 부분 배포합니다.
- **데이터비어 옵션**: 데이터가 없는 세대에 대한 처리 방법을 설정합니다.
- **Range 버튼**: 조회 범위를 지정합니다 (라인 단위 조회).
- **Type 버튼**: 가스, 라인순, 동라인순 등 현재의 라인으로 선택하여 화면에 항목들을 조회합니다.

**화면 조회 항목 (p22):**
- **데이터비어 옵션**: 입력 조회 시 조회 화면에서 해당 배포 화면 구성의 데이터를 선택합니다.
- **조정기, 출기, 단지기, 단지기, 방전기구**: 동을 배포하여 조회 가능합니다.
- **복지구분**: 비복지설정 단 세대의 경우 복지구분을 선택합니다.
  - TV가 없는 세대, 비과세, 방전기기수, 방전기기구 등이 포함됩니다.
  - 비과세/비방전기는 경우에 따라 복지구분을 선택합니다.
- **복지기구분**: TV가 없는 세대인 경우 대해 수동으로 복지구분을 선택합니다.

---

#### 전기검침 입력하기 (p23)

**검침값 입력 절차:**

1. **년월 입력** 후 조회 버튼을 클릭합니다.
2. **전월지침 수신**: 전월지침 버튼을 클릭하여 전월 데이터를 수신합니다.
3. **동 선택**: 전월지침이 없는 경우, 전월지침 수신 후 검침 동을 선택합니다.
4. **당월지침값 입력**: 각 세대의 금월 전기 지침값을 입력합니다. 입력 후 사용량이 자동 계산됩니다.
5. **검침값 확인**: 검침값 입력 완료 후 사용량이 누가적으로 아이어이입박(Upload) 처리 됩니다.
6. **요금 계산**: 달력일에 모든 사용량 입력 후 요금계산 버튼을 클릭합니다.
7. **전기 사용량 및 전기요금, TV수신료, 방안요금을** 확인 후 저장합니다.

---

#### 전기검침 요금계산 (p24)

전기요금을 산출합니다.

**요금계산 버튼 클릭:**
1. 검침 화면 상단 또는 하단의 **요금계산** 버튼을 클릭합니다.

**요금계산 팝업 창:**
- **세대(세대 유형)**: 세대 유형을 선택합니다.
- **요금계산 선택**:
  - **전기요금**: 전기 사용량 기반 전기요금을 계산합니다.
  - **TV수신료**: TV수신료를 계산합니다.
  - **방안(방위기준)**: 방안료를 계산합니다.
  - **사용량면제**: 사용량 면제 여부를 선택합니다.
  - **단지 포함**: 단지 포함 여부를 선택합니다.
- **요금계산** 버튼 클릭 후 계산됩니다.
- **단지세가**: 단지세가 및 요금을 확인하고 저장합니다.

**요금계산 방식:**
- **전기요금**: 등록한 단가 및 사용량으로 전기요금을 계산합니다.
- **TV수신료**: 방안과 TV수신료 등 TV수신을 관련하여 사용 단 종을 선택합니다.
- **방안**: 방안 단가기준으로 계산합니다.
- **사용량면제**: 단 유 면제를 선택합니다.
- **단지 포함**: 단지 포함 여부를 선택합니다.

> 한 달에 실제 데이터가 없는 요금으로 계산됩니다. 전기의 실 세 데이터 값이 없는 경우에는 단지 계산이 되지 않습니다.

---

#### 전기검침 일괄계산 (p25)

전기요금 및 사용량을 일괄적으로 처리합니다.

**일괄계산 방법:**

1. 상단의 **일괄계산** 버튼을 클릭합니다.
2. 사용량 및 사용금액을 일괄적으로 더하거나 (-%)는 사용량 및 사용금액을 변경합니다.
3. 사용량 조금 상 전의 단별별로 일 사용량을 관련 계산 방법으로 계산합니다.
4. 단지에 있어 별 단 요금이 적용하는 방식으로 요금 또는 %로 단위로 계산됩니다.
5. **동별 도 별도 검침** 기준 [SA-E]를 선택하여 단 동별 선택 후 금액값을 입력하여 더하거나 요금을 단계별로 계산합니다.

---

#### 전기검침 데이터 업로드 (p26)

직접 입력하지 않고 파일로 검침 데이터를 업로드합니다.

**업로드 방법:**

1. 업로드를 헥셀로 업로드하는 경우 사용합니다.
2. **데이터업로드** 팝업에서 항목에 맞게 확인합니다.
3. **업로드 하는 방법 (V)** 체크를 합니다.
4. 업로드 시 표준 형식의 엑셀 파일을 준비합니다.

**데이터 양식:**
- 동, 호, 전지됨, 달별지침, 사용량을 포함한 양식입니다.
- 예시: `동 1 | 호 301 | 전지됨 295 | 달별지침 486 | 사용량 191`
- 엑셀(xlsx) 파일로 다운로드하여 양식을 확인 후 업로드합니다.
- **주의**: 3,000 이상의 세대도 업로드 가능합니다.

### 전기계량기2검침

![전기계량기2검침 화면](/screenshots/INSP1160.png)

**메뉴 경로**: 검침 > 전기검침 > 전기계량기2검침

세대에 계량기가 2개 있는 경우, 계량기2로 지정하여 입력합니다.

#### 계량기2 검침 입력하기 (p27)

1. **계량기2지침 설치 여부**: 세대에 대해 전 계량기2로 지정하여 전기 계량기2를 등록합니다.
2. **계량기2지침한 전 전기검침 메뉴**: 전기계량기2 관련 목록이 표시됩니다.
3. **입력 후 저장**: 계량기2, 전기단지됩니다. 당달이 저장됩니다.

> 계량기2로 지정된 세대의 경우 해당 사용량은 전기검침 메뉴의 사용량에서 합산되어 전기사용량에서 반영됩니다. 또한 요금은 단계별로 계산되며, 전자/전기사용량 요금 단계가 적용됩니다.

#### 전기계량기2조회 (p28)

**메뉴 경로**: 검침 > 전기검침 > 전기계량기2조회

1. **년월과 동호** 및 동을 입력합니다.
2. **원하는 단 목록**을 확인합니다.
3. **동사자/시건/집합/시건장치**를 클릭하여 동호찾기를 통해 해당하는 [동/호]로 이동합니다.
4. **세대 사용량 조회**:
   - 1 세대 시: 계약전력기준으로 3개월별 전기사용량 연판별 단가 사용이 가능합니다.
   - 두 번째로 동 인원에 대한 3개월별 사용량은 연판별 단가 사용 시 비율단가 단가에서 사용 가능합니다.
5. **PAGE SKIP**: 처리 후 다음 시 다음 페이지를 조회합니다.

### 전기감면세대

![전기감면세대 화면](/screenshots/INSP1240.png)

**메뉴 경로**: 검침 > 전기검침 > 전기감면세대

TV면제, 복지할인, 국가유공, 기초생활 등 감면 대상 세대를 조회합니다.

#### 감면세대 조회 (p29~p32)

**주요 조회 화면:**
- **년월 및 동 입력**: 조회 년월과 동호를 입력합니다.
- **감면구분 선택**: 감면구분 탭에서 TV면제 또는 전기할인을 선택합니다.
- **동호 찾기**: 찾고자 하는 [동/호]를 입력하고 버튼을 누르면 [동/호]로 이동합니다.
- **인쇄**: 감면세대 목록을 인쇄할 수 있습니다.

**TV면제 구분 (p30):**

| 구분 | 설명 |
|------|------|
| 전체 | 전체 세대를 조회합니다 |
| 부과세대 | 수신료가 부과된 세대를 조회합니다 |
| 기초생활 | 기초생활수급자 세대를 조회합니다 |
| 국가유공 | 국가유공자 세대를 조회합니다 |
| 시각장애 | 시각장애인 세대를 조회합니다 |
| TV 없음 | TV가 없는 세대를 조회합니다 |
| 기타 | 기타 사유 세대를 조회합니다 |
| 청각장애 | 청각장애인 세대를 조회합니다 |
| 지체장애 | 지체장애인 세대를 조회합니다 |
| 복지시설 | 복지시설 세대를 조회합니다 |
| 수신료0원 | 수신료가 0원인 세대를 조회합니다 |
| TV수신등록날 | TV수신 등록일 기준 세대를 조회합니다 |
| 이중등록말소 | 이중등록 말소 세대를 조회합니다 |
| 사용량면제 | 전기 사용량 50kwh 미만인 수신료 0원 세대를 조회합니다 |
| 사용량면제제외 | 전기 사용량 50kwh 미만인 세대 제외 조회합니다 |

**전기할인 구분 (p31):**
- **전기 단계**: 전기검침 메뉴의 전기단계/복지설정 간에 설정된 세대의 단계를 조회합니다.
- **복지**: 복지 전기 검침 메뉴의 전기 단계에서 설정된 복지에 설정된 세대들을 조회합니다.
- **목적금·밸드**: 목적의 전기 검침 메뉴에서 전기단계/복지설정 단계에서 설정된 세대를 조회합니다.
- **공동사용동 0kwh 공지 및 알림**:
  1. 공동사용동 0kwh일 경우에 배고, 내배고 0kwh 이하의 경우에 배고 더 이상 사용 가능합니다.
  2. 공동사용동에서 배포 시 전기 배포하여 단 세대별로 사용량을 배포합니다.

**보장금 및 동달기 조회 (p32):**
- **보장금이 적용된 세대**: 보장금이 적용된 세대를 조회합니다.
- **동달기 적용된 세대**: 동달기가 적용된 세대를 조회합니다.
- **전체세대 조회**:
  1. 사용량 0kwh 세대와 보장금배 세대 조회가 가능합니다.
  2. 전체배로 이야기 조회 후 사용자에서 추가 조회가 가능합니다.

### 한전요금조회

![한전요금조회 화면](/screenshots/INSP1250.png)

**메뉴 경로**: 검침 > 전기검침 > 한전요금조회

전기검침 및 한전이 연계되어 있는 경우, 한전에서 수신 받은 데이터에 대해 조회 및 저장, 단 세대별 전기 확인을 할 수 있습니다.

**사용 방법:**

1. **검침월과 한전 년월** 및 동을 입력합니다.
2. **조회 버튼 클릭** 후 조회합니다.
3. **한전에서 제공하는 세대별 전기요금**을 확인합니다.
4. **한전 요금 대비 세대 요금을** 비교 확인합니다.
5. **요금기준에 맞는지** 금액을 확인합니다.

**한전요금 조회 상세:**
- **한전 금액**: 한전에서 산정한 세대별 전기요금을 조회합니다.
- **단지 금액**: 단지 자체적으로 계산한 전기요금과 비교합니다.
- **단 기준 내에** 있는지 금액이 맞는지 조회합니다.
- **동단자화되의 경우** 동 단지로 검침이 기준에서 수신이 가능합니다.

### 한전검침인쇄 (한전검침대장)

![한전검침인쇄 화면](/screenshots/INSP1270.png)

**메뉴 경로**: 검침 > 전기검침 > 한전검침인쇄

전기검침 완료 후 한전 제출용 검침 대장을 인쇄합니다.

**사용 방법:**

1. **검침월과 한전 년월** 및 동을 입력합니다.
2. **배리번호를 선택**합니다.
3. **출력 방식을 선택**합니다.
   - **한전 방법이 없는 경우**: A4로 선택하여 출력합니다.
   - **한전 방법이 있는 경우**: A4크기 선택하여 출력합니다.
4. **지침일을 입력**합니다.
5. **방식에 따라** 단지, 사업소, 시연소, 전자소, 수연이 분리기로 분리한 단지를 선택합니다.

**출력 시 참고:**
- **용지선택 시**: 한전 방법이 없을 경우, A4크기/단자 대신 출력하여 선택 출력합니다.
- **한전 방법이 있는 경우**: 전산지를 대상으로 10자리를 등록하면 출력 가능합니다.
- 입주자, 사업소, 시건소, 시연소, 전자소, 수연이 분리기로 분리한 경우 안내합니다.

### 인터넷검침전송 (한전검침송신작업)

![한전검침송신작업 화면](/screenshots/INSP1280.png)

**메뉴 경로**: 검침 > 전기검침 > 한전검침송신작업

한전의 인터넷 수행 프로그램을 통하여 단지의 사이를 사용합니다.

**사용 방법:**

1. **검침월과 한전 년월** 및 동을 입력합니다.
2. **수신받은 데이터를 선택**합니다.
3. **단지/사업소/시건/천소, 수소/시용**을 선택합니다.
4. **2021.01 이후** 정보 송신 가능합니다.
5. **당창이로 전송**합니다. 작업이 없으면 단지 알림처럼 됩니다.
6. **단지인데로 전달**: 현재 한 전단 데이터를 송신합니다.

**한전 전기검침 전반 방법:**
1. 단지 지침이 > 전기검침 > 한전검침단말기지를 선택하여 검침을 끝낸 이후에 한전단말기지검침을 단지 전기로 선택에서 전기검침 한전단말기지를 채워 합니다.
2. 이인단이면서 전기 검침은 전달로 이루어지며 단 점 제 단지 요청을 선택합니다.
3. 이후 단당 단지 데이터 업로드 후 전기검침 단말기지를 선택하여 검침을 완료합니다.

## 사용 방법 (종합)

1. **전기검침 메뉴 진입**: 검침 > 전기검침 > 전기검침
2. **검침 년월 선택**: 상단 년월 필드에 해당 월 입력
3. **전월지침 수신**: 전월지침 버튼 클릭으로 전월 데이터 수신
4. **동 선택 및 검침값 입력**: 동 선택 후 각 세대의 금월 지침값 입력
5. **에러 확인**: 이상 사용량(에러 표시 세대) 재확인 및 재검침
6. **요금계산**: 요금계산 버튼 클릭으로 전기요금 산출 (TV수신료, 방안요금 포함)
7. **저장**: 저장 버튼 클릭
8. **결과 확인**: 전기사용량조회 메뉴에서 확인

- 검침값은 전월 지침보다 작을 수 없습니다 (계량기 교체 시 제외)
- 이상 사용량 발생 시 에러 세대를 재검침 후 확인하세요
- 한전검침송신작업은 한전과의 연계 작업으로 전산 담당자와 협의 후 진행하세요
- TV수신료는 요금계산 시 자동 산출되지만 면제 세대 여부를 사전 확인하세요
- 계량기 고장 세대는 고장 처리 체크 후 요금을 별도 계산하세요
- 한전 인터넷 검침 전송은 2021년 1월 이후 기준으로 운영됩니다
