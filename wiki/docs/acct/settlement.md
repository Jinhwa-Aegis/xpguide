# 결산관리

> **메뉴 경로**: 회계 > 결산관리

## 개요
월별 또는 연간 회계 결산을 처리합니다. 시산표, 재무제표, 손익계산서, 세입세출서 등 결산 보고서를 생성하고, 회계연도 종료 시 연도 이월을 처리합니다.

## 소메뉴 상세

### 합계잔액시산표
![합계잔액시산표 화면](/screenshots/ACCT4010.png)

일정 기간 동안 발생한 모든 계정과목별 차변·대변 합계와 잔액을 조회하는 메뉴입니다.

**사용 방법**

- 일정 기간 동안 발생한 모든 계정과목별 차변·대변 합계와 잔액을 조회합니다.
- 조회 기간(시작일~종료일)을 설정하여 조회합니다.
- 조회된 결과는 인쇄 가능합니다.

### 대차대조표(재무상태표)
![대차대조표 화면](/screenshots/ACCT4020.png)

자산, 부채, 자본 현황을 나타내는 재무상태표로 조회 및 인쇄할 수 있는 메뉴입니다.

**사용 방법**

- 자산, 부채, 자본을 항목별로 조회합니다.
- 특정 시점의 재무 상태를 확인하기 위해 기준일을 설정합니다.
- 조회된 결과는 인쇄 가능합니다.

![손익계산서 화면](/screenshots/ACCT4030.png)

일정 기간 동안의 수익·비용을 집계하는 손익계산서를 조회 및 인쇄할 수 있는 메뉴입니다.

- 조회된 결과는 인쇄 가능합니다.

### 세입세출서
![세입세출서 화면](/screenshots/ACCT4080.png)

예산을 등록하거나 사용하는 메뉴로, 회계기간 및 특정 기간으로 결정 기간을 조회 및 인쇄할 수 있는 메뉴입니다.

**사용 방법**

1. 조회할 세입세출서 구분을 선택합니다: 세입세출서, 세입세출서, 세출세입서, 세출서
2. 세입·세출 기간(기간분):
   - 세입세출서: 1년(회계기간)
   - 세입세출서: 1년 이전 기간(1년, 9, 12월)
3. 조회 기간 선택 후 조회합니다.

**세입세출 계산식**

| 항목 | 설명 |
|------|------|
| 예산액 | 예산으로 확정된 금액 |
| 예산수정액 | 추경에 의한 예산 변경 금액 |
| 수입/지출액 | 실제 수입·지출 발생 금액 |
| 증감 | 예산금액 / 전도목적액 × 100 |

- **예산식**: 전년도 1년 전도된 금액 = 전년도세입총액 / 전년도내역 + 금사경계액 → 수입금액
- **결산식**: 각 산 '목' 예산식 + 추경계산 → 연도 수입금액 → 결산액

> ※ 회계 > 예산관리 > 세입관리 혹 메뉴에서 구분·세부/세출을 입력하면 성능이 전됩니다.

### 년도이월
<!-- 스크린샷 준비 중: ACCT4090 -->

회계가 년도마감되거나 회계연도가 끝난 경우 다음 연도로 해당 잔금의 장기수익을 이월할 수 있는 메뉴입니다.

**사용 방법**

1. 년이월이 항목 기수를 선택한 후 조회합니다.
2. 당기순이익을 이월할 금액의 계정과목별 잔액을 확인합니다.
3. 이월 대상 계정과목의 잔액이 다음 연도로 이월됩니다.

**이월 처리 원칙**

- 이후관련: 계정에서 제외된 각 해당 이월대상 잔액별 당기순이익이 다음연도로 이월됩니다.
- 이월된 다음년도 각 계정의 해당 잔액으로 연결됩니다.
- 이월 후 재실행이 필요한 경우 년도이월을 다시 진행할 수 있습니다.

**이월 시 주의 사항**

- 이월 완료 후에는 대차대조표와 손익계산서로 조회가 됩니다.
- 이월 이후에는 취소가 불가합니다. 실행 전 반드시 내용을 확인하세요.
- 이월 완료 후 문제가 발생한 경우 반드시 문의하시기 바랍니다.

## 주의사항
- 결산 전 모든 전표가 입력·승인 완료되었는지 확인하세요
- 결산 후에는 해당 기간의 전표 수정이 제한됩니다
- 합계잔액시산표의 차변 합계와 대변 합계가 일치해야 결산이 정상입니다
- 년도이월은 연 1회 회계연도 종료 시에만 처리합니다
- 이월 전 당해연도 모든 결산이 완료되어야 합니다
- 이월 완료 후에는 취소가 불가하므로 실행 전 반드시 내용을 확인하세요
