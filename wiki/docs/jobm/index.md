# 민원

> **메뉴 경로**: 민원

## 개요
입주민의 민원을 접수하고 처리 현황을 관리합니다. 작업 관련 기기 및 자재 입출고 관리도 포함합니다.

## 하위 메뉴

### [민원관리](/jobm/manage)

#### 기본정보
| 소메뉴 | 설명 |
|--------|------|
| 작업코드등록 | 민원 작업 유형 코드를 등록합니다 |
| 기기등록 | 작업에 사용하는 기기를 등록합니다 |
| 제품등록 | 사용 제품 정보를 등록합니다 |
| 분류명등록 | 민원 분류 명칭을 등록합니다 |
| 거래처등록 | 관련 거래처를 등록합니다 |

#### 작업정보
| 소메뉴 | 설명 |
|--------|------|
| 민원코드등록 | 민원 유형 코드를 등록합니다 |
| 세대민원접수및작업 | 세대 민원을 접수하고 작업을 처리합니다 |

#### 입/출고정보
| 소메뉴 | 설명 |
|--------|------|
| 기기보수 | 기기 보수 내역을 등록합니다 |
| 입고등록 | 자재 및 부품 입고를 등록합니다 |
| 출고등록 | 자재 및 부품 출고를 등록합니다 |
| 이월작업 | 미완료 작업을 다음 기간으로 이월합니다 |
