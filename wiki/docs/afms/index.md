# 장기수선계획

> **메뉴 경로**: 장기수선계획

## 개요
아파트 공용부분의 장기적인 수선 계획을 수립하고 장기수선충당금을 관리합니다. 공동주택관리법에 따라 의무적으로 수립해야 합니다.

## 서비스 안내

> 장기수선계획 서비스는 외부 서비스와 연동됩니다. 해당 메뉴 선택 시 외부 서비스 페이지로 이동합니다.

## 주요 기능
- 장기수선계획 수립 및 변경
- 장기수선충당금 적립 현황 조회
- 수선 공사 이력 관리
- 장기수선계획 보고서 출력
- 수선 항목별 집행 현황 관리

## 화면 구성
> 스크린샷 영역 (추후 추가 예정)

## 사용 방법
1. 장기수선계획 메뉴 진입
2. 서비스안내를 통해 외부 서비스 접속
3. 수선 항목 및 일정 확인
4. 충당금 적립 현황 조회
5. 필요 시 계획 조정

## 관련 메뉴
- [예산관리](/acct/budget) - 수선 공사 예산 연계
- [전표관리](/acct/voucher) - 수선 공사비 전표 처리

## 주의사항
- 장기수선계획은 3년마다 검토하여 조정해야 합니다
- 입주자대표회의 의결이 필요합니다
- 장기수선충당금은 의무 적립 대상 항목을 기준으로 관리합니다
- 수선 공사 시행 전 입주자대표회의 승인을 받으세요
