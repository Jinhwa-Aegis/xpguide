# 아파트아이

> **메뉴 경로**: 아파트아이

## 개요
아파트아이는 입주민을 위한 모바일 앱 서비스입니다. 관리비 조회, 민원 접수, 공지사항 확인 등의 서비스를 입주민에게 제공합니다.

## 소메뉴 상세

| 소메뉴 | 설명 |
|--------|------|
| 입주자단체등록 | 입주자 단체(동대표회 등) 정보를 등록하고 관리합니다 |
| 소방시설세대점검 | 세대별 소방시설 점검 결과를 등록하고 관리합니다 |
| 대피계획수립 | 화재 등 비상상황 시 대피 계획을 수립하고 관리합니다 |
| 모바일입주민카드 | 모바일 앱을 통한 입주민 인증 카드를 관리합니다 |
| 방문차량서비스설정 | 방문 차량 관리 서비스의 설정을 구성합니다 |
| 방문차량관리 | 단지 내 방문 차량을 등록하고 관리합니다 |
| 방문차량주차비부과 | 방문 차량에 대한 주차비를 부과합니다 |
| 전자고지서 | 관리비 등 전자 고지서 발송을 관리합니다 |

- [관리비부과조회](/impo/inquiry) - 관리비 고지와 연계

## 주의사항
- 입주민 앱 서비스 이용을 위한 별도 설정이 필요합니다
- 모바일입주민카드 발급 전 입주민 정보가 정확하게 등록되어 있어야 합니다
- 전자고지서 발송 전 수신 동의 여부를 확인하세요
- 소방시설 점검 결과는 법정 기한 내에 등록하세요
