# 연말정산

> **메뉴 경로**: 인사/급여 > 연말정산

## 개요

직원의 연간 근로소득에 대한 소득세 연말정산을 처리합니다. 연말정산기본정보 등록부터 전직장내역 등록, 사업장변경 처리, 연근로소득내역 등록, 개인공제자료 등록(부양가족·의료비·기부금·월세·연금저축 등), 연말정산 계산 작업, 결과 조회, 정산 파일 작성(국세청 전자신고), 징수환차/환급금 처리까지 연말정산 전 과정을 지원합니다.

## 연말정산 처리 순서

```
1. 연말정산기본정보등록 → 2. 전직장내역등록(해당자) → 3. 사업장변경처리(해당자)
→ 4. 연근로소득내역등록 → 5. 개인공제자료등록 → 6. 연말정산작업
→ 7. 연말정산결과조회 → 8. 정산디스켓작성(전자신고) → 9. 징수환차/환급금 처리
```

## 소메뉴 상세

### 연말정산기본정보등록

> **메뉴 경로**: 단지관리 > 단지정보 > 연말정산기본정보등록

연말정산 기본정보와 세금계산 기본정보를 등록하는 메뉴입니다.

**주요 기능**

- 연간도의 동일한 경우 조회 후 복사를 클릭합니다.
- 이미 등록된 사업자번호가 있는 경우 해당 연도의 연간사업자 정보가 복사됩니다.
  1) 세액표를 불러올 때 발행되어 연도사이비스를 확인하는 경우
  2) 사업장번호 형성 한 뒤 연도사업 서비스를 한 뒤 연도사업형태별을 입력할 경우
- 세금 가져오는 수를 4개월로 1,500만원 이상 무이자이며, 이후 경우 1,500만원씩 이상 경우는 자세바랍니다.

**입력 항목**

| 항목 | 설명 |
|------|------|
| 연도 | 정산 대상 연도 |
| 사업자번호 | 해당 단지/사업장 사업자등록번호 |
| 세액 계산 기준 | 정산 시 적용할 소득세 세율 기준 |

### 전직장내역등록

![전직장내역등록 화면](/screenshots/PAY4020.png)

해당년도 중간 입사한 직원에 대한 전직장 내역을 등록하는 메뉴입니다.

**주요 기능 및 설명**

- 해당년도 중간에 입사한 직원이 해당됩니다.
- 동록된 사원을 선택한 후 전직장에서 발급받은 한 전년도 원천징수영수증을 참고하여 등록합니다.
- 전진근무자에서 원천징수고표를 확인 후 관련 닫는 달 전년 이후 원천징수금액을 행해서 이 시행 합니다.
- 해당 금여 및 전직근무자가 (근로소득·근무소득세 포함)의 이고(4대) 원천징수영수증을 (이 시행)을 원 이행 시 이용 합니다.
- 부양가족지수가 수 이상의 경우인 경우에 관한 이용 종이 없이 금이 있는 특정납부이 원천징수영수증을 내어 이 작업을 사원에 가능합니다.
- 중간근무자가 4부 이상 이상인 경우 원천징수금을 (근로소득세 부분)을 (이행)을 원천징수이행에 배치 이 사용 합니다.

**입력 항목**

| 항목 | 내용 |
|------|------|
| 직원 선택 | 전직장이 있는 해당 연도 중도 입사자 선택 |
| 근무처명 | 전 직장 회사명 |
| 사업자등록번호 | 전 직장 사업자등록번호 |
| 급여 | 전 직장 총급여액 |
| 근무 기간 | 전 직장 근무 시작일~종료일 |
| 징수세액 | 전 직장에서 원천징수된 소득세 |
| 지방소득세 | 전 직장에서 원천징수된 지방소득세 |

::: tip
전직장내역은 해당 직원의 이전 직장 원천징수영수증 기준으로 입력합니다.
중간 입사자가 전직장내역을 제출하지 않으면 당해 연도 소득만으로 연말정산이 진행됩니다.
:::

### 사업장변경처리

> **메뉴 경로**: 인사/급여 > 연말정산 > 전직장내역등록 > 사업장변경처리

사업자나 변경시 변경직장에서 계속 근로하는 경우 직원의 연간근로소득을 계속 근로기간으로 처리합니다.

**주요 기능**

- 회사(사업장)가 변경된 직원을 처리합니다.
- 사업자번호 변경 전 고근무기간을 조회 후 근무기간 프로를 불러옵니다.
- 연 전 사업자까지 사업자들의 공소부까지 결정합니다.

**적용 유형**

| 상태 | 해당 유형 |
|------|-----------|
| 동일한 직원 없는 상태 | 전 직장 원천징수영수증 있는 경우 |
| 없는 직원 | 전직장 있는 경우(r/r-이날) 경우 이날 경우 없는 경우 |
| 근처한 직원 | 전 두 이상 현: 근로(기업소) 기간 2022.01~2022.03.31 이 있는 경우 |

**사업장변경처리 예시 (p.34)**

```
사업장변경 전 근무기간 : 2022.01.01~2022.03.31
→ 연근로소득에서: 2022.04~2022.12.12
```

### 연근로소득내역등록

![연근로소득내역등록 화면](/screenshots/PAY4010.png)

해당년도에 계산하여 자산정보로 발행으로 수신한 근여를 개인별로 수동으로 처리합니다.

**주요 기능**

- 조회 버튼을 클릭하여 직원 목록을 불러옵니다.
- 사원을 클릭한 후 하단에 급여계산에서 급여(비과세)가 먼저 출력되어 있습니다.
- 급여계산 결과 상 실제 급여 후 첫 번째 렬에서 급여(비과세)를 확인하세요.
  수정 및 절에 경우에 연근로소득내역 수정을 확인하여 처리합니다.

**참고 사항 (p.35~38)**

- 직원자/직원의의 중 급여계산에서 3000만원 이하 경우에 이 급여(근)보수서를 확인한 경우에 직원에서 받아야 볼 수 없습니다. 1 직원사용·부서직원이, 중 단지명시·사업장명 사용·수신중에서 근무소득내내역에서 연근로소득내역부터 한 번에 수정합니다. 연급/부서별대에서 2) 수정년도 처음에 자 시 발행된 연근로소득내역내에 해당 소득금액을 이 수정합니다.
- **건강보험료 장기기여금법령**을 납부하고 납부한 직원에 가산자 급여에서 법정건강보험료·장기기여법령이 이라는 거 처리됩니다.
- 사업장변경이 다로 연근로소득내역에 해당 급여에서 직원의 연근로소득계산에서 필요에 따른 복잡성의 처리가 다. 해당 직원의 처리 항목이 나합니다.
- **시기기준변경이 이 경우 이**: 총근무기간 중 시기기준변경에서 기간에서 경우 누적 경우 이 건 확인합니다.
- **선택연금**: 회사기준이 급여에서 시기기준에서 연금금액 이상 근무 이상의 경우 자금이 기여금 납입에 수수할 경우 이 선택합니다.

**연근로소득내역 항목**

| 항목 | 설명 |
|------|------|
| 근무기간 | 해당 연도의 실제 근무 시작일~종료일 |
| 총급여 | 과세 급여 합계 |
| 비과세소득 | 식대, 차량유지비 등 비과세 급여 합계 |
| 상여 | 상여금 지급 총액 |
| 인정상여 | 법령에 따른 인정상여 금액 |
| 소득세 | 원천징수된 소득세 합계 |
| 지방소득세 | 원천징수된 지방소득세 합계 |

### 개인공제자료등록

![개인공제자료등록 화면](/screenshots/PAY4030.png)

근로자의 가족사항 및 추가공제상, 공제범위 등을 입력하는 메뉴입니다.

**탭 구성**

| 탭 | 내용 |
|----|------|
| 부양가족 | 기본공제 대상 부양가족 정보 |
| 보험/연금저축 | 보험료 및 연금저축 공제 |
| 주택자금 | 주택임차·장기주택마련저축 공제 |
| 신용카드등 | 신용카드·체크카드·현금영수증 사용액 |
| 의료비 | 의료비 세액공제 |
| 교육비 | 교육비 세액공제 |
| 기부금 | 기부금 세액공제 |
| 기타 | 기타 공제 항목 |

**부양가족 탭 (p.39)**

- 전직장내역등록을 통해 연근로소득내역에서 차료된 공제 항목들을 불러옵니다.
- 배당가족별 연도내 연근로소득내역에서 차료된 공제 항목들을 불러옵니다.
- 등록된 직원은 하단 다이얼로그에서 공제 항목을 입력합니다.

**홈택스 간소화 자료 연동 (p.40~43)**

국세청 홈택스에서 개인공제자료를 PDF로 다운로드하여 시스템에 업로드할 수 있습니다.

**다운로드 절차**

1. 홈택스(www.hometax.go.kr) 접속 후 로그인합니다.
2. [조회/발급] > [연말정산] 메뉴를 클릭합니다.
3. 소득·세액공제 자료 조회에서 한번에 조회하기를 클릭합니다.
4. 내려받기 항목을 확인 후 **PDF로 내려받기** 버튼을 클릭합니다.
5. 문서기암호(비밀번호)는 설정하지 않습니다.
6. 건강보험 등 금액을 확인 후 한번에 내려받기 버튼을 클릭합니다.

**업로드 절차 (p.42~43)**

1. PDF 파일을 사원별로 준비합니다.
2. 자동 선택 후 다운로드한 PDF 자료를 선택합니다.
3. 보험류 항목별 데이터가 자동으로 불러와집니다.
4. 추가공제자료에 관련 데이터를 반드시 확인하세요.
   - 추가·수정이 없다면 자동으로 저장합니다.
5. 이름상 방법으로 업로드 후 금액이 다르게나 비교됩니다.

::: warning
PDF 자료의 부양가족 주민등록번호가 없을 경우 일치하는 부양인이 없다고 표시됩니다.
업로드 후 금액을 반드시 확인하고 불일치 항목은 수동으로 수정하세요.
:::

**개인공제자료 상세 입력 내용 (p.44~49)**

**보험/연금저축 탭**

| 항목 | 설명 |
|------|------|
| 국민연금 | 연로소득공제 포함 (국민연금) |
| 건강보험료 / 고용보험료 | 특별소득공제 (보험료) |
| 일반보장성보험 | 기본공제 대상자 명의로 가입한 보험료 |
| 장애인전용보장성보험 | 장애인 전용 보험료 |
| 연금저축 | 연금계좌 세액공제 |
| 퇴직연금 (DC·IRP) | 퇴직연금 세액공제 |

**주택자금 탭**

- 장기주택저당차입금 이자상환액 공제 (구주택 상한 연간 소득공제)
  1) 주택저당차입금 상한 금액 소득공제
  2) 장기주택저당차입금 이자상환 소득공제
- 주택임차 대주주택: 소득공제, 공주주택 대주주택 세액공제 기능

**주택임차 대상**: 무주택자, 공무원주택, 단독주택 대 주거 집합체, 상가건물도 대상
주택임차입지:   대상여부와 기준에 따라 정부부, 상가건물부, 부동산부, 상가건물부

- 장기집합투자증권저축: 납입금액 40% 이상 이상의 국내 주식 투자비 기초 투자기간이 10년 이상 남짓 납입한 600만원, 제우기간 10년 이상 등도 적금입은 것

**의료비 탭 (p.51)**

- 조회된 직원을 선택 후 오른쪽 한번에 팝업 버튼을 클릭합니다.
- 의료비 자료를 활용하기 기간별로 기간별 건강내용 등 입력합니다.
- 5도: 국세청에서 제공하는 자료는 크게 대한 환경이 있습니다.
  - 건강보험공단에서 전달될 의료비는 (-)부호를 이용합니다.
  - 사업자/법인은 의료비의 0 ( )을 (상) 의료비로부터 이용하여 입력합니다.
- 자료코드는 \*의료비총금코드 내에서 내용을 검색하거나 직접합니다.

**기부금 탭 (p.52)**

- 조회된 직원을 선택한 후 기부금을 클릭합니다.
- 기부 종류를 선택한 기부금 금액에 이 금액 등의 내용을 등록합니다.
- 바다: 기부금자료공제자료등록을 이 전반 사원의 기부금이 있는 경우 단지의 기부금 조회 표시가 보이지 않을 수도 있습니다.
- 이번공제가 보이지 않을 경우 기부금 조회 후 조회한 기부금 버전을 높이 동록합니다.

**월세액/주택임차등록 (p.53)**

- 조회된 직원을 선택한 후 오른쪽 조회 완료에서 팝업이 를 클릭합니다.
- 상단의 급여와 급여계산창에서 공제총 월세액으로 총계를 확인합니다.
- 주택임차(주택의 주거에 있는 취득목적으로 임차)는 경우가 있습니다.
- 시/법령지역: 필수 지역에 따라 취정하여 임직원의 거주지, 고정자, 건강 지역 등의 경우 이 임차하여 처리됩니다.
- 소 금전이에 대해서 타인 지급이 경우 임금/직급에서 대법원에 직접 임직원 기기에 있는 사업장과 동일하여 처리에 담당 지역 등에서 이 지정됩니다. 필수 지역에 따라 해당 직원에 임 직원지역에서 더 처리에 사업자 등록은 이루어지도록 합니다.
- 사보험이 단지 반드시 임직원 경우 임금/직급에서 대법원에 직접 임직원 기기에서 이 사업장과 동일하여 처리에 담당 지역 등이 이 지정됩니다.

**연금/저축공제자료등록 (p.54)**

- 조회된 직원을 선택합니다.
- 연금계좌·연금저축·장기집합투자저축소 공제 내용을 입력합니다.
- 연금·연금저축·소득공제 및 세액공제 구분별로 별도로 입력합니다.
- 퇴직연금으로 처리·퇴직금·연금저축을 등 면제자격 구분을 선택 후 입력
- 주택자금소득·소득세·대출 및 연금저축·자금저축을 구분을 선택 후 입력

### 연말정산작업

![연말정산작업 화면](/screenshots/PAY4040.png)

직원별 입력된 자금을 산정하고 전직장내역에, 연근로소득내역, 개인공제자료를 기록한 후 연말정산 계산을 실행합니다.

**주요 기능**

- 전직장·연근로소득내역 자료를 입력 후 전직장자료를 포함하고 전직장은 전직장 방향으로 정산합니다.
- 처리자가 퇴직한 경우 기록 작업을 하고 처리를 한 후 전직장내역 자료를 올리에 급여에서 급여가 연말자정산 업무를 처리합니다.
- 특정 직원에 경우 정산 작업이 됩니다. 정산내역을 조회하면 전체적으로 볼 수 있습니다.
- 연말 정산 이후에도 항상 이 기록합니다. 이 처리 자료를 기반으로 하면 최적 처리 자료를 클릭하여 현황 연말 정산 업무를 처리합니다.

**연말정산 실행 절차**

1. 정산할 직원을 선택합니다.
2. 전직장내역, 연근로소득내역, 개인공제자료가 모두 등록되었는지 확인합니다.
3. 연말정산 작업 버튼을 클릭합니다.
4. 계산이 완료되면 결과를 조회합니다.

::: warning
연말정산 작업 전 모든 공제 자료(개인공제자료, 전직장내역, 연근로소득내역)가 입력 완료되어야 정확한 계산이 이루어집니다.
:::

### 연말정산결과조회

![연말정산결과조회 화면](/screenshots/PAY4041.png)

연말정산 작업을 완료한 근로자의 연말정산 금액을 확인하고 원천징수영수증/소득자료세제합계표 등을 출력합니다.

**주요 내용**

- 정산이 완료된 사원의 경우 신고구분에 없는 소득자를 선택 후 처리합니다.
- **소득세 금액**: 비과세소득 계로 포함됩니다.
- **소득세 금액**: 소득세 금액으로 경상세표합니다.

**출력물 종류 (p.55~56)**

1. **근로소득원천징수영수증**: 직원에게 교부하는 개인별 연말정산 결과 영수증
2. **소득자료세제합계표**: 사업장 전체 직원의 소득 및 세액 합계

**특별소득공제 참고 (p.56)**

- 국세청에서 특별소득공제자에 세액공제를 산정하지 않을 경우 특별소득공제에 해당 표준세액을 날 수 없고 있고, 기재합니다.
- XpERP에서는 특별소득공제자에 특별소득공제에 공제한 경우 표준에 해당하는 공제금은 정확성에 의해 결과 결과를 따른 공제를 가장 유리한 방향으로 자동 계산합니다.
- 특별소득공제자에 세액공제 기부자 등록 0 인의 경우 기준 00소득세로 특별소득공제 신청에 해당합니다.

### 정산디스켓작성

![정산디스켓작성 화면](/screenshots/PAY4050.png)

연말정산결과를 확인한 후 국세청 신고를 위한 전자 신고 그를 위한 필요한 디스크 파일을 만드는 메뉴입니다.

**주요 기능**

- 디스크 교육 파일이 있고 파일이 먼저 빠져나가는 것입니다.
- 전제지의방법이자의 법 유형을 선택합니다.
- 전체/일부에서 직접/사업장 단위 및 부양가족을 선택합니다.
- 적합성을 결정하여 전자신고를 처리합니다.

**전자신고 절차**

1. 연말정산결과조회에서 검토 완료 후 정산디스켓작성 메뉴로 이동합니다.
2. 신고 유형(전체/일부)을 선택합니다.
3. 저장 경로를 지정하여 파일을 생성합니다.
4. 국세청 홈택스에 해당 파일을 업로드하여 신고합니다.

### 징수환차/환급금

![징수환차/환급금 화면](/screenshots/PAY4060.png)

연말정산 납부 완료 후 차감/징수금액을 급여에 반영하는 메뉴입니다.

**주요 기능**

- 정산년도를 클릭하여 조회합니다.
- 급여년월과 급여일을 선택하면 해당 직원의 징수/환급 금액을 확인합니다.
- 지자료가 이 개인 전 소득 기소득세/지방소득세에서 급여(연말금)에서 급여지급금로를 반영합니다.
- 급여지급년월을 조회하면 다음 급여기간에서 환급금/징수금이 급여에 자동 반영됩니다.
- 급여계산 후 명세서를 출력하면 직원의 징수금/환급금 총계를 확인할 수 있습니다.

**환급/징수 처리 방법**

- 연말정산 후 추가 납부액(징수액)이 발생한 경우: 다음 급여에서 차감 처리
- 환급액이 발생한 경우: 다음 급여에 합산하여 지급
