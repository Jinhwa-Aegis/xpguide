# 급여관리/조회

> **메뉴 경로**: 인사/급여 > 급여관리 / 인사/급여 > 급여조회

## 개요

직원 급여를 산정하고 지급 처리합니다. 급여기본정보 등록부터 월별 급여 계산, 명세서 출력, 이체 처리, 급여 항목별 조회, 보험료 공제 현황, 원천징수이행신고서 작성까지 급여 관련 전 과정을 지원합니다.

## 소메뉴 상세

### 급여기본정보

![급여기본정보 화면](/screenshots/PAY3010.png)

급여계산에 필요한 직원별 급여정보 및 전약정보를 등록하는 메뉴입니다.

**화면 구성**

| 탭 | 설명 |
|----|------|
| 기본정보 | 기본급, 각종 수당 등 급여 항목 설정 |
| 건강보험 | 건강보험 자격 및 보수월액 정보 |
| 고용보험 | 고용보험 자격 및 취득일 정보 |
| 국민연금 | 국민연금 기준소득월액 정보 |
| 장기기여금 | 장기기여금 정보 |

**주요 기능 및 입력 항목**

- 조회 버튼을 클릭합니다.
- 입력방법: 사원번호로 이동 (입력방법: 합력방법 - 카서이 가여야 이동)
- **탭 별 처리 내용**:
  - **1번 탭**: 기본급, 각종 수당 설정. 0.1개의 단위로 자동계산이 가능합니다.
  - **국민연금·건강보험·고용보험**: 건강보험공단 EDI 신고 기반으로 자동 처리됩니다.

- **소득세 계산 연동**: 소득세정의 경우 연말이 되기 전에 연금급에서 직원 세금이 계산됩니다. 년도 소득세가 반영된 전반 시 납부에서 계산(세금은),급여계산에 배분하기 위해서, 표준세금은 이 급여(급여세 남인 세금)표준에서 고 더해진다.
- **사원범주 선택 없는 경우 입력**: 급여가 아이다 직급/직종에 직원 필수를 입력하는 사원 없이 선택되어 별도로 처리됩니다.
- **경감대신 선택 없는 경우 타입**: 급여가 아이다. 직종에서 직원 목록을 입력하기 위해서 사원 없이 선택되어 별도로 처리됩니다.
- **장기기여금 자동 여부**: 정기기여금 선택 없는 경우 기준이 지정된 후 별도 여부 등 급여 산정/전달에 처리됩니다.
- **직원 기입 후 완료 시 연동**: 급여수수기준의 경우 이 연기기기금에 급여기금에서 자신 급여 확인에서 자신 연금제도로 선택합니다.

**입력 탭별 상세 내용 (p.17~19)**

1. **기본정보 탭**
   - 소득세/지방소득세 연말이후 전표등록을 직원 입력하여 급여계산을 처리합니다.
   - 년도소득세 반영된 소득세의 환급 납부에서 계산하여 급여에서 배분하여 처리합니다.
   - 사원별 선택 없을 경우 입력이 됩니다.

2. **건강보험·국민연금·고용보험 탭**
   - 건강보험료 인적정보를 입력 후 건강보험·연금·고용보험 정보를 등록합니다.
   - 국연계산 여부 및 고용보험대상 여부를 기입합니다.
   - 정보입력이 완료 된 사원의 경우 직접 입력 사원 없이 선택되어 자동으로 됩니다.
   - 장기기여금 선택 없이 입력하거나 자직 연연기금입이 될 경우 특허된 사원 없이 자동으로 처리됩니다.

3. **소득세 관련**
   - 표를 수작입으로 입력하면 소득세들 급여에서 차감하여 급여계산시 급여를 줍니다.
   - 급여기금 선택 후 이이 환급이 자동으로 받아지고 경우에 따라서 이전에는 음수를 수 있 없는 점도 있습니다.
   - 추가적인 금액을 입력 가능 경우 입력하기 위해서 기관이 급여 항목을 넣으면 되지 않습니다.
   - 준공수가 완료된 정규 경우 소득세 직원 연금에서 자동으로 선택합니다.

**사용 절차**

1. 인사기본정보 직원 인사정보를 인사입력 처리하여 내용을 불러옵니다.
2. 인사정보를 기준으로 소득세/지방소득세를 처리합니다.
3. 고용보험 및 건강보험의 경우 국세청에서 신고하거나 단지에서 직접 신청할 수 있습니다. 국세청에서 집계된 방법을 직접 확인하여 방법을 처리합니다.
4. 장기기여금 여부 및 연말기기금에 급여에서 자신 연금으로 등록합니다.

::: warning
급여기본정보를 등록하지 않은 직원은 급여 계산 대상에서 제외됩니다.
입사 후 반드시 급여기본정보를 등록하세요.
:::

### 급여계산

![급여계산 화면](/screenshots/PAY3020.png)

매 월 지급하는 급여를 계산하는 메뉴입니다.

**화면 구성**

| 영역 | 설명 |
|------|------|
| 상단 조건 | 년도·월 선택, 계산 대상 직원 범위 설정 |
| 직원 목록 | 계산 대상 직원 목록 |
| 지급 내역 | 기본급, 수당 등 지급 항목별 금액 |
| 공제 내역 | 소득세, 4대 보험 등 공제 항목별 금액 |
| 합계 | 지급합계, 공제합계, 차인지급액 |

**급여계산 주요 기능**

1. 보여지는 화면을 관련된 글로 클릭합니다.
2. 급여계산 설정을 확인 후 급여계산 기능을 활성화합니다.
3. 지자계산된 날짜의 값은 없이 경우 급여자 날짜가 생겼을 때 값 자동으로 처리가 이루어집니다.
4. **동도 입사자**의 경우 입사자 급여지급이 급여지급구분이 출력해야 이루어집니다.
   - 자동 출력시나 입력이 되기 전에, 출력해야 이루어지지 않습니다.
5. 급여계산 후 에서 이전 회사처럼 급여 이후에서 급여처럼 데이터를 처리할 수 있습니다. 참고로 이전에 동도 입사하여 사람들이 이 있을 것이기 때문에 급여이후에 자동으로 기록이 안됩니다.
6. **소득세 자동계산**: 급여계산 시 기준 금액으로 소득세를 자동 계산합니다.

**급여계산 절차 (p.20~22)**

1. 보여지는 화면을 관련된 글로 클릭합니다.
2. 급여계산과 지급일을 입력 후 조회합니다.
3. 자동계산된 기여의 값을 점수 기준으로 소득세·수수를 자동으로 계산합니다.
4. 선택 내: 전계 - 급여계산 전 / 전: 급여계산 후
5. 급여가에 따라 다르게 급여계산이 급여에따라, 급여과제에 따라 급여를 차별로 처리합니다.
6. 사원을 클릭하면 해당 사원의 급여계산이 금액이 급여계산이 표시됩니다.

::: tip
급여계산 전 반드시 보험료율, 소득세율 등 공통정보가 최신 상태인지 확인하세요.
급여계산 후에는 금액을 반드시 검토하고 확정 처리하세요.
:::

**일할계산 (p.22)**

- 급여내역과 지급일을 입력 후 조회합니다.
- 급여계산 설정을 확인 후 급여계산을 시작합니다.
- **동도 입사자** 기준으로 급여지급이 아직 이루어지지 않는 경우 각 입사 시점에 따라 자동으로 처리됩니다.
  - 수정 또는 삭제를 한번 입력한 후에는 재직 사원은 없이 선택되어 급여계산 자동 결과를 처리하게 됩니다.
- 급여계산 후 소득세(지방소득세)를 자동으로 빼지 않습니다.
- 급여계산 후 연말정산 결과를 기반으로 급여 소득세 공제됩니다.

### 급여이체출력

![급여이체출력 화면](/screenshots/PAY3110.png)

급여납부와 지급일이 확인 후 이체할 리스트를 출력할 수 있는 메뉴입니다.

**주요 내용**

- 지급년월과 지급일을 선택하면 급여대장과 이체 내용이 조회됩니다.
- 급여년월 선택 후 급여기본정보에서 아이 직원, 예금주가 입력되어 있어야 합니다.
- 급여내용은 지급 후 급여계산창에서 아이 직원, 예금주가 동기화됩니다.
- **06분 급여이체 내용**: 출력하여 은행 이체가 이루어지도록 합니다.

**출력 내용**

이체 리스트에는 다음이 포함됩니다.
- 직원 성명
- 은행명 / 계좌번호
- 이체 금액 (차인지급액)
- 합계

### 명세서출력

![명세서출력 화면](/screenshots/PAY3120.png)

급여에서 다양한 종류의 명세서를 인쇄할 수 있는 메뉴입니다.

**명세서 종류**

1. **급여명세서**
   - 직원에게 급여를 기본으로, 근무시간 등을 설정하는 1~3개의 선택기간 동안의 13개의 처리 클릭합니다.
   - 직원직자를 출력하기 위한 기본으로 선택자의 일별 직원구에 직원 사원 중에서 선택 후 선택 후 수정을 진행합니다.
   - 직업별 자직 비용을 높이려면 직원에서 급여명세서에 배정 완료됩니다.

2. **지급대장 A, B (출력물)**
   - 지급대장 A를 출력 또는 인쇄합니다.
   - 지급대장과 급여·공제 내역의 칸을 선택 후 지급대장에 선택 후 설정합니다.
   - 지급대장과 급여/공제 순서 각기를 기준으로 달리 설정합니다.

3. **자관한상공증명서**
   - 지급대장 기능 사원에서 자관에서 직원까지 직원자관상공근무자증명서를 출력합니다.
   - 기준이 된 일 에서 단지공사일의 내역기준의 기본적인 직원증명서를 출력합니다.

4. **5D이이급여명세서**
   - 출력을 요청할 경우 자직원 이용에 따른 인원으로 보내주는 급여 이서비스를 기본적인 처리합니다.
   - 모바일 이 기능이 가능하되, 이 기능은 가능할 경우 기능부분이 있습니다.

**명세서 출력 절차 (p.24~26)**

1. 조회 버튼을 클릭합니다.
2. 직원의 근무, 급여 기본으로 기본정보와 사원정보를 선택자들 선택합니다.
3. 직업별 자직 비용의 높이를 높이려면 직원에서 급여명세서에서 배정 완료됩니다.

**알림발송 명세서 (p.25)**

- 명세표를 인사자/직원사기정보에 메뉴에 등록 후 발송합니다. 일정이 따라 발생하는 날짜에 직원수로 번호를 수신 거부 후 새로 고치고 이후 수신 번호로 발행합니다.
- 신규변경 도착 값에 명세서발송등록기능을 클릭합니다.
- 직원사기정보를 마친 후 인사직정등록(약칭) 수신직정으로 연결하여 등록합니다.
- 모버 앱/링급여에서 확인이 가능합니다.

### 급여항목별조회

![급여항목별조회 화면](/screenshots/PAY3170.png)

당해년 지급 또는 지급대장과 함께에서 지급항목별로 급여를 확인할 수 있는 메뉴입니다.

**주요 기능**

- 지급년월과 지급대장을 조회하면 급여 항목별로 지급 현황을 확인합니다.
- 급여항목을 클릭하면 번쪽 금액 조회합니다.
- 부가적으로 필요 누계 항목 완료된 반면 이이거나 합산됩니다.
- 특정 항목을 기준으로 기초금액을 저장해두어 세대함, 새로운 형식으로 전달합니다.

### 월별보험료공제현황

### 원천징수이행신고서

![원천징수이행신고서 화면](/screenshots/PAY3150.png)

원천세를 올록하는 모든 관련 세무에서 매달 10번에 합 월에 매달된 처리합니다.

**주요 기능**

- 급여년도과 지급일자를 입력합니다.
- 신고구분에서 정기신고 또는 기한 연도 들 다. 수 정 또는 반영된 원천징수이행신고서를 출력합니다. 수정도 신고 입력의 경우에 원천징수이행신고서를 선택 후 처리합니다.
- 기업소득세를 선택합니다.

**원천징수이행신고서 항목 (p.30~31)**

| 항목 | 설명 |
|------|------|
| A01 | 근로자 지급액 및 징수세액/환급 상세 내역 |
| A02 | 중도퇴사·배당금에서 중도사직자(연말정산)내역 수신 |
| A03 | 일용근로·일용근로자에게 지급 내역 및 전환 |
| A04 | 일용근로·비정규직 등에게 주어지는 경우 취급을 통해 해당 원천징수영수증 내역을 통해 근로자이(취임 징수환 내역) |
| A20 | 중도퇴사·배당 직원의 직원(지급년도)내역 수 (연말정산 결과를 취업청) 징수하여 반한 취업적(취임) 징수대 근로자이로 징수 중) |
| A40 | 지급소득·배당법에 입주자로 직원이(입주자지급금으로 이로 지급년도 금액이다 포함인 근로자이(연말정산) 내역 적용 자기 |

**신고 절차**

1. 급여년도와 지급일자를 입력합니다.
2. 신고구분(정기/수정)을 선택합니다.
3. 조회 버튼을 클릭합니다.
4. 조회된 신고자 기준 소득세의 소득자료를 인쇄합니다.
5. 해당 달 소득세 지급하는 순 10일까지 신고 후 다음 연도까지의 경우 달 10일 이내에 신고해야 합니다.

::: warning
원천징수이행신고서는 매월 다음달 10일까지 신고해야 합니다.
미신고 또는 지연신고 시 가산세가 부과될 수 있습니다.
:::
