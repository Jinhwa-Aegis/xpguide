# 공통정보관리

> **메뉴 경로**: 인사/급여 > 공통정보관리

## 개요

인사/급여 업무에 필요한 공통 정보를 관리합니다. 급여 계산의 기반이 되는 직종·부서·직급 코드, 급여 항목, 세율, 보험료율, 퇴직 환경 설정, 상여율, 근로소득 간이세액표 등을 사전에 등록합니다. 이 메뉴의 설정값은 급여 계산, 연말정산, 퇴직정산 전반에 직접 영향을 미치므로 관련 법령 변경 시 즉시 갱신해야 합니다.

## 소메뉴 상세

### 인사코드정보

![인사코드정보 화면](/screenshots/PAY1010.png)

부서, 직급, 직종 등 인사 관련 코드를 등록하고 관리합니다.

**주요 기능**

- 등록된 직종, 직급, 부서에 신규 코드를 추가하고 기존 코드를 수정·삭제할 수 있습니다.
- 기존에 등록된 직종, 직급, 부서를 수정하면 해당 코드를 사용하는 모든 직원 정보에 즉시 반영됩니다.
- 필요한 직종에 세액계산 여부를 지정하여 비과세 적용 여부를 제어합니다.
- 소속 단지가 여러 개인 경우 관리자 메뉴에서 해당 직원의 공유 코드로 변환됩니다.

**사용 절차**

1. 조회 버튼을 클릭하여 현재 등록된 코드 목록을 불러옵니다.
2. 신규 코드를 추가하려면 빈 행에 코드명을 입력한 뒤 저장합니다.
3. 기존 코드를 수정할 경우 해당 행을 클릭하여 내용을 수정한 뒤 저장합니다.

::: tip
인사코드는 급여기본정보 등록 전에 반드시 완성해야 합니다. 코드가 없으면 급여 항목 배정이 불가합니다.
:::

### 급여기초정보

![급여기초정보 화면](/screenshots/PAY1020.png)

인사/기초정보에서 필요한 직급, 직종, 부서를 선고하여 급여 항목을 추가 및 수정할 수 있는 메뉴입니다.

**화면 구성**

| 영역 | 설명 |
|------|------|
| 상단 직원 목록 | 지급 항목·공제 항목을 설정할 직원을 선택 |
| 지급 항목 탭 | 기본급, 각종 수당 등 지급 항목 목록 |
| 공제 항목 탭 | 소득세, 4대 보험, 기타 공제 항목 목록 |
| 하단 요약 | 지급 합계 / 공제 합계 / 차인지급액 자동 계산 |

**주요 설정 내용**

- **지급 항목**: 지급/공제 항목별로 과세 여부 및 구분(급여·상여·기타)을 선택합니다. 항목은 최대 22개 항목으로 급여대장에 출력됩니다.
- **법령 연동**: 법령으로 정해지는 항목의 경우 직접 입력이 아닌 자동 계산으로 처리됩니다.
- **급여계산 순서**: 지급 항목 → 지급대장 순서대로 급여계산에 반영됩니다.
- **4대 보험 연동**: 건강보험·국민연금·고용보험·산재보험은 건강보험공단 EDI 신고 기반으로 자동 계산됩니다.

**주요 설정 항목별 설명**

| 번호 | 내용 |
|------|------|
| 1 | 과세 설정으로 급여 및 공제 항목을 구분하여 선택합니다 |
| 2 | 급여(선택)라고 선택한 항목은 보조적입니다. 항목 타입을 선택하세요 |
| 3 | 법령으로 정해지는 항목의 경우 직접 입력이 아닌 산정식으로 동작합니다 |
| 4 | 급여계산식의 메뉴+/지급대장의 순서대로 빈칸을 채워 넣습니다 |
| 5 | 급여계산식에서 사용 기준 50번, 연봉제는 15번을 기준으로 합니다 |
| 6 | 등록/조회 필요 여부이며 급여계산에서 반영되지 않습니다 |
| 7 | 기타옵션을 설정합니다 |
| 8 | 근태적인 시 건강보험료 자동적용 여부를 선택합니다 |
| 9 | 건강보험료 저촉 여부 중 확인할 항목을 선택합니다 |

**보험요율 자동 적용 관련**

- 보험요율이 미리 설정됩니다.
- 노동부에서 지정하는 기준으로 계산됩니다.
- 64세 이상 입사자(2014.1.1기준)인 경우 국민연금이 적용되지 않습니다.
- 지원 문의처: 관련 법령 변경은 고용노동부 또는 국민건강보험공단에 확인하시기 바랍니다.

**상세 옵션 설명 (p.5~6)**

| 번호 | 내용 |
|------|------|
| 7 | 부주건 설정수로 설정합니다 |
| 8 | 보험료율이 미리 설정됩니다 |
| 9 | 급여계산에서 시 시작 시 적용합니다 |
| 10 | 급여계산에서 사용 기준 50번, 연봉제는 15번을 기준으로 합니다 |
| 11 | 등록/조회 필요 여부로서 급여계산에서 반영되지 않습니다 |
| 12 | 기타옵션을 설정합니다 |
| 13 | 근태적인 시 건강보험료 자동적용 여부를 선택합니다 |

### 지급대장양식관리

![지급대장양식관리 화면](/screenshots/PAY1060.png)

지급대장에서 지급/공제 내역을 출력하는 방식으로 설정합니다.

**주요 기능**

- 지급대장 A 또는 B로 설정할 수 있습니다.
- 지급대장과 급여/공제 내역의 칸을 급여(지급/공제 순서기반으로 한 번에 선택 후 설정합니다.
  - 지급대장 내역 수는 사업장마다 순기를 달리 설정할 수 있습니다.
- 급여/공제정산 순서 각 기준으로 순기를 달리 설정하고 해당 메뉴에서 서식을 저장합니다.

**사용 절차**

1. 지급대장 A 또는 B를 선택합니다.
2. 지급대장과 급여·공제 내역의 칸을 선택 후 설정합니다.
3. 급여/공제정산 순서 기준으로 각 기능을 기준으로 설정합니다.

### 퇴직환경설정

![퇴직환경설정 화면](/screenshots/PAY1030.png)

퇴직충당금산에 대한 환경정보를 하는 메뉴로 단지별로 산 급여를 지급하는 방식에 따라 설정합니다.

**주요 설정 항목**

| 항목 | 내용 |
|------|------|
| 설치수 단정 | 퇴직 산정 시 작용 설정합니다 |
| 퇴직금 산정 방법 | 퇴직금 산정 시 퇴직충전금을 단지별 내역 방법으로 계산<br>- 수습 중 선이 퇴직금전금율 × (평균급여 × 30/12) × 재직기간(년) |
| 상여처리 | 3년 상여를 산정해 줍니다<br>- 3년 미만: 1년간 12월분 상여 ÷ 12월분 × 재직 기간으로 지급<br>- 3년 이상 경우에 재직중인 직원이 급여대장에 넣어도 체크 안 됩니다<br>- 3개월 합산: 상여처리를 제한한 경우에 해당 3개월 상여를 12개월×3개월 |

**참고 사항**

- 01~06번 코드는 기본 설정으로 퇴직기간이 없어 됩니다.
- 평균임금을 기준으로 세금 및 퇴직금 계산이 이루어집니다.
- 출력항목 설정이 필요합니다.
- 구조소는 공통복지에서 등록한 항목에서 이산 금여 항목을 선택합니다.
- 퇴직정산 완성 작성에서 이 구간산 퇴직(중간)정산산에 항목을 조 가해 변할 경우 출력항목/유형에 나타나지 않습니다.

**퇴직환경설정 사용 절차 (p.9)**

1. 01~06번 코드는 기본 설정으로 퇴직기간이 적합 합니다.
2. 평균임금을 기준으로 특목 설정이 필요합니다.
3. 출력항목을 선택합니다.
4. 구소소는 구성복지에서 등록한 항목에서 이산 급여 항목을 선택합니다.

### 상여율등록

![상여율등록 화면](/screenshots/PAY1040.png)

상여율을 등록하여 급여계산 시 상여금을 적용하는 메뉴입니다.

**주요 내용**

- 12개월로 나뉘어져 있으며 상여율에 따라 개월별 % 를 넣고 저장합니다.
- 예를 들어 1년 상여율이 300%이고, 매월 상여를 지급하는 단지의 경우 개월별 25%로 등록하면 급여계산 시 적용됩니다.

**사용 절차**

1. 상여를 지급할 월을 지정합니다 (1월~12월 개월별 입력).
2. 각 월에 지급할 상여율(%)을 입력합니다.
3. 저장 버튼을 클릭하여 등록합니다.

::: tip
상여율은 급여계산 시 자동으로 적용됩니다. 연간 상여율이 300%이고 매월 지급하는 경우 각 월에 25%씩 등록하면 됩니다.
:::

### 근로소득간이세액표

![근로소득간이세액표 화면](/screenshots/PAY1050.png)

소득세액표에 따라 세액을 확인할 수 있는 메뉴입니다.

**주요 내용**

- 국세청 고시에 따라 제정되는 근로소득 간이세액표를 조회합니다.
- 월 급여(세전급여/기본공제)와 부양가족 수(소득세×20+4세아이) 자녀 수의 비율(자녀수/20)으로 세액이 계산됩니다.
- 직전 정년 종류 이제 이미 자녀수가 확인 이 됩니다.
- 연말정산 작업 시 기준금액으로 활용됩니다. 연간 소득 금액이 100만원을 초과하지 않는 경우에 한해 급여 지급 20세 이하의 자녀수를 계산합니다.

**사용 방법**

- 급여 조회 상단에서 급여(세전) 금액 범위와 부양가족 수를 선택하면 해당 세액을 확인할 수 있습니다.
- 국세청 근로소득 간이세액표는 매년 국세청 고시에 따라 갱신됩니다.

## 주의사항

- 세율 및 보험료율은 관련 법령 변경 시 반드시 업데이트하세요.
- 급여기초정보 변경은 해당 월 급여 계산 전에 완료해야 합니다.
- 근로소득간이세액표는 매년 국세청 고시에 따라 갱신됩니다.
- 64세 이상 입사자(2014.1.1 기준)는 국민연금이 자동 제외됩니다.
- 퇴직환경설정 변경 후 반드시 해당 월 퇴직정산 계산을 재실행하여 결과를 확인하세요.
- 상여율은 연간 기준으로 등록하며, 예를 들어 연간 300%라면 월 25%씩 12개월로 분산 등록합니다.
