# 인사관리/조회

> **메뉴 경로**: 인사/급여 > 인사관리 / 인사/급여 > 인사조회

## 개요

관리사무소 직원의 인사 정보를 등록하고 관리합니다. 인사기본정보 등록부터 근태 관리, 입퇴사자 현황 조회, 인사기록카드 출력, 재직·경력증명서 발행까지 인사 관련 전 과정을 지원합니다. 인사기본정보에 등록된 직원 정보는 급여기본정보와 연동되어 급여 계산의 기초가 됩니다.

## 소메뉴 상세

### 인사기본정보

![인사기본정보 화면](/screenshots/PAY2010.png)

직원의 인사 기본 정보(성명, 주소, 부서, 직급 등)를 등록하고 관리합니다.

**화면 구성**

| 영역 | 설명 |
|------|------|
| 상단 직원 목록 | 등록된 직원 목록을 조회 (재직자 기준 기본 표시) |
| 기본 정보 탭 | 성명, 생년월일, 주민등록번호, 주소, 연락처 등 기본 정보 |
| 직급/부서 정보 | 부서, 직급, 직종, 입사일, 퇴사일 |
| 하단 버튼 | 저장, 삭제, 출력 |

**주요 기능 및 설명**

- 사원번호는 0001~1000의 범위 내에서 직접 입력하거나 자동 부여를 선택합니다.
- **별표(\*)**: 필수 입력 항목으로, 미입력 시 저장되지 않습니다.
- 주민등록번호를 입력하면 생년월일이 자동으로 채워집니다.
- 주소입력 시 우편번호를 조회하여 입력합니다.
- 직급·직종·부서는 인사코드정보에서 사전에 등록된 코드 목록에서 선택합니다.
- 퇴사일을 입력 시 퇴직정산 메뉴에서 퇴직금 계산이 자동으로 연동됩니다.
- **퇴직적립 시** 인사기본정보에 퇴직기간 및 직급 정보가 정확히 입력되어 있어야 합니다.
- 직급·직종·부서의 경우 코드가 변경되면 모든 연결 직원의 데이터에 반영됩니다.

**사용 절차**

1. 조회 버튼을 클릭하여 현재 직원 목록을 불러옵니다.
2. 신규 직원 등록 시 사원번호와 성명 등 필수 항목(\*)을 입력합니다.
3. 주민등록번호 입력 후 생년월일 자동 확인이 됩니다.
4. 직급·부서·직종은 드롭다운 목록에서 선택합니다.
5. 입사일과 퇴사일을 정확히 입력합니다.
6. 저장 버튼을 클릭합니다.

::: warning
퇴사 처리 시 퇴직정산 메뉴와 연계하여 반드시 퇴직정산을 실행하세요.
퇴사일 입력 후 급여 계산을 하면 해당 직원의 급여에 일할 계산이 적용됩니다.
:::

### 근태관리

![근태관리 화면](/screenshots/PAY2020.png)

사원별로 근태를 관리하고 입력된 정보를 급여명세서에 반영합니다.

**주요 기능 및 설명**

- 조회 버튼을 클릭합니다.
- 출력할 데이터를 복사합니다.
- 처리항목: 연가, 무급, 조퇴, 결근 등 입력하면 됩니다.
- 연간으로는 미근간으로 표시 포함됩니다. 단, 미근간일시간의 소환급에서 기준 값으로 환산하여 해당 급여 총 소환급에서 빼나가는 방식으로 반영됩니다.
- **일할계산 기준**: 공통복지나 급여(소환급)에서 지급 기준이 상이 합니다. 다, 미근간일시간의 소환급에서 기준 값으로 환산하여 해당 급여 총 소환급에서 빼나가는 방식으로 반영됩니다.
- 통상시간(법) 기본 근무시간의 근무시간: 기준이 적립됩니다.
- 통상시간(월) 시 사업장에서 통사시간의 근무시간: 기준이 됩니다.
- **알림**: 시간기록의 의한 시간 환산 관리는 이루어지지 않습니다.
- 미근간인 시간 간 그 급여기본정보에서 이전에 빠져나간 금액이 있으면, 급여명세서 상에서 이전에 빠진 금액이 반드시 기준이 합니다.

**근태 항목 종류**

| 항목 | 설명 |
|------|------|
| 연가 | 연간 근무한 시간으로 근무 시간 기준 |
| 무급 | 급여기본정보에 따른 근무 시간 기준 |
| 조퇴 | 급여계산에 자동으로 반영 |
| 결근 | 급여기본정보를 기준으로 급여에서 차감 |

::: tip
근태 데이터 입력은 급여 계산 전에 완료해야 정확한 급여가 산정됩니다.
통상시간(법) 기반의 일할계산을 원하는 경우 공통정보관리에서 기준 설정을 확인하세요.
:::

### 입퇴사자현황

![입퇴사자현황 화면](/screenshots/PAY2110.png)

입사자, 퇴사자의 간단한 인사기본정보를 조회 및 인쇄할 수 있는 메뉴입니다.

**주요 기능**

- 조회 버튼을 클릭합니다.
- 부서나 직급군별, 성명, 직급/직종을 선택하면 해당 입퇴사자를 선택하여 조회하면 해당 직원의 관련 인사기본정보, 입퇴사 날짜를 지정하여 직원 및 단체를 파악할 수 있습니다.

**조회 항목**

| 항목 | 설명 |
|------|------|
| 조회 기간 | 입사 또는 퇴사일 기준으로 기간 지정 |
| 부서/직급 | 특정 부서나 직급의 입퇴사자만 필터링 |
| 인사기본 정보 | 성명, 부서, 직급, 입사일, 퇴사일, 근무기간 |

**인사기본 정보 조회서 출력 내용**

출력 시 다음 내용이 포함됩니다.
- 성명, 부서, 직종, 직급
- 입사일, 퇴사일
- 고용 구분(건강보험 관련 정보)
- 전화번호, 주소

### 인사기록카드

![인사기록카드 화면](/screenshots/PAY2130.png)

직원의 인사기록카드를 조회하고 인쇄할 수 있는 메뉴입니다.

**주요 기능**

- 조회 버튼을 클릭합니다.
- 부서나 직급군별, 또는 직원을 선택하여 해당 직원의 관련 인사기록카드를 인쇄할 수 있습니다.
- 인사기록카드 조회 후 직원 전반 관련 내용을 파악합니다.

**인사기록카드 출력 내용**

인사기록카드에는 다음 정보가 포함됩니다.
- 기본 인적사항 (성명, 생년월일, 주소, 연락처)
- 학력 및 경력사항
- 자격증·면허 사항
- 부서·직급·직종 이력
- 입사일·퇴사일

::: tip
인사기록카드 메뉴는 급여기정산 메뉴에서 등록된 정보를 토대로 탭별 내용을 파악할 수 있습니다.
:::

### 증명서출력

![증명서출력 화면](/screenshots/PAY2140.png)

사원별 재직증명서, 경력증명서, 퇴직증명서를 출력합니다.

**주요 기능**

- 조회 버튼을 클릭합니다.
- 출력 서식을 선택합니다.
- 증명서 사원을 선택한 뒤 확인합니다.
- 직총별 여성 중도 입사인원에 따라 인쇄됩니다.

**증명서 종류**

| 종류 | 내용 |
|------|------|
| 재직증명서 | 현재 재직 중인 직원의 재직 사실 증명 |
| 경력증명서 | 재직 또는 퇴직 직원의 근무 경력 증명 |
| 퇴직증명서 | 퇴직한 직원의 퇴직 사실 및 퇴직일 증명 |

**출력 시 포함 내용**

재직증명서는 다음 항목을 포함합니다.
- 직원 성명, 부서, 직급
- 입사일, 재직 기간
- 발급일, 발급 기관명, 관리자 서명란

::: tip
증명서 발급 전 인사기본정보에 모든 사항이 정확히 입력되어 있는지 확인하세요.
퇴직증명서는 인사기본정보에 퇴사일이 입력된 직원에게만 발급할 수 있습니다.
:::

## 주의사항

- 입사 등록 후 급여기본정보를 함께 등록해야 급여 계산이 가능합니다.
- 퇴사 처리 시 퇴직정산 메뉴와 연계하여 처리하세요.
- 증명서 출력 전 인사 정보가 정확한지 확인하세요.
- 주민등록번호는 반드시 실제 번호를 정확히 입력해야 소득세 계산이 올바르게 이루어집니다.
- 근태 데이터는 급여 계산 전에 입력을 완료해야 급여에 정상 반영됩니다.
- 부서·직급 코드 변경 시 해당 코드를 사용하는 모든 직원 데이터에 영향이 있으므로 신중하게 처리하세요.
