# 수납처리

> **메뉴 경로**: 수납 > 수납처리

## 개요

부과된 관리비를 수납(납부) 처리합니다. 일반수납, 부분수납, 지점이체수납, 수납취소, 연체료수정 후 수납, 수납전표연계, 자동수납오류처리, 수납일자 변경, 가수금관리 등 다양한 방식의 수납 업무를 처리합니다.

---

## 일반수납

![수납처리 화면](/screenshots/RECP1010.png)

### 기능 설명

관리비 등의 납부 내역을 등록 및 관리 하는 메뉴입니다.

### 사용 방법

1. **수납일/수납기간 입력**: 수납일이나 수납기간을 입력합니다.
2. **미납 세대 선택**: 수납할 세대를 선택합니다. 수납하는 미납으로 진행됩니다.
3. **부과 내역 확인**: 조회된 내역에서 해당 세대의 수납 내역을 확인합니다. 수납내역에 아래쪽의 1번~3번 수납처리 방법이 지정됩니다.
4. **수납 처리**: 조회된 내역에서 수납금액을 확인하고, 해당 3~2 일 않은 수납하는 선택했던 것을 처리합니다.

> **참고**: 조회된 내역에서 수납금액 연산으로 핀 경우, 영제처리 요금가격 수납내역에 관련 한 건의 내역에 반드시 수납처리 합니다.

## 부분수납

### 기능 설명

납금완료된 납부 해야 할 금액보다 작은 금액을 사용합니다.

### 사용 방법

1. **수납일/수납기간, 동호를 조회** 후 수납금액 연계를 선택합니다. 분납이자 에러이가 나타나면 확인, 확인버튼을 클릭한 현, 분납처리가 됩니다.
2. **차액처리 구분**:
   - **차액처리 (일반처리)**: 부과비/일반(세대이의부금) 우선 처리 / 잔액: 부과비+잔액 업은 단위로 분납처리 납부 우선 처리
   - **미납선택 (입금처리)**: 부과+잔액 업은 입력된 환경 분납 처리 / 미납선택은 미납선택으로 처리

## 지점이체수납

### 기능 설명

자동이체에 의할 수납 차처시 사용합니다. (분할 자동이체는 처음 수납 처리 방식입니다.)

### 사용 방법

1. **수납방법 선택**: 지점이체를 선택합니다.
2. **수납방법 확인**: 납부방법별로 처리 시 자동 시작 입력됩니다.
3. **장수납금 확인**: 수납금액의 선택 목록을 확인합니다.
4. **수납처리 버튼 클릭**: 수납처리를 완료합니다.

---

## 수납취소

### 기능 설명

입금처리한 수납하여 내역을 취소할 수 있습니다.

### 사용 방법

1. **수납취소 할 동호를 수납세대의 미납 > 수납내용으로 변경**: 해당 수납내용이 조회됩니다.
2. **조회된 내역에서 취소할 항목을 선택**합니다.
3. **수납취소 버튼을 클릭**합니다.
4. **취소메시지 창을 확인**하여 수납취소를 완료합니다.

---

## 연체료수정후수납처리

### 기능 설명

수납처리 전에 연체료 금액을 수정하여 수납처리를 할 수 있습니다.

### 사용 방법

1. **연체료수정 버튼을 클릭**합니다.
2. **선택 및 동호를 입력** 후 조회합니다.
3. **미납 연체료 및 연체료를 수정** 후 저장합니다.
4. **연체료수정완료 후** 수납처리 시 수정된 연체료가 반영되어 수납처리가 가능하게 됩니다.

> **\* 분납처리는 수납할 수 있습니다.**

## 수납전표연계

![수납전표연계 화면](/screenshots/RECP1023.png)

### 기능 설명

수납처리된 내역을 회계전표번호로 회계자료를 등록 하는 메뉴입니다.

### 수납전표연계 절차 (p7-9)

1. **수납기간을 조회**합니다.
2. **전표 등록할 수납일자를 선택** 체크합니다.
3. **전표버튼을 클릭**하여 전표를 등록합니다.

#### 전표연계 시 주의사항

- 수납일자가 등록되어 있지 않은 경우에는 전표처리가 안되며, 전체로 전역된 연계 수납일자에 연결 완료 해야만 회계로 연결 확인이 됩니다.
- 수납 취소/재처리 발생할 경우 수납건에 대한 수납전표연계에서 연계처리가 안 되는 건의 오류가 발생하지 않기 위해서는 반드시 관리가 필요합니다.

#### 출력 양식

- 출력옵션을 선택하여 인쇄내역을 클릭합니다.
- 출력옵션에 따라 수납합계서, 수납대장으로 수납내역이 정리됩니다:
  - **대체 전표** (복식부기 전표 양식)
  - **일자별 관리비 수납 합계서**
  - **[관리비] 수납대장**

## 자동수납오류처리

![자동수납오류처리 화면](/screenshots/RECP1020.png)

### 기능 설명

자동이체 내역(카드, 은행, 아파트아이 등) 수납처리의 오류 건을 확인 및 처리할 수 있습니다.

### 오류 구분별 내역

- **정상입금**: 미입금완료 처리 내역
- **미입금**: 미입금완료 처리 내역
- **가수금**: 항목된 이의 미이 자리에서 구분되는 가수금으로 처리되는 내역
- **차이금액**: 입금으로 미입금수납하는 처리되는 내역

### 처리 방법

1. **수납일을 입력** 후 조회하여 내역을 확인 및 처리할 수 있습니다.
2. **수납일을 확인** (아파트아이, 카드, 운영사와 관련된 자동이체 내역 등이 표시됩니다).
3. 각 오류 구분에 따라 처리합니다.

## 수납일자(은행)변경

![수납일자(은행)변경 화면](/screenshots/RECP1030.png)

### 기능 설명

수납처리 하는 경우 은행 또는 은행일을 일을 변경하는 경우 시 활용하는 메뉴입니다.

### 사용 방법

1. **수납처리 한 수납기 또는 수납일자를 입력** 후 조회합니다.
2. **변경일자는 은행 변경기간을 선택**합니다.
3. **조회하여 해당 변경할 세대 선택 후 변경할 사대 선택** 후 변경에 체크합니다.
4. **변경 버튼을 클릭**합니다.
5. **변경 알림시지 창이 나타나면 변경 후 변경을 완료**합니다.

## 가수금관리

![가수금관리 화면](/screenshots/RECP1051.png)

### 기능 설명

입금된 관리비 금액이 초과가 있는 경우 초과금액으로 입금에 사용을 관리하는 메뉴입니다.

### 가수금 처리 방법

1. **조회 후 신규 버튼을 클릭**하여 가수금으로 등록 되어 있는 관련 총청산 잔금을 클릭합니다.
2. **부과완료 처리되어 신규금액, 사용금, 소유금, 등의 사용 구분을** 처리할 수 있습니다.

#### 부과완료 처리의 주의사항

- **가수금을 사용하려고 한다**: 입력금이 부과조건에 따라서 남은 입금을 유지 처리 된 금액이나 가수금으로 처리될 경우에 있는 금액 입금은 남은 금액이 처리 됩니다.
- **가수금 사용하지 않으면 사항**: 수납(수납~입주자+수납일자)에서 사용하는 기능입니다.
