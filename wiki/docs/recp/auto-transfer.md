# 자동이체수납

> **메뉴 경로**: 수납 > 자동이체수납

## 개요
은행 자동이체를 통해 수납된 관리비를 일괄 처리합니다. 수납내역조회와 자동납부신청자조회 기능을 제공합니다.

## 소메뉴 상세

### 수납내역조회

| 소메뉴 | 설명 |
|--------|------|
| 월별 자동이체 | 월별 자동이체 내역 |
| 월별 기타수납 | 월별 기타수납 내역 |
| 월별 이중수납 | 이중수납 내역 |

### 자동납부신청자조회

| 소메뉴 | 설명 |
|--------|------|
| 월별 신청자 상세검색 | 자동납부 신청자 검색 |
| 이중 신청자 전체 | 이중 신청자 조회 |
| 이중 출금 신청자 | 이중 출금 조회 |

## 사용 방법
1. 수납 > 자동이체수납 메뉴 진입
2. 자동이체 결과 파일 업로드
3. 수납 처리 실행
4. 처리 결과 확인
