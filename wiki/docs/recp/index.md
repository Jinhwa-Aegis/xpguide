# 수납

> **메뉴 경로**: 수납

## 개요
수납 메뉴는 부과된 관리비의 수납(납부) 처리, 수납 조회, 미납 관리, 자동이체 수납 등의 업무를 처리합니다.

| 수납전표연계 | 수납과 회계전표 연계 |
| 자동수납오류처리 | 자동수납 오류 건 처리 |
| 수납일자(은행)변경 | 수납일자 변경 |

## 업무 흐름
1. [부과마감](/impo/closing) 후 수납 업무 시작
2. [자동이체수납](/recp/auto-transfer) 처리
3. [수납처리](/recp/process) (개별/창구 수납)
4. [수납조회](/recp/inquiry)로 확인
5. [미납조회](/recp/overdue)로 미납 세대 관리
