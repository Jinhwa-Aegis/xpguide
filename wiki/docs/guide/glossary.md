# 용어 사전

XpERP에서 사용하는 주요 용어를 쉽게 설명합니다. 회계·아파트 관리 용어가 낯설다면 이 페이지를 참고하세요.

::: tip 찾는 방법
브라우저에서 `Ctrl + F`를 눌러 원하는 용어를 검색하세요.
:::

## ㄱ

| 용어 | 쉬운 설명 | 예시 |
|------|----------|------|
| **가수금** | 누구에게서 온 돈인지 아직 확인되지 않은 입금. 확인 후 해당 세대 수납으로 처리 | 통장에 150,000원 입금됐는데 입금자를 모를 때 |
| **감가상각** | 시설·장비의 가치가 시간이 지나면서 줄어드는 것을 회계에 반영 | 엘리베이터 교체 비용을 매년 나눠서 비용 처리 |
| **감면** | 특정 조건(복지, 장애 등)에 해당하는 세대의 요금을 줄여주는 것 | 기초생활수급자 전기요금 감면 |
| **검침** | 전기, 수도, 가스 등 계량기의 수치를 읽어 사용량을 산정하는 작업 | 매월 1~2주차에 각 세대 계량기를 읽음 |
| **결산** | 일정 기간의 수입과 지출을 정리하여 재무 상태를 확정하는 작업 | 12월 말에 연간 결산을 실행 |
| **계정과목** | 돈의 이동을 분류하는 이름표. 수입/지출/자산/부채 등으로 구분 | 보통예금, 관리비수입, 수선유지비, 미수금 등 |
| **고지서** | 세대에 관리비를 알려주는 문서 | 매월 발송하는 관리비 고지서 |
| **공용부분** | 아파트에서 모든 입주민이 함께 사용하는 공간 | 복도, 엘리베이터, 주차장, 놀이터 |
| **관리비** | 아파트 공용부분의 유지관리 및 운영에 필요한 비용 | 일반관리비, 청소비, 경비비, 수선유지비 등 |
| **관리비부과** | 산정된 관리비를 각 세대에 청구하는 작업 | 매월 3주차에 관리비를 부과 |
| **관리비예치금** | 입주 시 납부하는 보증금 성격의 관리비. 전출 시 환급 | 선수관리비라고도 함 |
| **관리외수입** | 관리비 외에 발생하는 수입 | 주차장 외부 개방 수입, 광고 수입 |
| **귀속 연월** | 해당 급여나 세금이 속하는 기간 | 3월 급여의 귀속 연월은 2026년 3월 |

## ㄴ

| 용어 | 쉬운 설명 | 예시 |

| **동별** | 아파트 동(棟) 단위로 구분하여 처리하는 방식 | 101동, 102동별 관리비 집계 |

## ㅁ

| 용어 | 쉬운 설명 | 예시 |
|------|----------|------|
| **마감** | 해당 월(또는 연도)의 회계를 끝내고 확정하는 것. 마감 후 수정 제한 | 3월 31일에 3월 회계 마감 |
| **미납** | 납부 기한까지 관리비를 납부하지 않은 상태 | 3월 관리비를 4월 말까지 안 낸 경우 |
| **미수금** | 부과했지만 아직 받지 못한 돈 | 관리비를 부과했는데 아직 안 낸 금액 |
| **미지급금** | 서비스를 받았지만 아직 지급하지 않은 돈 | 청소 용역비를 아직 안 준 경우 |
| **민원** | 입주민이 관리사무소에 요청하는 사항 | 수도 고장, 주차 민원, 소음 민원 |

## ㅂ

| 용어 | 쉬운 설명 | 예시 |
|------|----------|------|
| **바우처 할인** | 에너지 바우처(복지 대상) 금액을 관리비에서 차감 | 한국지역난방공사 에너지바우처 할인 |
| **발생금액** | 해당 항목의 총 비용 (세대별 배분 전 금액) | 공용전기료 총액 500만원 |
| **별도금액** | 자동 계산이 아닌 수동으로 추가/차감하는 금액 | 인테리어 공사 보증금 추가 부과 |
| **보조부원장** | 특정 계정과목의 세부 거래 내역을 기록한 장부 | 미수금 보조부원장 → 세대별 미수금 내역 |
| **보통예금** | 은행 보통예금 계좌. 관리비 수납·지출에 사용 | 관리비 수납 통장 |
| **부과** | 관리비를 산정하여 각 세대에 청구하는 것 | 매월 3주차에 부과 처리 |
| **부과그룹** | 관리비를 다르게 적용하는 세대 묶음 | 일반세대, 상가세대 등 그룹 구분 |
| **부과기초작업** | 월별 부과의 기본 틀(기간·납부기한)을 설정하는 작업 | 부과처리 전에 반드시 실행 |
| **부가세(VAT)** | 부가가치세. 물건/서비스 가격의 10% | 100만원 공급가에 부가세 10만원 |
| **분개** | 거래를 차변과 대변으로 나누어 기록하는 것 | 관리비 수납: 차변(보통예금), 대변(미수금) |
| **분리검침** | 공용/전용 사용량을 분리하여 검침하는 것 | 상가와 주거 분리 검침 |
| **분리부과** | 입주자/사업자/소유자별로 관리비를 나눠 부과하는 것 | 상가 입주 단지에서 2분리 또는 3분리 |

## ㅇ

| 용어 | 쉬운 설명 | 예시 |
|------|----------|------|
| **연말정산** | 1년간 원천징수한 소득세를 정산하여 환급/추징하는 절차 | 매년 1~2월 직원 연말정산 |
| **연체료** | 납부기한을 넘겨 추가로 부과되는 금액 | 1개월 연체 시 연체율 적용 |
| **예산** | 회계연도의 수입/지출 계획 | 연초에 수립하는 연간 예산 |
| **예치금** | 입주 시 납부하는 관리비 보증금 (= 관리비예치금) | 전출 시 환급 |
| **원천징수** | 급여 지급 시 소득세를 미리 떼어 국가에 납부하는 것 | 매월 급여에서 소득세 공제 |
| **월마감** | 해당 월의 회계를 종료하고 수정을 제한하는 것 | 마감 후 전표 입력/수정 불가 |

| **입주** | 새로운 소유자/임차인이 세대에 입주하는 것 | 신규 입주자 등록 |
| **일계표** | 하루 동안의 모든 거래를 계정과목별로 집계한 표 | 일별 차변/대변 합계 확인 |
| **일할 계산** | 월 중간 입주/퇴거 시 날짜 비율로 금액을 나누는 것 | 15일 거주 = 관리비 × 15/30 |

## ㅈ

| 용어 | 쉬운 설명 | 예시 |
|------|----------|------|
| **자동분개** | 자주 사용하는 전표 패턴을 미리 등록해 자동으로 불러오는 기능 | 매월 동일한 용역비 전표를 자동 생성 |

| **장기수선계획** | 아파트 공용부분의 장기적인 수선 계획 | 외벽 도색(5년 주기), 엘리베이터 교체(15년) |
| **장기수선충당금** | 장기수선계획에 따른 수선비용을 충당하기 위해 매월 적립하는 금액 | 세대별 매월 일정액 적립, 전출 시 환급 |
| **장부** | 거래 내역을 기록한 회계 문서의 총칭 | 총계정원장, 보조부원장, 현금출납장 등 |

| **전자결재** | 문서의 결재를 전자적으로 처리하는 시스템 | 지출결의서 → 결재 → 이체 실행 |
| **전출** | 기존 소유자/임차인이 세대에서 퇴거하는 것 | 전출 시 중간정산 필요 |
| **전표** | 회계 거래를 기록하는 문서. 차변과 대변으로 구성 | 입금전표, 출금전표, 대체전표 |

| **조견표** | 관리비 산정 기준표. 면적, 세대수 등에 따른 부과 기준 | 주차비 조견표: 1대당 월 30,000원 |
| **중간정산** | 월 중간에 전출 시 거주 일수만큼 관리비를 정산하는 것 | 15일 거주 시 관리비 절반 정산 |
| **지급명세서** | 직원에게 지급한 소득 내역을 세무서에 신고하는 문서 | 근로소득 지급명세서 (다음 해 3월 10일까지 제출) |
| **지출결의서** | 지출을 승인받기 위한 결재 문서 | 수선비 지출결의서 → 결재 → 이체 |

## 차변과 대변 쉽게 이해하기

모든 회계 거래는 **차변(왼쪽)**과 **대변(오른쪽)**으로 기록합니다.

| 상황 | 차변 (받는 쪽) | 대변 (주는 쪽) |
|------|---------------|---------------|
| 관리비를 부과했을 때 | 미수금 (받을 돈 생김) | 관리비수입 (수입 발생) |
| 관리비를 수납했을 때 | 보통예금 (통장에 돈 들어옴) | 미수금 (받을 돈 줄어듦) |
| 용역비를 지급했을 때 | 관리용역비 (비용 발생) | 보통예금 (통장에서 돈 나감) |
| 사무용품을 구매했을 때 | 소모품비 (비용 발생) | 보통예금 (통장에서 돈 나감) |

::: tip 핵심 규칙
**차변 합계 = 대변 합계**가 항상 같아야 합니다. 같지 않으면 전표를 저장할 수 없습니다.
:::
