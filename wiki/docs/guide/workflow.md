# 월간 업무 흐름

아파트 관리사무소 경리(회계·사무 담당자)의 매월 반복되는 핵심 업무 흐름을 설명합니다.

::: tip
단지별로 차이가 있으며 일반적인 예시 사항이므로, 각 단지 상황에 맞추어 업무를 처리하세요.
:::

## 월간 업무 사이클

하나의 관리비 청구 주기는 아래 순서로 진행됩니다.

```
검침 → 부과 → 수납 → 회계처리
```

실제 업무에서는 **전월 수납·회계 마감**과 **당월 검침·부과**가 동시에 진행됩니다. 아래 주별 업무 가이드를 함께 참고하세요.

## 주별 업무 가이드

| 주차 | 주요 업무 | 관련 메뉴 |
|------|----------|-----------|
| **1주** | 전월 말일자 수납 마감, 전월 회계전표 정리, 체납세대 문서발송(문자·전화·내용증명·지급명령 등), 검침 시작, 은행 업무, 결재 준비 | [수납처리](/recp/process), [전표관리](/acct/voucher), [미납조회](/recp/overdue), [검침](/insp/), [아파트뱅크](/bank/), [Xp전자결재](/appr/) |
| **2주** | 검침 완료, 재무제표 1차 작성, 부과 준비, 은행 잔액증명서 확인, 은행 업무 | [검침](/insp/), [결산관리](/acct/settlement), [기초정보](/impo/basic), [아파트뱅크](/bank/) |
| **3주** | 부과 완료 → 전산업체 전송, 재무제표 2차 완성 → 대표회의 결재, 고지서·부과내역서 배부, 급여 준비(4대보험 확인 등), 은행 업무 | [부과처리](/impo/process), [부과마감](/impo/closing), [관리비부과조회](/impo/inquiry), [급여관리](/pay/salary), [아파트뱅크](/bank/) |
| **4주** | 급여 지급, 당월 수납 시작(자동이체 등), 말일 결재 준비, 은행 업무 | [급여관리](/pay/salary), [수납처리](/recp/process), [Xp전자결재](/appr/), [아파트뱅크](/bank/) |
| **5주** | 말일 결재, 수납 계속, 은행 업무 | [수납처리](/recp/process), [Xp전자결재](/appr/), [아파트뱅크](/bank/) |

## 단계별 상세

### 1단계: 검침 (1~2주차)

각 세대의 계량기를 검침하여 사용량을 산정합니다.

- [전기검침](/insp/electric) - 전기 사용량 입력
- [수도검침](/insp/water) - 수도 사용량 입력
- [온수검침](/insp/hot-water) - 온수 사용량 입력
- [가스검침](/insp/gas) - 가스 사용량 입력
- [난방검침](/insp/heating) - 난방 사용량 입력

### 2단계: 부과 (3주차)

검침 데이터를 기반으로 관리비를 산정하고 부과합니다.

1. [기초정보](/impo/basic) 확인
2. [부과처리](/impo/process) 실행
3. [관리비부과조회](/impo/inquiry)로 확인
4. [부과마감](/impo/closing) 처리

### 3단계: 수납 (4주차~익월 1주차)

부과된 관리비를 수납 처리합니다.

1. [자동이체수납](/recp/auto-transfer) 처리
2. [수납처리](/recp/process) (개별 수납)
3. [수납조회](/recp/inquiry)로 확인
4. [미납조회](/recp/overdue)로 미납 세대 확인

### 4단계: 회계처리 (익월 1~2주차)

수납 내역을 회계에 반영합니다.

1. [전표관리](/acct/voucher) - 전표 생성/승인
2. [장부관리](/acct/ledger) - 장부 확인
3. [결산관리](/acct/settlement) - 재무제표 작성

## 업무 영역별 분류

| 영역 | 업무 내용 | 비고 |
|------|----------|------|
| **사무업무** | 문서접수, 문서발송, 기안, 지출결의서, 공고문 작성, 체납세대 내용증명 발송 등 | |
| **인사업무** | 입사, 퇴사, 4대보험 처리, 연말정산 등 | 자치관리는 자체 관리, 도급회사는 위탁사 처리 |
| **민원업무** | 각종 민원 관련 업무 처리 등 | |
| **회계업무** | 수납, 급여, 회계, 재무제표 작성, 부과, 전표철 작성, 세금계산서 관련(발행, 부가세·법인세 신고) 등 | |
| **기타업무** | 체납세대관리, 전출 관련(중간관리비 정산, 장기수선충당금, 선수관리비 계산), 인테리어 공사 관련, 물건 구매 등 | 커뮤니티센터가 있는 단지는 커뮤니티 업무 포함 |

## 지급명세서 제출 일정

::: warning 기한을 반드시 준수하세요
지급명세서 제출 기한을 초과하면 가산세가 부과됩니다.
:::

| 소득 구분 | 제출 시기 |
|-----------|----------|
| 근로·퇴직·사업소득·종교인소득·연금계좌 | 다음 연도 **3월 10일** |
| 근로소득 간이지급명세서 | 반기 마지막 달의 다음 달 말일 (1~6월분: **7/31**, 7~12월분: **1/31**) |
| 일용근로소득 | 지급일이 속하는 달의 **다음 달 말일** |
| 이자·배당·기타소득 등 그 밖의 소득 | 지급일이 속하는 연도의 **다음 연도 2월 말** |
