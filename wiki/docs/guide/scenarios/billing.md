# 관리비 부과

> **소요 시간**: 약 30분 | **난이도**: ★★★ | **주기**: 매월 3주차

검침 데이터와 기초정보를 바탕으로 각 세대에 관리비를 산정·부과하는 전체 프로세스입니다.

## 전체 흐름

```
1. 사전 확인        (5분)  → 기초정보, 검침 완료 여부
2. 부과기초작업      (3분)  → 부과 기간·수납 기간 설정
3. 별도금액 등록     (5분)  → 추가/차감 금액 입력
4. 관리비 부과처리   (10분) → 발생금액 입력·계산
5. 총괄표 확인      (3분)  → 전월 대비 확인
6. 부과마감         (2분)  → 확정 처리
7. 고지서 출력/전송  (2분)  → 인쇄 또는 전자고지
```

---

## 1단계: 사전 확인 (5분)

**왜 필요한가?** 검침이 완료되지 않으면 부과 금액이 정확하지 않고, 기초정보가 틀리면 세대별 금액이 잘못 산정됩니다.

### 체크리스트

- [ ] 모든 검침(전기·수도·온수·가스·난방)이 완료되었는가?
- [ ] 신규 입주·전출 세대가 반영되었는가?
- [ ] 부과 기준 단가 변경 사항이 있는가?
- [ ] 부과제외 세대 등록이 최신 상태인가?

### 확인 방법

| 확인 항목 | 메뉴 경로 | 화면 코드 |
|-----------|-----------|-----------|
| 검침 완료 | [검침](/insp/) > 각 검침 메뉴에서 조회 | INSP1120 등 |
| 기초정보 | [부과](/impo/) > [기초정보](/impo/basic) | IMPO1020 |
| 부과제외세대 | [부과](/impo/) > [기초정보](/impo/basic) > 부과제외세대등록 | IMPO1040 |

---

## 2단계: 부과기초작업 (3분)

> **메뉴 경로**: 부과 > [부과처리](/impo/process) > 부과기초작업 (IMPO2005)

이번 달 부과의 기본 틀을 만드는 단계입니다.

### 절차

1. **부과기초작업** 화면 진입
2. **행추가** 버튼 클릭
3. 아래 항목 입력:
   - **부과 기간**: 당월 1일 ~ 말일 (예: 2026-03-01 ~ 2026-03-31)
   - **수납 기간**: 부과 월 포함 수납 허용 기간
   - **납부 기한**: 보통 다음 달 말일
4. **저장** 클릭

::: warning 주의
부과기초작업을 먼저 완료해야 관리비부과처리를 진행할 수 있습니다. 순서를 건너뛰면 부과처리 화면에서 데이터가 조회되지 않습니다.
:::

---

## 3단계: 별도금액 등록 (5분)

> **메뉴 경로**: 부과 > [부과처리](/impo/process) > 별도금액등록 (IMPO2010)

자동 계산되지 않는 추가/차감 금액을 세대별로 입력합니다.

### 언제 사용하나?

- 개별 세대 추가 부과 (예: 인테리어 공사 보증금)
- 관리비 감면 (예: 경로우대 할인)
- 전월 오류 조정 (예: 과다 부과분 차감)

### 절차

1. 별도금액등록 화면 진입
2. 해당 세대의 동·호 선택
3. 부과항목 선택 후 금액 입력
   - **(+)** 추가 부과
   - **(-)** 차감
   - **(=)** 기존 금액 대체
   - **(/)** 나누기 적용
4. **저장** 클릭

::: tip 대량 등록
여러 세대를 한 번에 처리하려면 **엑셀 업로드**를 사용하세요. 엑셀 양식: 부서코드, 동, 호수, 항목코드, 정보, 금액 비고. 파일 형식은 **Excel 97-2003(.xls)** 이어야 합니다.
:::

## 4단계: 관리비 부과처리 (10분)

> **메뉴 경로**: 부과 > [부과처리](/impo/process) > 관리비부과처리 (IMPO2020)

실제 관리비를 계산하고 세대별 금액을 확정하는 핵심 단계입니다.

### 절차

1. 부과 연월 선택
2. 각 부과항목별 **발생금액** 입력
   - 발생금액 = 해당 항목의 총액 (예: 공용전기료 총액)
3. **계산** 버튼 클릭
   - 시스템이 자동으로 세대별 금액을 배분 (면적·세대수 기준)
   - 계산식: 세대별 단가 = 발생금액 ÷ 총 부과 대상
4. 하단 그리드에서 **세대별 금액** 확인
   - 이상 금액이 있으면 직접 수정 가능
5. **저장** 클릭

### 확인 포인트

- [ ] 각 항목의 발생금액이 전월과 크게 차이나지 않는가?
- [ ] 세대별 배분 결과가 합리적인가?
- [ ] 별도금액(3단계)이 정상 반영되었는가?

::: warning 분리부과가 필요한 경우
입주자/사업자/소유자별로 관리비를 나눠야 하는 단지는 **분리부과처리**(IMPO2030)를 추가로 실행하세요. 2분리(입주자/사업자) 또는 3분리(입주자/사업자/소유자) 선택이 가능합니다.
:::

## 5단계: 총괄표 확인 (3분)

> **메뉴 경로**: 부과 > [부과처리](/impo/process) > 총괄표 (IMPO2040)

부과 결과를 전월과 비교하여 이상 여부를 최종 점검합니다.

### 절차

1. 총괄표 화면 진입
2. 당월 부과 총액 vs 전월 부과 총액 비교
3. 항목별 증감 내역 확인

### 확인 포인트

- [ ] 전월 대비 총액 변동이 합리적인가? (계절 요인 등 고려)
- [ ] 특정 항목이 급격히 변동하지 않았는가?
- [ ] 세대수 변동(전출입)이 정확히 반영되었는가?

---

## 6단계: 부과마감 (2분)

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) (IMPO9010)

부과 내역을 확정합니다. **마감 후에는 수정이 불가**합니다.

### 절차

1. 부과마감 화면 진입
2. 부과 연월 선택
3. 항목별 금액 최종 확인
4. **비율** 버튼 클릭 (항목별 비율 확인)
5. **마감** 버튼 클릭

::: danger 되돌리기 불가
부과마감 후에는 수정할 수 없습니다. 반드시 총괄표와 세대별 금액을 확인한 뒤 마감하세요. 마감 후 오류를 발견하면 다음 달 **별도금액**으로 조정해야 합니다.
:::

---

## 7단계: 고지서 출력/전송 (2분)

마감이 완료되면 세대에 고지서를 전달합니다.

### 방법 1: 인쇄 출력

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) > 출력의뢰(부과마감)

1. 출력의뢰 화면에서 동·호 범위 입력
2. 출력 요청 실행

### 방법 2: 전자고지서

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) > 전자고지서 (IMPO9011)

1. 전자고지서 화면 진입
2. 부과 연월 확인
3. 내용 확인 후 **전송** 실행

## 자주 발생하는 문제

### 부과처리 화면에 데이터가 안 나와요

**원인**: 부과기초작업(2단계)이 완료되지 않았습니다.
**해결**: 부과기초작업에서 해당 월의 부과 기간을 먼저 등록하세요.

### 특정 세대가 부과 목록에 없어요

**원인**: 부과제외세대로 등록되어 있거나, 입주 처리가 안 된 세대입니다.
**해결**: [기초정보](/impo/basic) > 부과제외세대등록(IMPO1040)에서 확인하고, [입주처리](/occp/move-in)를 확인하세요.

### 전월보다 금액이 크게 달라요

**원인**: 검침값 오류, 단가 변경, 별도금액 입력 오류 등
**해결**: 총괄표(IMPO2040)에서 항목별로 전월 대비 비교하여 변동 원인을 파악하세요.

### 마감 후 오류를 발견했어요

**원인**: 마감 후에는 부과 내역 수정이 불가합니다.
**해결**: 다음 달 부과 시 **별도금액등록**(IMPO2010)에서 (+)추가 또는 (-)차감으로 조정하세요.
