# 장기수선계획 관리

> **소요 시간**: 검토 약 2시간 / 집행 건당 약 30분 | **난이도**: ★★★ | **주기**: 연 1회 검토 + 수시 집행

공동주택관리법에 따라 의무적으로 수립·관리해야 하는 장기수선계획을 검토하고, 수선 공사를 집행하는 프로세스입니다.

::: info 충당금 관리는 경리 업무입니다
장기수선충당금 적립 현황 관리와 전출 시 충당금 정산은 경리 담당이 처리합니다. → [장기수선충당금 관리](/guide/scenarios/reserve-fund)
:::

## 전체 흐름

### 계획 수립 및 검토 (연 1회)

```
1. 장기수선계획 수립   (외부 서비스) → 수선 항목·일정·비용 산정
2. 수선 항목 확인      (30분)        → 시설별 수선 주기·금액 검토
3. 입주자대표회의 의결 (별도)        → 계획 승인·변경 의결
4. 보고서 출력         (5분)         → 의결 결과 보관
```

### 수선 공사 집행 (수시)

```
1. 입주자대표회의 승인    (별도)  → 집행 전 의결
2. 공사비 전표 처리       (15분)  → 회계 > 전표관리 입력
3. 예산 집행 확인         (5분)   → 예산관리 잔액 확인
4. 집행 이력 관리         (10분)  → 이력 기록·보관
```

---

## 법적 의무 사항

::: tip 공동주택관리법 의무 규정
- **수립 의무**: 300세대 이상 또는 승강기가 있는 공동주택은 장기수선계획을 의무적으로 수립해야 합니다.
- **정기 검토**: **3년마다** 계획을 검토·조정해야 합니다.
- **충당금 적립**: 관리비와 함께 세대별로 매월 적립합니다.
- **사용 승인**: 수선 공사 집행 전에 **입주자대표회의 의결**이 필요합니다.
:::

---

## 1단계: 장기수선계획 수립 (외부 서비스 연동)

> **메뉴 경로**: [장기수선계획](/afms/)

메뉴 선택 시 **외부 서비스 페이지로 이동**합니다. 외부 서비스에서 아래 항목을 작성합니다.

### 주요 작성 내용

| 항목 | 설명 |
|------|------|
| 수선 대상 시설 | 엘리베이터, 외벽, 지붕, 배관 등 주요 시설 |
| 수선 주기 | 시설별 법정 또는 권장 수선 주기 |
| 수선 방법 | 전면 교체 / 부분 보수 / 점검 |
| 예상 비용 | 시설별 공사 예상 금액 |
| 수선 연도 | 연도별 공사 계획 |

::: tip 외부 서비스 이동
장기수선계획 메뉴 클릭 시 XpERP 외부의 전문 서비스로 이동합니다. 해당 서비스에서 작성한 계획이 시스템에 연동됩니다.
:::

## 2단계: 수선 공사 집행

### 집행 전 확인사항

::: danger 입주자대표회의 의결 필수
수선 공사를 집행하기 전에 반드시 **입주자대표회의의 의결**을 받아야 합니다. 의결 없이 집행하면 법적 문제가 발생할 수 있습니다.
:::

- [ ] 장기수선계획에 포함된 항목인가?
- [ ] 입주자대표회의 의결이 완료되었는가?
- [ ] 예산 잔액이 충분한가?
- [ ] 공사 계약 서류가 준비되었는가?

### 공사비 전표 처리

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 전표입력 (ACCT2010)

수선 공사비를 지출 전표로 기록합니다.

#### 입력 예시: 엘리베이터 교체 공사비 50,000,000원

| 구분 | 계정과목 | 적요 | 금액 |
|------|----------|------|------|
| **차변** | 장기수선충당금 사용 | 엘리베이터 교체 공사비 | 50,000,000 |
| **대변** | 보통예금 (충당금 계좌) | 엘리베이터 교체 대금 지급 | 50,000,000 |

::: tip 전표 입력 자세한 방법
전표 입력 절차는 [지출 전표 입력](/guide/scenarios/expense-voucher) 시나리오를 참고하세요.
:::

### 예산 집행 확인

> **메뉴 경로**: 회계 > [예산관리](/acct/budget)

공사비 집행 후 예산 잔액을 확인합니다.

- 당초 예산 vs 집행액 비교
- 잔여 예산 확인
- 예산 초과 시 추가 편성 필요

## 3단계: 정기 검토 (3년마다)

### 검토 절차

1. 기존 장기수선계획 조회 (외부 서비스)
2. 실제 시설 상태와 비교 검토:
   - 계획대로 수선이 이루어졌는가?
   - 추가 수선이 필요한 시설은 없는가?
   - 공사 단가 변동이 반영되었는가?
3. 계획 수정·조정
4. 입주자대표회의 의결 → 변경 계획 확정
5. 보고서 출력·보관

### 체크리스트

- [ ] 전기 시설 (수변전설비, 배선) 점검 이력 확인
- [ ] 기계 시설 (급수·오수·소방 설비) 수선 상태 확인
- [ ] 건축물 (외벽, 지붕, 복도) 노후화 상태 확인
- [ ] 엘리베이터 정기 검사 결과 반영
- [ ] 충당금 잔액이 향후 계획 집행에 충분한가?

## 자주 발생하는 문제

### 장기수선계획 메뉴에 들어가면 외부 페이지로 이동해요

**원인**: 장기수선계획은 외부 전문 서비스와 연동됩니다.
**해결**: 정상적인 동작입니다. 외부 페이지에서 작업을 진행하세요. 접속이 안 되면 네트워크 상태를 확인하거나 관리자에게 문의하세요.

### 수선 공사 집행 후 예산이 초과되었어요

**원인**: 공사 예산 편성 금액보다 실제 공사비가 증가한 경우
**해결**: [예산관리](/acct/budget)에서 추가 예산을 편성하고 입주자대표회의의 추가 의결을 받아야 합니다.

### 3년 검토 시기를 놓쳤어요

**원인**: 정기 검토 일정 관리 미흡
**해결**: 즉시 검토를 진행하고 입주자대표회의에 보고하세요. 검토 이력은 관련 서류로 보관하고, 이후에는 연초 업무 계획에 검토 일정을 포함하세요.
