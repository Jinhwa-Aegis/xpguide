# 예산 편성

> **소요 시간**: 약 2시간 | **난이도**: ★★★ | **주기**: 연 1회 (12월~1월)

다음 연도 단지 운영에 필요한 수입·지출 예산을 계정과목별로 편성하고, 입주자대표회의 의결을 거쳐 시스템에 확정 등록하는 프로세스입니다.

## 전체 흐름

```
1. 전년 실적 분석         (20분) → 예산실적대조표로 전년 집행 검토
2. 예산안 작성            (60분) → 계정과목별 금액 입력 및 월별 배분
3. 입주자대표회의 의결     (외부)  → 예산안 확정 (시스템 외 절차)
4. 예산 확정 등록         (10분) → 의결된 예산 저장
5. 추경예산 등록          (10분) → 변경 사항 발생 시에만 처리
6. 예산집행현황 모니터링   (5분)  → 월별 집행 현황 수시 확인
7. 예산서/예산실적대조표 출력 (5분) → 관리·보고용 출력
```

---

## 사전 확인

::: tip 예산 편성 시기
통상 **12월 초~1월 중**에 다음 연도 예산을 편성합니다. 관리규약에서 정한 일정을 먼저 확인하세요.
:::

### 체크리스트

- [ ] 당해 연도 예산실적대조표를 검토했는가?
- [ ] 전년 대비 변동 예정 항목(인건비 인상, 용역 계약 갱신 등)을 파악했는가?
- [ ] 관리규약상 예산 편성 기준과 입주자대표회의 의결 일정을 확인했는가?
- [ ] 세입·세출 계정과목 코드가 최신 상태인가?

---

## 1단계: 전년 실적 분석 (20분)

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산실적대조표 (ACCT6050)

전년도 예산 대비 실제 집행 현황을 확인하여 다음 연도 예산 수립의 근거로 활용합니다.

### 절차

1. 예산실적대조표 화면 진입
2. **조회 연도** 선택 (전년도)
3. **세입** / **세출** 구분 선택 후 조회
4. 항목별 **예산액 vs 실적액 vs 증감** 확인

### 확인 포인트

| 항목 | 확인 내용 |
|------|-----------|
| **집행률이 낮은 항목** | 과다 편성 여부 → 감액 검토 |
| **예산 초과 항목** | 부족 원인 파악 → 증액 검토 |
| **신규 지출 예정 항목** | 전년 미편성 항목 신규 추가 |

---

## 2단계: 예산안 작성 (60분)

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산등록 (ACCT6010)

계정과목별 연간 예산 금액을 입력합니다.

### 절차

1. 예산등록 화면 진입
2. **년이월** 선택 → 등록할 연도 입력
3. **조회** 클릭 → 전년도 계정과목 목록 불러오기
4. **세입** / **세출** 탭을 전환하며 각 계정과목 금액 입력
5. 금액 입력 후 **저장** 클릭

### 월별 배분 방법

계정과목 행을 **더블클릭**하면 12개월 월별 배분 팝업이 열립니다.

| 배분 방식 | 설명 |
|-----------|------|
| **자동 균등 배분** | 연간 금액 ÷ 12 → 각 월에 동일 배분 |
| **수동 입력** | 계절성 지출(냉·난방 등)은 월별로 직접 입력 |

::: tip 계절성 항목 예시
난방비(11~3월), 냉방비(6~9월), 청소 특별소독비(하절기) 등은 월별 수동 배분이 권장됩니다.
:::

### 세입·세출 구분

| 구분 | 설명 | 예시 |
|------|------|------|
| **세입** | 단지로 들어오는 수입 | 관리비 수입, 이자 수입, 잡수입 |
| **세출** | 단지에서 나가는 지출 | 인건비, 용역비, 수선유지비, 보험료 |

---

## 3단계: 입주자대표회의 의결 (외부 절차)

예산안을 관리규약에 따라 **입주자대표회의에 제출·의결**합니다. 이 단계는 시스템 외부 절차입니다.

- 예산서 출력: 회계 > [예산관리](/acct/budget) > 예산서 (ACCT6040)
- 의결이 완료된 후 4단계를 진행합니다.

::: warning 의결 전 확정 금지
입주자대표회의 의결 전에 예산을 확정하지 마세요. 의결된 내용과 시스템 내 금액이 반드시 일치해야 합니다.
:::

---

## 4단계: 예산 확정 등록 (10분)

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산등록 (ACCT6010)

의결된 예산을 최종 저장하여 집행 기준으로 확정합니다.

### 절차

1. 예산등록 화면에서 해당 연도 조회
2. 의결 내용에 맞게 금액 수정 (변경 사항이 있는 경우)
3. **저장** 클릭
4. 예산서(ACCT6040)를 출력하여 의결 내용과 최종 대조 확인

---

## 5단계: 추경예산 등록 (필요 시, 10분)

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 추경예산등록 (ACCT6011)

연중 예산 확정 후 불가피한 사정으로 **예산을 변경**해야 할 때만 사용합니다.

### 언제 사용하나?

- 예상치 못한 대규모 시설 수선 발생
- 관리비 단가 변경으로 세입 구조 변화
- 입주자대표회의에서 예산 변경 의결

### 절차

1. 추경예산등록 화면 진입
2. 해당 연도·회차 선택 (1차 추경, 2차 추경 등)
3. 변경 항목 금액 수정 후 **저장**

::: warning 추경은 꼭 필요할 때만
추경예산은 예산 확정 후 변경이 필요한 경우에만 사용합니다. 일상적인 소액 변동은 예산 조정이 아닌 집행 단계에서 관리합니다.
:::

---

## 6단계: 예산집행현황 모니터링 (5분)

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산집행현황 (ACCT6020)

월별로 예산 대비 집행 잔여 현황을 확인합니다.

### 절차

1. 예산집행현황 화면 진입
2. **조회 기간** 선택 (월별 또는 누계)
3. **세입** / **세출** 구분 선택 후 조회
4. 항목별 **예산액 / 집행액 / 잔액 / 집행률** 확인

### 확인 포인트

- [ ] 집행률이 지나치게 낮은 항목(예산 미집행 위험)은 없는가?
- [ ] 예산 잔액이 부족하여 초과 지출 위험이 있는 항목은 없는가?

::: tip 수시 모니터링 권장
예산집행현황은 매월 말 마감 전에 확인하는 것이 좋습니다. 예산 초과가 예상되면 추경예산 절차를 조기에 진행하세요.
:::

---

## 7단계: 예산서/예산실적대조표 출력 (5분)

### 예산서

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산서 (ACCT6040)

연간 세입·세출 예산을 보고용으로 출력합니다.

### 예산결산서

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산결산서 (ACCT6030)

연도말 결산 후 예산 대비 결산 현황을 출력합니다.

### 예산실적대조표

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산실적대조표 (ACCT6050)

전기 예산 vs 실제 집행 증감을 비교하는 표입니다. 다음 연도 예산 편성 시 필수 참고 자료입니다.

## 자주 발생하는 문제

### 예산등록 화면에서 전년도 계정과목이 안 나와요

**원인**: 년이월 후 조회를 하지 않았거나, 전년도 예산이 등록되지 않은 상태입니다.
**해결**: 년이월 항목을 올바르게 선택한 후 **조회** 버튼을 클릭하세요. 최초 연도 설정이라면 계정과목을 직접 추가해야 합니다.

### 월별 배분 더블클릭이 작동하지 않아요

**원인**: 계정과목 행이 아닌 금액 셀이나 다른 영역을 클릭한 경우입니다.
**해결**: 계정과목 코드 또는 항목명이 있는 **행의 왼쪽 영역**을 더블클릭하세요.

### 예산 초과 지출이 발생했어요

**원인**: 예산 편성 시 미반영된 지출이 발생했거나 집행 관리가 미흡한 경우입니다.
**해결**: 입주자대표회의 의결을 거쳐 추경예산(ACCT6011)으로 예산을 증액하세요. 의결 없이 예산 초과 지출은 관리규약 위반이 될 수 있습니다.

### 세입과 세출 합계가 맞지 않아요

**원인**: 관리비 수입과 지출 예산이 일치하지 않는 경우입니다.
**해결**: 예산 편성 원칙상 세입과 세출은 균형을 이루어야 합니다. 항목별로 재검토하여 누락된 세입 또는 세출 항목이 있는지 확인하세요.
