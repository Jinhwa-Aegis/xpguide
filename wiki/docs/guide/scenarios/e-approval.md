# 전자결재 처리

> **소요 시간**: 건당 약 5분 | **난이도**: ★☆☆ | **주기**: 수시

문서를 작성하고 결재선을 지정하여 상신한 뒤, 결재자가 승인 또는 반려하는 전자결재 프로세스입니다. 완료된 문서는 Xp문서함에 자동 보관됩니다.

## 전체 흐름

```
1. 결재라인 사전 설정   (최초 1회)  → 단지관리에서 결재라인 등록
2. 문서 작성 (기안)     (3분)       → 장표 종류 선택 후 내용 입력
3. 결재선 지정          (1분)       → 결재자 순서 지정
4. 상신                 (1분)       → 결재 요청 전송
5. 결재자 승인/반려     (수신자)    → 결재자가 검토 후 처리
6. 완료 문서 보관       (자동)      → Xp문서함에 자동 저장
```

---

## 사전 준비: 결재라인 등록 (최초 1회)

> **메뉴 경로**: 단지관리 > 단지정보 > 결재라인등록 (DANG0051)

전자결재를 사용하기 전에 결재 흐름(누가 누구에게 결재를 받는지)을 먼저 설정합니다.

### 절차

1. 결재라인등록 화면 진입
2. **장표 종류** 선택:

| 장표 종류 | 설명 |
|-----------|------|
| 지출결의서 | 지출 승인 요청 |
| 품의서 | 업무 사항 결재 |
| 보고서 | 업무 보고 |
| 기타 | 단지별 설정 |

3. 해당 장표에 결재라인 등록:
   - 결재자 직급·이름 순서대로 추가
   - 합의자·수신자(참조) 추가 (해당 시)
4. **저장** 클릭

::: tip 결재라인은 장표 종류별로 설정
장표 종류(지출결의서, 품의서 등)마다 별도의 결재라인을 등록할 수 있습니다. 업무 유형에 따라 결재 경로가 다를 때 활용하세요.
:::

---

## 1단계: 문서 작성 (기안) (3분)

> **메뉴 경로**: Xp전자결재 > [전자결재](/appr/)

### 절차

1. 전자결재 화면 진입
2. **문서 작성** 또는 **기안** 클릭
3. **장표 종류** 선택 (지출결의서, 품의서 등)
4. 문서 내용 입력:

| 항목 | 설명 |
|------|------|
| **제목** | 문서의 제목 |
| **내용** | 결재 요청 내용 상세 기술 |
| **금액** | 지출결의서의 경우 금액 입력 |
| **첨부파일** | 관련 증빙 자료 첨부 (선택) |

5. 내용 확인 후 다음 단계로 이동

---

## 2단계: 결재선 지정 (1분)

문서 작성 화면에서 결재선을 지정합니다.

### 절차

1. **결재선** 항목에서 **결재라인 불러오기** 클릭
2. 사전 등록된 결재라인 선택
3. 필요 시 이번 문서에만 결재자 추가·변경
4. 결재 순서 확인

### 결재선 구성

| 구분 | 설명 |
|------|------|
| **결재자** | 승인 권한이 있는 결재 담당자 (순서대로 결재) |
| **합의자** | 결재 과정에 의견을 제시하는 참여자 |
| **수신자** | 결재 완료 후 문서를 공유받는 참조자 |

---

## 3단계: 상신 (1분)

> **메뉴 경로**: Xp전자결재 > [전자결재](/appr/) > 상신

1. 문서 내용과 결재선 최종 확인
2. **상신** 버튼 클릭
3. 첫 번째 결재자에게 알림 전송

::: warning 상신 후 수정 불가
상신 이후에는 문서 내용을 **수정할 수 없습니다**. 상신 전에 내용과 첨부파일을 반드시 확인하세요. 수정이 필요하면 결재자에게 반려 요청 후 재작성해야 합니다.
:::

---

## 4단계: 결재자 승인/반려 (결재자)

결재 요청을 받은 결재자가 처리합니다.

### 승인 절차

1. 전자결재 수신함에서 해당 문서 선택
2. 내용 검토
3. **승인** 클릭 → 다음 결재자에게 자동 이관 (또는 최종 승인 처리)

### 반려 절차

1. 전자결재 수신함에서 해당 문서 선택
2. 내용 검토
3. **반려** 클릭 → 반려 사유 입력
4. 기안자에게 반려 알림 전송

::: tip 반려 후 재기안
반려된 문서는 수정 후 새로 기안하여 상신해야 합니다.
:::

---

## 5단계: 완료 문서 보관 (자동)

모든 결재자의 승인이 완료되면 문서가 자동으로 Xp문서함에 저장됩니다.

> **메뉴 경로**: [Xp문서함](/docs-menu/)

### 문서함 활용

1. Xp문서함 화면 진입
2. 검색 조건 입력 (기간, 장표 종류, 제목, 기안자 등)
3. 해당 문서 선택 후 열람·출력

---

## 아파트뱅크 이체 연계

지출결의서 결재 완료 후 실제 이체로 이어지는 흐름입니다.

```
지출결의서 작성 → 결재 완료 → 아파트뱅크 이체 실행
```

> **메뉴 경로**: 아파트뱅크 > [이체](/bank/transfer)

1. 지출결의서 전자결재 완료
2. 아파트뱅크 이체 화면에서 해당 결의서 연계
3. 이체 실행 (공인인증서 서명)
4. 이체 결과 확인

::: tip 결재와 이체를 연계하면
지출 승인과 실제 이체가 시스템으로 연결되어 **이중 확인**이 되므로 횡령·오류 예방에 효과적입니다.
:::

## 자주 발생하는 문제

### 결재라인이 자동으로 불러와지지 않아요

**원인**: 결재라인등록(DANG0051)에서 해당 장표 종류의 결재라인이 등록되지 않은 경우
**해결**: 단지관리 > 단지정보 > 결재라인등록에서 장표 종류별 결재라인을 먼저 등록하세요.

### 상신한 문서를 수정해야 해요

**원인**: 상신 후 내용 오류 발견
**해결**: 현재 결재 중인 결재자에게 **반려** 요청을 합니다. 반려 후 문서를 수정하여 새로 기안·상신하세요.

### 결재 완료 문서가 Xp문서함에서 보이지 않아요

**원인**: 검색 기간 또는 조건이 맞지 않는 경우
**해결**: Xp문서함에서 검색 기간을 넓히거나, 장표 종류·제목으로 다시 검색하세요.

### 반려된 문서를 다시 올리고 싶어요

**원인**: 결재자 반려 후 재상신 필요
**해결**: 반려 사유를 확인하고 내용을 수정한 뒤, **새 문서로 기안**하여 다시 상신하세요. 반려된 문서는 직접 수정하여 재상신할 수 없습니다.

## 체크리스트

### 기안 전

- [ ] 장표 종류가 맞는가?
- [ ] 결재 내용이 정확하고 충분한가?
- [ ] 첨부 증빙 자료가 준비되었는가?
- [ ] 결재선(결재자 순서)이 올바른가?

### 상신 후

- [ ] 첫 번째 결재자에게 알림이 전송되었는가?
- [ ] 결재 진행 상태를 확인했는가?

### 결재 완료 후

- [ ] Xp문서함에 문서가 저장되었는가?
- [ ] 지출결의서라면 아파트뱅크 이체가 실행되었는가?
