# 차량 등록/관리

> **소요 시간**: 건당 약 5분 | **난이도**: ★★☆ | **주기**: 수시 (전출입 발생 시)

입주 세대의 차량을 등록·관리하는 프로세스입니다.

## 전체 흐름

```
1. 입주 차량 등록       (3분)  → 개별 입력 또는 엑셀 일괄 등록
2. 차량 현황 확인       (2분)  → 등록 차량 조회
3. 전출 시 차량 해제    (1분)  → 퇴거 세대 차량 삭제
```

---

## 1단계: 입주 차량 등록 (3분)

> **메뉴 경로**: 입주자 > [차량관리](/occp/vehicle) > 차량일괄등록 (OCCP0130)

입주 세대의 차량을 동/호수 기준으로 등록합니다.

### 개별 등록 절차

1. 차량일괄등록 화면 진입
2. 동·호수 입력 후 **조회** 클릭
3. 해당 세대의 차량 행에 **차량번호** 입력
4. 차종, 색상, 차량 소유자 정보 입력 (해당 시)
5. **저장** 클릭

### 엑셀 일괄 등록 절차

신규 단지 또는 대량 등록이 필요한 경우:

1. 화면 상단 **엑셀자료올리기** 버튼 클릭
2. 엑셀 템플릿 양식 다운로드
3. 동, 호수, 차량번호 등 정보 입력
4. 파일 형식: **Excel 97-2003(.xls)**
5. 파일 업로드 후 내용 확인
6. **저장** 클릭

### 체크리스트

- [ ] 동·호수와 차량번호가 정확한가?
- [ ] 세대당 등록 가능한 대수를 초과하지 않는가?
- [ ] 전출 세대의 차량이 등록 해제되었는가?

---

## 2단계: 전출 시 차량 해제

> **메뉴 경로**: 입주자 > [차량관리](/occp/vehicle) > 차량일괄등록 (OCCP0130)

세대 전출 처리 후 해당 세대의 차량 등록을 해제합니다.

1. 차량일괄등록 화면에서 전출 세대 동·호수 조회
2. 등록된 차량번호 삭제
3. **저장** 클릭

::: warning 전출 시 차량 등록 해제
세대 전출 처리 후 반드시 **차량 등록 해제** 여부를 확인하세요. 해제하지 않으면 퇴거 세대 차량이 계속 등록 상태로 남습니다.
:::

---

## 자주 발생하는 문제

### 차량번호가 등록되지 않아요

**원인**: 차량번호 형식 오류 (특수문자, 공백 포함) 또는 동·호수 미입력
**해결**: 차량번호는 숫자와 한글만 입력합니다. 동·호수를 먼저 조회한 후 차량번호를 입력하세요.

### 전출 세대 차량이 아직 등록되어 있어요

**원인**: 전출 처리 시 차량 등록 해제 누락
**해결**: 차량일괄등록(OCCP0130)에서 해당 세대를 조회한 후 차량번호를 삭제하고 저장하세요.
