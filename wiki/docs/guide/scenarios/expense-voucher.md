# 지출 전표 입력

> **소요 시간**: 건당 약 5분 | **난이도**: ★★☆ | **주기**: 수시 (지출 발생 시)

아파트 관리사무소에서 발생하는 각종 지출(관리 용역비, 시설 수선비, 사무용품 구매 등)을 회계 전표로 기록하는 프로세스입니다.

## 전체 흐름

```
1. 지출 증빙 확인       (1분)  → 영수증·세금계산서 등 증빙 확인
2. 전표 입력            (3분)  → 차변/대변 계정과목·금액 입력
3. 증빙 연결            (1분)  → 증빙 문서 첨부 (선택)
4. 저장 및 확인         (1분)  → 전표 저장·인쇄
```

---

## 전표의 기본 개념

::: tip 전표란?
돈의 이동을 기록하는 장부입니다. 모든 거래는 **차변(받는 쪽)**과 **대변(주는 쪽)**으로 나누어 기록하며, 차변 합계와 대변 합계는 항상 같아야 합니다.
:::

### 전표 유형

| 유형 | 코드 | 설명 | 예시 |
|------|------|------|------|
| **대체전표** | 1 | 현금 이동 없는 거래 | 감가상각, 미수금 대체 |
| **입금전표** | 2 | 돈이 들어오는 거래 | 관리비 수납, 이자 수입 |
| **출금전표** | 3 | 돈이 나가는 거래 | 용역비 지급, 물품 구매 |

### 자주 사용하는 계정과목 (지출)

| 계정과목 | 용도 | 차변/대변 |
|----------|------|-----------|
| 관리용역비 | 청소·경비 등 용역 대금 | 차변(비용) |
| 수선유지비 | 시설 수선·보수 비용 | 차변(비용) |
| 소모품비 | 사무용품·소모품 구매 | 차변(비용) |
| 보통예금 | 은행 계좌에서 출금 | 대변(자산 감소) |
| 미지급금 | 아직 지급하지 않은 대금 | 대변(부채 증가) |

---

## 1단계: 지출 증빙 확인 (1분)

전표를 입력하기 전에 증빙 자료를 확인합니다.

### 증빙 종류

- **세금계산서**: 사업자 간 거래 (부가세 포함)
- **카드 매출전표**: 법인카드 결제
- **현금영수증**: 현금 결제 시
- **이체확인증**: 계좌이체 시
- **간이영수증**: 소액 거래 (법정 한도 이내)

### 체크리스트

- [ ] 증빙 자료가 있는가?
- [ ] 금액이 정확한가?
- [ ] 날짜가 맞는가?
- [ ] 공급받는 자(우리 단지) 정보가 정확한가?

---

## 2단계: 전표 입력 (3분)

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 전표입력 (ACCT2010)

### 절차

1. 전표입력 화면 진입
2. **전표일자** 입력 (실제 거래일)
3. 전표번호는 **자동 부여**
4. **차변** 입력:
   - 계정과목 선택 (예: 수선유지비)
   - 금액 입력
   - 적요 입력 (예: "3월 엘리베이터 점검비")
5. **대변** 입력:
   - 계정과목 선택 (예: 보통예금)
   - 금액 입력 (차변과 동일)
   - 적요 입력
6. **차변 합계 = 대변 합계** 확인
7. **저장** 클릭

### 입력 예시: 엘리베이터 점검비 500,000원 계좌이체

| 구분 | 계정과목 | 적요 | 금액 |
|------|----------|------|------|
| **차변** | 수선유지비 | 3월 엘리베이터 점검비 | 500,000 |
| **대변** | 보통예금 | 3월 엘리베이터 점검비 | 500,000 |

### 입력 예시: 사무용품 구매 100,000원 (세금계산서, 부가세 포함)

| 구분 | 계정과목 | 적요 | 금액 |
|------|----------|------|------|
| **차변** | 소모품비 | 사무용품 구매 (공급가) | 90,909 |
| **차변** | 부가세대급금 | 사무용품 부가세 | 9,091 |
| **대변** | 보통예금 | 사무용품 대금 지급 | 100,000 |

::: tip 빠른 입력 방법
**자동분개** 기능을 활용하면 자주 사용하는 전표 패턴을 불러올 수 있습니다. 전표입력 화면에서 자동분개선택(ACCT2020) 버튼을 클릭하세요. 사전에 [기초정보](/acct/basic) > 자동분개등록(ACCT1050)에서 패턴을 등록해 두어야 합니다.
:::

::: tip 적요코드 활용
적요를 매번 입력하기 번거로우면 **적요코드**를 사용하세요. 코드를 입력하면 미리 등록한 적요가 자동 입력됩니다.
:::

## 3단계: 증빙 연결 (1분, 선택)

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 전표입력 > 증빙연결 (ACCT2020)

### 절차

1. 전표 입력 후 **증빙연결** 버튼 클릭
2. 사전에 증빙관리에 등록된 증빙 선택
3. 연결 완료

### 계좌거래내역 연결

> 전표입력 > 계좌거래내역연결 (ACCT2020)

아파트뱅크의 거래내역과 직접 연결할 수도 있습니다.

---

## 4단계: 저장 및 확인 (1분)

1. **저장** 클릭 → 차변/대변 합계 일치 확인 후 저장됨
2. 필요 시 전표 **인쇄**

### 전표 수정이 필요한 경우

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 전표입력 > 전표찾기

1. 전표찾기에서 해당 전표 검색 (기간, 계정, 적요, 금액으로 검색 가능)
2. 전표 선택 후 수정
3. 저장

### 전표 복사/이동

| 기능 | 설명 |
|------|------|
| **수정복사** | 기존 전표를 복사하여 일부 수정 후 새 전표 생성 |
| **전표복사** | 다른 날짜로 전표 복제 |
| **전표이동** | 전표를 다른 날짜로 이동 |

## 입력 확인 방법

### 분개장

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 분개장 (ACCT2061)

일별/월별 차변·대변 내역을 확인합니다.

### 현금출납장

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 현금출납장 (ACCT2090)

현금 입출금과 잔액을 확인합니다.

### 전표수정이력

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 전표수정이력 (ACCT2054)

전표의 수정·삭제 이력을 추적합니다.

## 자주 발생하는 문제

### 저장이 안 돼요

**원인**: 차변 합계 ≠ 대변 합계
**해결**: 금액을 다시 확인하세요. 차변과 대변의 합계가 정확히 일치해야 저장됩니다.

### 어떤 계정과목을 써야 할지 모르겠어요

**원인**: 회계 계정 체계에 익숙하지 않은 경우
**해결**: [기초정보](/acct/basic) > 계정과목코드관리(ACCT1020)에서 계정 목록을 확인하세요. 서울시계·국토부계 등 기준에 따라 다를 수 있습니다. 확실하지 않으면 기존에 유사한 거래의 전표를 전표찾기로 검색하여 참고하세요.

### 이미 마감된 월에 전표를 입력해야 해요

**원인**: 월마감 후 누락 전표 발견
**해결**: 관리자에게 월마감 해제를 요청하거나, 다음 달 날짜로 전표를 입력하세요.

### 전표를 잘못 입력해서 삭제하고 싶어요

**원인**: 오입력
**해결**: 전표찾기에서 해당 전표를 찾아 삭제하세요. 삭제 이력은 전표수정이력(ACCT2054)에 남습니다. 이미 관련 처리(수납 연계 등)가 된 전표는 역분개(반대 전표 입력)로 처리하는 것이 안전합니다.
