# 업무 시나리오 가이드

실제 업무 흐름에 맞춰 **단계별로 따라 하는 가이드**입니다. 각 시나리오는 업무 목적, 전체 흐름, 체크리스트, 주의사항을 포함합니다.

::: tip 이 가이드의 활용법
메뉴 설명이 아닌 "**이 업무를 어떻게 처리하는가**"에 초점을 맞추었습니다. 처음 해보는 업무라면 순서대로 따라가세요. **본인 직무에 해당하는 섹션**부터 확인하면 빠르게 찾을 수 있습니다.
:::

---

## 시설 담당

검침·민원 등 시설 관리 업무를 담당합니다.

| 시나리오 | 소요 시간 | 난이도 | 주기 |
|----------|-----------|--------|------|
| [검침 처리](/guide/scenarios/meter-reading) | 약 1~2시간 | ★★☆ | 매월 |
| [민원 접수/처리](/guide/scenarios/complaint) | 건당 10분 | ★★☆ | 수시 |

::: info 검침은 단지마다 담당이 다릅니다
검침은 경리가 하는 단지도 있고 시설 담당이 하는 단지도 있습니다. XpERP상 부과·수납 흐름의 첫 단계이므로 경리 섹션에도 포함되어 있습니다.
:::

## 월간 업무 타임라인

```
1주차  ──── 전월 수납 마감 + 검침 시작
2주차  ──── 검침 완료 + 부과 준비
3주차  ──── 관리비 부과 + 고지서 발송
4주차  ──── 수납 처리 + 급여 지급
월말   ──── 월말 마감 + 미수금 확인
```

전체 업무 흐름은 [월간 업무 흐름](/guide/workflow)을 참고하세요.
