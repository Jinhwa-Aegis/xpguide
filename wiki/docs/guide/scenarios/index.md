# 업무 시나리오 가이드

실제 업무 흐름에 맞춰 **단계별로 따라 하는 가이드**입니다. 각 시나리오는 업무 목적, 전체 흐름, 체크리스트, 주의사항을 포함합니다.

::: tip 이 가이드의 활용법
메뉴 설명이 아닌 **"이 업무를 어떻게 처리하는가"**에 초점을 맞추었습니다. 처음 해보는 업무라면 순서대로 따라가세요.
:::

## 월간 업무 타임라인

```
1주차  ──── 전월 수납 마감 + 검침 시작
2주차  ──── 검침 완료 + 부과 준비
3주차  ──── 관리비 부과 + 고지서 발송
4주차  ──── 수납 처리 + 급여 지급
월말   ──── 월말 마감 + 미수금 확인
```

전체 업무 흐름은 [월간 업무 흐름](/guide/workflow)을 참고하세요.
