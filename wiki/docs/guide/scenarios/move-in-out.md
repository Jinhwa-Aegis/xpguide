# 전출입 처리

> **소요 시간**: 건당 약 15분 | **난이도**: ★★☆ | **주기**: 수시 (전출입 발생 시)

세대 소유자/임차인의 전입(입주)과 전출(퇴거)을 처리하고, 중간정산·장기수선충당금 정산을 수행하는 프로세스입니다.

## 전체 흐름

### 전입(입주) 처리

```
1. 이전 세대주 전출 확인   (1분)  → 전출 처리 완료 여부 확인
2. 신규 입주자 등록        (5분)  → 세대 정보·가족·차량 입력
3. 관리비예치금 처리       (3분)  → 선수관리비 입금 확인
4. 부과 기준 반영          (1분)  → 부과그룹·자동이체 확인
```

### 전출(퇴거) 처리

```
1. 미납 확인              (2분)  → 체납 관리비 정리
2. 장기수선충당금 정산     (5분)  → 환급액 계산
3. 전출 처리 실행         (3분)  → 중간관리비 정산
4. 관리비예치금 환급       (3분)  → 선수관리비 반환
5. 차량·카드 정리         (2분)  → 출입카드·차량 등록 해제
```

---

## 전입(입주) 처리

### 1단계: 이전 세대주 전출 확인 (1분)

::: warning 전출 먼저
이전 세대주의 전출 처리가 완료되어야 새 입주자를 등록할 수 있습니다.
:::

[수납](/recp/) > [미납조회](/recp/overdue)에서 해당 세대의 미납이 모두 정리되었는지 확인합니다.

### 2단계: 신규 입주자 등록 (5분)

> **메뉴 경로**: 입주자 > [입주처리](/occp/move-in) > 입주등록 (OCCP1010)

#### 절차

1. 입주등록 화면 진입
2. 조건 검색 (동·호수·상태)으로 해당 세대 찾기
3. 세대 선택 → **수정** 클릭
4. 기본 정보 입력:
   - **직업/일반 구분** 선택
   - **입주 상태** 탭: 소유자/임차인 구분, 입주일
   - **가족 구성원** 탭: 동거인 정보
   - **차량** 탭: 차량 등록
   - **반려동물** 탭: 해당 시 입력
   - **임대 정보** 탭: 임차인인 경우 계약 정보
5. 소유자와 입주자가 다르면 별도 입력
6. **저장**

#### 대량 등록 (엑셀)

신규 단지 등 대량 입주 시:
1. 엑셀 템플릿 다운로드
2. 동, 호수 등 정보 입력
3. 파일 형식: **Excel 97-2003(.xls)**
4. 업로드 실행

::: warning 개인정보 동의
입주자 등록 전에 **개인정보 수집·이용 동의서**를 반드시 받으세요. [개인정보](/occp/privacy) 관리 기능을 활용할 수 있습니다.
:::

### 3단계: 관리비예치금 처리 (3분)

> **메뉴 경로**: 입주자 > [입주처리](/occp/move-in) > 관리비예치금(선수관리) (OCCP1040)

입주 시 납부하는 관리비 예치금(선수관리비)을 등록합니다.

1. 해당 세대 선택
2. 예치금 금액 등록
3. 입금 확인 후 납부 상태 변경
4. 필요 시 예치금 안내서·영수증 출력

### 4단계: 부과 기준 반영 (1분)

- [기초정보](/impo/basic)에서 해당 세대가 올바른 **부과그룹**에 포함되어 있는지 확인
- **자동이체** 신청 세대는 자동이체세대등록(IMPO1060) 확인
- **부과제외** 해제 필요 시 부과제외세대등록(IMPO1040) 확인

---

## 전출(퇴거) 처리

### 1단계: 미납 확인 (2분)

> **메뉴 경로**: 수납 > [미납조회](/recp/overdue) > 미납대장 (RECP5070)

해당 세대의 미납 관리비를 확인합니다.

::: danger 미납 정리 필수
미납이 있으면 전출 처리를 진행할 수 없습니다. 전출 전에 모든 체납 관리비를 정산하세요.
:::

### 2단계: 장기수선충당금 정산 (5분)

> **메뉴 경로**: 입주자 > [전출처리](/occp/move-out) > 장기수선충당금 (OCCP2015)

입주 기간 동안 납부한 장기수선충당금의 환급액을 계산합니다.

#### 절차

1. 장기수선충당금 화면 진입
2. 정산 기간 설정
3. **처리** 클릭 → 시스템이 자동 계산
4. 정산 내역 확인:
   - 입주 기간
   - 총 납부 금액
   - 환급 금액
5. 확인서 출력 (필요 시)

::: tip 정산 기준
2022년 1월 13일 이후는 **월 단위** 기준으로 계산됩니다.
:::

### 3단계: 전출 처리 실행 (3분)

> **메뉴 경로**: 입주자 > [전출처리](/occp/move-out) > 전출처리 (OCCP2020)

#### 절차

1. 전출처리 화면 진입
2. **동·호수**와 **전출일자** 입력
3. 시스템이 체납 현황 표시 → 체납이 있으면 먼저 정산
4. **중간정산** 클릭:
   - 전출일까지의 관리비를 일할 계산
   - 장기수선충당금 환급액 확인
5. 정산 내역 확인 후 **전출 처리** 실행
6. 중간정산서 출력

::: warning 전출 후 복구
전출 처리 후 되돌리기가 어렵습니다. 정산 내역을 충분히 확인한 후 실행하세요.
:::

### 4단계: 관리비예치금 환급 (3분)

> **메뉴 경로**: 입주자 > [입주처리](/occp/move-in) > 관리비예치금(선수관리) (OCCP1040)

1. 해당 세대의 예치금 내역 확인
2. 환급 처리
3. 환급 영수증 출력

### 5단계: 차량·카드 정리 (2분)

| 항목 | 메뉴 경로 |
|------|-----------|
| 차량 등록 해제 | [입주자](/occp/) > [차량관리](/occp/vehicle) |
| 출입카드 회수/비활성화 | [입주처리](/occp/move-in) > 출입카드발급등록 |

## 전입/전출 현황 조회

> **메뉴 경로**: 입주자 > [전출처리](/occp/move-out) > 전입/전출자현황 (OCCP2010)

기간별 전입·전출 내역을 조회할 수 있습니다.

- 필터: 전입/전출 구분, 기간 설정
- 세대 선택 시 상세 정보 확인

---

## 소유주 변경 (매매)

아파트 매매로 소유자만 변경되는 경우 (입주자는 동일):

> **메뉴 경로**: 입주자 > [입주처리](/occp/move-in) > 소유주변동처리

1. 해당 세대 선택
2. 새 소유자 정보 입력
3. **저장** → 시스템에 자동 반영

---

## 체크리스트

### 전입 시

- [ ] 이전 세대주 전출 처리가 완료되었는가?
- [ ] 입주자 정보(이름, 연락처, 가족 등)를 정확히 입력했는가?
- [ ] 차량을 등록했는가?
- [ ] 관리비예치금(선수관리비)을 수납했는가?
- [ ] 부과그룹에 포함되어 있는가?
- [ ] 자동이체 신청을 했는가?
- [ ] 개인정보 동의서를 받았는가?

### 전출 시

- [ ] 미납 관리비가 모두 정산되었는가?
- [ ] 장기수선충당금 환급액을 계산했는가?
- [ ] 중간정산(일할 관리비)을 완료했는가?
- [ ] 관리비예치금을 환급했는가?
- [ ] 차량 등록을 해제했는가?
- [ ] 출입카드를 회수/비활성화했는가?
- [ ] 정산서를 출력·교부했는가?

## 자주 발생하는 문제

### 전출 처리가 안 돼요

**원인**: 해당 세대에 미납 관리비가 남아 있는 경우
**해결**: [미납조회](/recp/overdue)에서 체납 내역을 확인하고, 먼저 수납 처리하세요.

### 전출 세대의 미수금을 새 입주자에게 청구해도 되나요?

**아닙니다.** 이전 세대주의 미수금은 **이전 세대주 본인**에게 청구해야 합니다. 새 입주자에게 전가할 수 없습니다.

### 중간정산 금액이 이상해요

**원인**: 전출일 기준 일할 계산 오류, 검침 미반영 등
**해결**: 전출일이 정확한지 확인하고, 해당 월 검침이 완료되었는지 확인하세요.

### 매매인데 전출입을 해야 하나요?

소유자만 변경되고 입주자가 동일하면 **소유주변동처리**만 하면 됩니다. 입주자(임차인)까지 변경되면 전출입 처리가 필요합니다.
