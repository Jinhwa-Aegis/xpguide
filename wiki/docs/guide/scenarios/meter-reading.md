# 검침 처리

> **소요 시간**: 약 1~2시간 | **난이도**: ★★☆ | **주기**: 매월 1~2주차

각 세대의 계량기(전기·수도·온수·가스·난방)를 검침하여 사용량을 산정하는 프로세스입니다. 검침 데이터는 관리비 부과의 기초가 됩니다.

## 전체 흐름

```
1. 전월 지침 불러오기   (1분)  → 전월 데이터 로드
2. 당월 지침 입력       (30분~) → 세대별 검침값 입력
3. 요금 계산           (3분)  → 사용량·요금 자동 계산
4. 이상값 확인         (5분)  → 비정상 사용량 점검
5. 저장               (1분)  → 확정 저장
───── 위 과정을 전기·수도·온수·가스·난방 각각 반복 ─────
```

---

## 검침 종류 및 순서

| 순서 | 검침 항목 | 메뉴 경로 | 화면 코드 |
|------|-----------|-----------|-----------|
| 1 | 전기검침 | [검침](/insp/) > [전기검침](/insp/electric) | INSP1120 |
| 2 | 수도검침 | [검침](/insp/) > [수도검침](/insp/water) | INSP2110 |
| 3 | 온수검침 | [검침](/insp/) > [온수검침](/insp/hot-water) | - |
| 4 | 가스검침 | [검침](/insp/) > [가스검침](/insp/gas) | - |
| 5 | 난방검침 | [검침](/insp/) > [난방검침](/insp/heating) | - |

::: tip 순서가 중요한가요?
검침 항목 간 순서는 자유입니다. 다만 **모든 검침을 완료한 후**에 관리비 부과를 진행해야 합니다.
:::

## 전기검침 상세 절차 (대표 예시)

> **메뉴 경로**: 검침 > [전기검침](/insp/electric) (INSP1120)

다른 검침(수도·온수·가스·난방)도 동일한 패턴으로 진행됩니다.

### 1단계: 전월 지침 불러오기

1. 전기검침 화면 진입
2. **부과 연월** 선택 (예: 2026-03)
3. **구역** 선택 (해당되는 경우)
4. **전월지침** 버튼 클릭 → 전월 데이터가 자동으로 채워짐

### 2단계: 당월 지침 입력

**방법 A: 직접 입력**

1. 각 세대의 **당월지침** 칸에 계량기 숫자를 입력
2. **사용량**은 자동 계산됨 (당월지침 - 전월지침)

**방법 B: 엑셀 업로드 (대량 입력)**

1. 엑셀 파일 준비 (칼럼: 동, 호수, 전월지침, 당월지침, 사용량)
2. 파일 형식: **.xlsx**
3. 업로드 실행

::: warning 당월지침이 전월보다 작은 경우
계량기를 교체한 경우가 아니면 당월 지침은 전월보다 커야 합니다. 시스템이 경고를 표시하면 계량기 상태를 확인하세요.
:::

### 3단계: 요금 계산

1. 모든 세대 입력 완료 확인
2. **요금계산** 버튼 클릭
3. 복지 구분 확인 (기초생활, 국가유공자, 장애인 등 감면 대상)
4. 세대별 요금이 자동 계산됨
   - 전기요금 = 사용량 × 단가 (구간별 차등)
   - TV수신료 자동 포함

### 4단계: 이상값 확인

전월 대비 사용량이 비정상적으로 높거나 낮은 세대를 확인합니다.

> **메뉴 경로**: 검침 > [전기검침](/insp/electric) > 전기사용량조회 (INSP1210)

- 전월 대비 **20% 이상** 급증한 세대 → 오입력 또는 실제 사용량 증가인지 확인
- 사용량이 **0**인 세대 → 공실 또는 미입력인지 확인
- 마이너스 사용량 → 지침 입력 오류 가능

### 5단계: 저장

1. 이상값을 모두 확인/수정한 후 **저장** 클릭
2. 저장 완료 메시지 확인

## 수도검침 주의사항

> **메뉴 경로**: 검침 > [수도검침](/insp/water) (INSP2110)

- 수도+온수를 **묶어서 처리**하는 단지가 있음 → 단지 설정 확인
- 지역별 감면 옵션이 있음 (경남, 경기 등) → 해당 지역이면 감면 설정 필요
- 복수 계량기가 있는 세대는 **모든 계량기** 입력 필요

---

## 특수 상황 처리

### 계량기 교체

계량기를 교체한 세대는:
1. 구 계량기 최종 지침 → 전월 검침에 마감
2. 신 계량기 시작 지침 → 당월 검침에 초기값 입력
3. 사용량 = (구 계량기 최종값 - 전월지침) + (당월지침 - 신 계량기 시작값)

### 전기 이중계량기 세대

> **메뉴 경로**: 검침 > [전기검침](/insp/electric) > 전기계량기2검침 (INSP1160)

계량기가 2개인 세대는 별도로 두 번째 계량기를 입력합니다. 사용량 합산 후 구간별 요금이 적용됩니다.

### 감면 세대 확인

> **메뉴 경로**: 검침 > [전기검침](/insp/electric) > 전기감면세대 (INSP1240)

TV수신료 면제, 복지 할인(기초생활·국가유공자·장애인 등) 대상 세대를 확인할 수 있습니다.

## 자주 발생하는 문제

### 전월지침 버튼을 눌러도 데이터가 안 나와요

**원인**: 전월 검침이 저장되지 않았거나, 부과 연월 선택이 잘못된 경우
**해결**: 부과 연월을 정확히 선택하고, 전월 검침이 저장되어 있는지 확인하세요.

### 요금계산 결과가 예상과 달라요

**원인**: 단가 변경, 감면 적용 누락, 구간별 요금 기준 변경
**해결**: 단가 설정과 감면 세대 등록을 확인하고, 한전요금조회(INSP1250)로 비교해 보세요.

### 엑셀 업로드가 안 돼요

**원인**: 파일 형식 오류
**해결**: 전기검침은 **.xlsx**, 다른 업로드는 **Excel 97-2003(.xls)** 형식을 사용해야 합니다. 칼럼 구조도 양식에 맞는지 확인하세요.
