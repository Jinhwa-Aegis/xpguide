# 아파트뱅크 이체

> **소요 시간**: 건당 약 10분 | **난이도**: ★★★ | **주기**: 수시 (지출 발생 시)

아파트 관리사무소에서 발생하는 지출(용역비, 수선비, 급여 등)을 아파트뱅크를 통해 안전하게 이체하는 프로세스입니다. 지출결의서 작성부터 결재 승인, 공인인증서 서명, 이체 결과 확인까지 단계별로 진행합니다.

## 전체 흐름

```
1. 지출결의서 작성     (3분)  → 지급처, 금액, 사유, 계좌 정보 입력
2. 출금전표 작성       (3분)  → 계정과목, 금액, 적요 입력
3. 출금전표 결재요청   (1분)  → 결재 담당자에게 승인 요청
4. 결재 승인 대기      (외부) → 결재권자 승인
5. 자금이체 실행       (2분)  → 공인인증서 서명 후 이체
6. 이체결과 확인       (1분)  → 이체 완료 여부 및 확인증 출력
```

---

## 사전 확인

::: warning 이체 전 필수 확인
이체 실행 전 반드시 **출금 계좌의 잔액**을 확인하세요. 잔액 부족 시 이체가 실패하며, 수수료가 발생할 수 있습니다.
:::

### 체크리스트

- [ ] 지급처(수취인) 정보(은행, 계좌번호, 예금주)가 정확한가?
- [ ] 이체 금액이 세금계산서·계약서와 일치하는가?
- [ ] 출금 계좌에 잔액이 충분한가?
- [ ] 결재권자가 결재 가능한 상태인가?
- [ ] 공인인증서 유효기간이 남아 있는가?

---

## 1단계: 지출결의서 작성 (3분)

> **메뉴 경로**: 아파트뱅크 > [이체](/bank/transfer) > 지출결의서작성 (BANK1200)

지출의 목적과 내용을 기록하는 공식 결의 문서를 작성합니다.

### 절차

1. 지출결의서작성 화면 진입
2. 아래 항목 입력:

| 입력 항목 | 설명 | 예시 |
|-----------|------|------|
| **결의일자** | 결의하는 날짜 | 2026-03-03 |
| **지급처** | 수취인 이름 또는 업체명 | ○○청소용역(주) |
| **지급 금액** | 이체할 총금액 | 1,500,000원 |
| **지급 사유** | 이체 목적 | 3월 청소용역비 |
| **출금 계좌** | 출금할 단지 계좌 | 관리비 통장 |
| **입금 계좌** | 수취인 계좌 정보 | 은행·계좌번호·예금주 |

3. **저장** 클릭

::: tip 자주 사용하는 지급처
매월 반복 지급하는 용역업체는 수취인 정보를 미리 등록해 두면 빠르게 불러올 수 있습니다.
:::

---

## 2단계: 출금전표 작성 (3분)

> **메뉴 경로**: 아파트뱅크 > [이체](/bank/transfer) > 온라인출금전표작성 (BANK1070)

이체 금액을 회계 계정과목에 연결하는 출금전표를 작성합니다.

### 절차

1. 온라인출금전표작성 화면 진입
2. 앞서 작성한 **지출결의서** 불러오기
3. 아래 항목 입력:

| 입력 항목 | 설명 | 예시 |
|-----------|------|------|
| **계정과목** | 지출 항목 분류 | 관리용역비 |
| **금액** | 지출결의서 금액과 동일 | 1,500,000원 |
| **적요** | 거래 내용 요약 | 3월 청소용역비 지급 |
| **부서** | 해당 부서 (단지에 따라 상이) | 관리팀 |

4. **저장** 클릭

### 자주 사용하는 계정과목

| 계정과목 | 용도 |
|----------|------|
| 관리용역비 | 청소·경비·소독 등 용역 대금 |
| 수선유지비 | 시설 수선·보수 비용 |
| 인건비 | 직원 급여 (아파트뱅크 이체 시) |
| 소모품비 | 사무용품·소모품 구매 |
| 보험료 | 단지 보험 납부 |

---

## 3단계: 출금전표 결재요청 (1분)

> **메뉴 경로**: 아파트뱅크 > [이체](/bank/transfer) > 출금전표결재요청 (BANK1080)

작성한 출금전표를 결재권자에게 승인 요청합니다.

### 절차

1. 출금전표결재요청 화면 진입
2. 결재요청할 전표 선택
3. 결재선 확인 (단지 설정에 따라 1단계 또는 다단계 결재)
4. **결재요청** 클릭

::: warning 결재 승인 전 이체 불가
이체는 **결재 승인 완료 후에만** 실행할 수 있습니다. 결재 승인 없이 이체를 시도하면 진행되지 않습니다.
:::

---

## 4단계: 결재 승인 대기 (외부 절차)

결재권자(관리소장, 입주자대표 등)가 결재를 승인합니다. 이 단계는 시스템 외부(또는 결재권자 화면)에서 진행됩니다.

- 결재 완료 여부는 아파트뱅크 결재 목록에서 확인합니다.
- 긴급 이체가 필요한 경우 결재권자에게 직접 연락하여 신속 승인을 요청하세요.

---

## 5단계: 자금이체 실행 (2분)

> **메뉴 경로**: 아파트뱅크 > [이체](/bank/transfer) > 자금이체실행 (BANK1090)

결재가 승인된 전표를 공인인증서로 서명하여 실제 이체를 실행합니다.

### 절차

1. 자금이체실행 화면 진입
2. **승인 완료** 상태의 전표 확인
3. 이체 정보 최종 확인:
   - 출금 계좌 / 입금 계좌
   - 이체 금액
   - 수취인 예금주명
4. **공인인증서 서명** 실행
5. 이체 **실행** 클릭

### 공인인증서 서명 시 주의사항

- 공인인증서의 **유효기간**을 사전에 확인하세요.
- 인증서 비밀번호를 여러 번 틀리면 잠길 수 있습니다.
- 인증서 갱신이 필요한 경우 거래 은행에서 갱신 후 진행하세요.

::: danger 이체 실행은 취소 불가
이체 실행 후에는 즉시 은행으로 전송됩니다. 계좌번호, 금액, 수취인을 **반드시 실행 전에** 다시 한번 확인하세요.
:::

---

## 6단계: 이체결과 확인 (1분)

> **메뉴 경로**: 아파트뱅크 > [이체](/bank/transfer) > 이체결과/이체확인증 (BANK1030)

이체가 정상적으로 완료되었는지 확인하고, 필요 시 이체확인증을 출력합니다.

### 절차

1. 이체결과/이체확인증 화면 진입
2. 해당 이체 건 조회
3. **이체 상태** 확인:
   - **완료**: 정상 이체
   - **실패**: 이체 오류 (사유 확인 필요)
   - **처리 중**: 은행 처리 중 (잠시 후 재확인)
4. 필요 시 **이체확인증 출력** (세금계산서 첨부용 등)

### 이체 실패 시 조치

| 실패 사유 | 조치 |
|-----------|------|
| 수취 계좌 오류 | 지급처 계좌번호 재확인 후 재이체 |
| 잔액 부족 | 출금 계좌에 입금 후 재이체 |
| 은행 시스템 오류 | 잠시 후 재시도 또는 은행 문의 |
| 한도 초과 | 이체 한도 확인 및 은행에 한도 상향 요청 |

---

## 이체 후 전표 확인

이체 완료 후 회계 전표가 자동 생성된 경우 내용을 확인합니다.

> **메뉴 경로**: 회계 > [전표관리](/acct/voucher) > 전표입력 (ACCT2010)

아파트뱅크와 회계 모듈이 연동된 경우 이체 내역이 자동으로 전표에 반영될 수 있습니다. 자동 생성된 전표의 계정과목과 적요가 정확한지 확인하세요.

---

## 자주 발생하는 문제

### 결재요청 후 이체 버튼이 안 눌려요

**원인**: 결재 승인이 아직 완료되지 않은 상태입니다.
**해결**: 결재권자에게 승인 완료를 확인하거나, 결재 목록에서 승인 상태를 확인하세요. **결재 완료 후에만** 이체 실행이 활성화됩니다.

### 공인인증서 오류가 나요

**원인**: 인증서 만료, 비밀번호 오류, 인증서 미설치 등입니다.
**해결**: 인증서 유효기간을 확인하고 만료된 경우 거래 은행에서 갱신하세요. 비밀번호를 5회 이상 틀리면 인증서가 잠기므로 주의하세요.

### 이체가 실패했는데 잔액은 빠져나갔어요

**원인**: 이체 처리 중 오류로 출금 후 환불이 지연되는 경우입니다.
**해결**: 이체결과(BANK1030)에서 상태를 확인하세요. 은행 측에서 통상 1영업일 내에 자동 환불됩니다. 지연될 경우 거래 은행 및 아파트뱅크 고객센터에 문의하세요.

### 계좌번호를 잘못 입력했어요

**원인**: 수취인 계좌 정보 오입력입니다.
**해결**: 이체 전에 발견한 경우 결의서/전표를 수정하면 됩니다. 이체 **완료 후**에 잘못된 계좌로 보낸 경우 즉시 거래 은행에 연락하여 착오 송금 반환 절차를 진행하세요.
