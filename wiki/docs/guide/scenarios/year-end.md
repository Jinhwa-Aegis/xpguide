# 연말 결산

> **소요 시간**: 약 2시간 | **난이도**: ★★★ | **주기**: 연 1회 (회계연도 말)

회계연도의 모든 수입·지출을 확정하고, 최종 재무제표를 작성한 뒤, 차기 연도로 잔액을 이월하는 프로세스입니다.

## 전체 흐름

```
1. 12월 월말 마감         (40분)  → 마지막 월의 일반 마감
2. 연간 재무제표 작성     (20분)  → 연간 시산표·대차대조표·손익계산서
3. 예산 대비 실적 확인    (15분)  → 세입세출서·예산결산서
4. 감사 준비 자료 출력    (15분)  → 각종 장부·보고서 인쇄
5. 년도이월 실행          (5분)   → 차기 연도로 잔액 이월
6. 차기 예산 등록         (15분)  → 새 연도 예산 입력
7. 연말정산 (급여)        (별도)  → 직원 소득세 정산
```

---

## 1단계: 12월 월말 마감 (40분)

12월의 일반 월말 마감을 먼저 완료합니다.

상세 절차는 [월말 마감](/guide/scenarios/month-closing) 시나리오를 참고하세요.

### 연말 추가 확인 사항

- [ ] 12월분 모든 수납이 처리되었는가?
- [ ] 12월분 모든 전표가 입력되었는가?
- [ ] 모든 통장 잔액이 시스템과 일치하는가?
- [ ] 미수금 잔액이 정확한가?
- [ ] 장기수선충당금 잔액이 정확한가?
- [ ] 12월 월마감이 완료되었는가?

---

## 2단계: 연간 재무제표 작성 (20분)

### 합계잔액시산표

> **메뉴 경로**: 회계 > [결산관리](/acct/settlement) > 합계잔액시산표 (ACCT4010)

1. 기간 설정: 회계연도 전체 (예: 2026-01-01 ~ 2026-12-31)
2. **조회** 실행
3. 확인: **차변 합계 = 대변 합계**

### 대차대조표

> **메뉴 경로**: 회계 > [결산관리](/acct/settlement) > 대차대조표 (ACCT4020)

회계연도 말 시점의 자산·부채·자본 현황입니다.

- [ ] 자산 총계 = 부채 총계 + 자본 총계인가?
- [ ] 주요 자산 항목(보통예금, 미수금 등)이 실제와 일치하는가?

### 손익계산서

> **메뉴 경로**: 회계 > [결산관리](/acct/settlement) > 손익계산서 (ACCT4030)

연간 수입·지출·당기순이익을 확인합니다.

### 월별 손익계산서

> **메뉴 경로**: 회계 > [결산관리](/acct/settlement) > 월별손익계산서 (ACCT4031)

1~12월 월별 손익 추이를 확인합니다.

---

## 3단계: 예산 대비 실적 확인 (15분)

### 세입세출서

> **메뉴 경로**: 회계 > [결산관리](/acct/settlement) > 세입세출서 (ACCT4080)

예산 대비 실적 달성률을 확인합니다.

- 계산식: (실적 ÷ 예산) × 100
- 세입(수입), 세출(지출), 차인(차이)별로 확인 가능

### 예산결산서

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산결산서

전기 기초잔액 + 당기 증감을 기준으로 예산 집행 결과를 확인합니다.

### 예산실적대조표

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산실적대조표

예산 금액과 실적 금액의 증감을 계정과목별로 비교합니다.

### 확인 포인트

- [ ] 예산 대비 과다 지출 항목이 있는가? → 사유 정리
- [ ] 미집행 예산이 있는가? → 이월 또는 불용 처리
- [ ] 추경 예산이 있다면 반영되었는가?

---

## 4단계: 감사 준비 자료 출력 (15분)

결산 감사를 위해 아래 자료를 출력·준비합니다.

| 자료 | 메뉴 경로 | 화면 코드 |
|------|-----------|-----------|
| 합계잔액시산표 | 결산관리 > 합계잔액시산표 | ACCT4010 |
| 대차대조표 | 결산관리 > 대차대조표 | ACCT4020 |
| 손익계산서 | 결산관리 > 손익계산서 | ACCT4030 |
| 세입세출서 | 결산관리 > 세입세출서 | ACCT4080 |
| 총계정원장 | [장부관리](/acct/ledger) > 총계정원장 | ACCT3010 |
| 보조부원장 | [장부관리](/acct/ledger) > 보조부원장 | ACCT3020 |
| 계정과목별잔액장 | [장부관리](/acct/ledger) > 계정과목별잔액장 | ACCT3030 |
| 관리비용명세서 | [장부관리](/acct/ledger) > 관리비용명세서 | ACCT3050 |

## 5단계: 년도이월 실행 (5분)

> **메뉴 경로**: 회계 > [결산관리](/acct/settlement) > 년도이월 (ACCT4090)

당기 잔액을 차기 연도의 기초잔액으로 이월합니다.

### 절차

1. 년도이월 화면 진입
2. 이월 대상 회계연도 확인
3. 잔액 내역 확인
4. **이월 실행**

::: danger 되돌릴 수 없습니다
년도이월은 **비가역적**입니다. 한 번 실행하면 취소할 수 없습니다. 반드시 아래 사항을 확인한 후 실행하세요.
:::

### 이월 전 최종 확인

- [ ] 12월 월마감이 완료되었는가?
- [ ] 모든 재무제표를 확인·출력했는가?
- [ ] 예산 대비 실적을 확인했는가?
- [ ] 감사 자료를 모두 준비했는가?
- [ ] 시산표 차변 = 대변인가?
- [ ] **이 작업은 되돌릴 수 없다**는 것을 확인했는가?

---

## 6단계: 차기 예산 등록 (15분)

> **메뉴 경로**: 회계 > [예산관리](/acct/budget) > 예산등록 (ACCT6010)

새 회계연도의 예산을 입력합니다.

### 절차

1. 예산등록 화면 진입
2. 수입 예산: 계정과목별 연간 예산 금액 입력
   - 항목 더블클릭 → 월별 배분 가능
3. 지출 예산: 계정과목별 연간 예산 금액 입력
4. **저장** 클릭

::: tip 추경예산
연중에 예산 변경이 필요하면 **추경예산등록**(ACCT6011)에서 수정합니다. 원본 예산은 보존됩니다.
:::

---

## 7단계: 연말정산 (급여) - 별도 프로세스

직원의 연간 소득세를 정산하는 절차로, 매년 1~2월에 진행합니다.

상세 절차는 [급여 처리](/guide/scenarios/payroll) 시나리오의 연말정산 부분과 [연말정산](/pay/year-end) 메뉴를 참고하세요.

### 간략 흐름

```
기본정보 등록 → 전직장 내역 → 연근로소득 등록 → 개인공제 등록
→ 정산 실행 → 결과 확인 → 디스켓 작성(홈택스 제출) → 환급/추징
```

### 주요 기한

| 항목 | 기한 |
|------|------|
| 근로·퇴직·사업소득 지급명세서 | 다음 연도 **3월 10일** |
| 연말정산 디스켓 홈택스 제출 | **2월 말** |

---

## 연말 결산 종합 체크리스트

### 마감 전

- [ ] 12월 월말 마감 완료
- [ ] 모든 통장 잔액 대사 완료
- [ ] 미수금 잔액 확인 완료

### 재무제표

- [ ] 시산표 차변 = 대변 확인
- [ ] 대차대조표 자산 = 부채 + 자본 확인
- [ ] 손익계산서 확인
- [ ] 예산 대비 실적 확인

### 이월

- [ ] 감사 자료 출력 완료
- [ ] 년도이월 실행 완료
- [ ] 차기 예산 등록 완료

### 연말정산

- [ ] 직원 연말정산 완료
- [ ] 홈택스 제출 완료
- [ ] 환급/추징 급여 반영 완료

## 자주 발생하는 문제

### 년도이월 후 전년도 수정이 필요해요

**원인**: 이월 후 전년도 오류 발견
**해결**: 년도이월은 되돌릴 수 없습니다. 차기 연도에서 조정 전표를 입력하여 수정하세요.

### 차기 연도 기초잔액이 이상해요

**원인**: 이월 전 재무제표 확인 미흡
**해결**: 기초잔액등록(ACCT1030)에서 확인하고, 필요 시 수동 조정하세요.

### 연말정산 기한을 놓쳤어요

**원인**: 기한 미준수
**해결**: 지급명세서 제출 기한 초과 시 가산세가 부과됩니다. 즉시 홈택스에 제출하고, 가산세 규정을 확인하세요.
