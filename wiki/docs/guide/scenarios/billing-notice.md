# 고지서 출력/전송

> **소요 시간**: 약 15분 | **난이도**: ★★☆ | **주기**: 매월 (부과마감 후)

부과마감이 완료된 관리비 고지서를 세대별로 인쇄하거나, 전자고지서·카카오톡·SMS로 전송하는 프로세스입니다.

## 전체 흐름

```
1. 부과마감 확인        (2분)  → 마감 완료 여부 및 전월비교 검증
2. 세대공지사항 입력    (3분)  → 고지서에 첨부할 공지 문구 입력 (선택)
3. 고지서 인쇄          (5분)  → 종이 고지서 출력의뢰
4. 전자고지서 발송      (2분)  → 전자고지 신청 세대에 발송
5. 모바일 발송          (2분)  → 카카오톡/SMS 발송
6. 발송이력 확인        (1분)  → 발송 결과 확인
```

---

## 사전 확인

::: danger 부과마감 완료 후 진행
고지서 발송은 **반드시 부과마감(IMPO9010)이 완료된 후에만** 진행하세요. 마감 전 고지서를 발송하면 금액이 변경될 경우 재발송해야 합니다.
:::

---

## 1단계: 부과마감 확인 (2분)

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) (IMPO9010)

### 절차

1. 부과마감 화면 진입
2. 해당 연월의 **마감 상태** 확인 (마감 완료 여부)
3. **전월비교** 버튼 클릭하여 전월 대비 증감 확인

### 전월비교 확인 포인트

- [ ] 세대별 총 부과 금액이 전월 대비 합리적인 범위인가?
- [ ] 특정 항목이 급격히 변동하지 않았는가?
- [ ] 이상 세대(금액이 0원이거나 비정상적으로 높은 세대)가 없는가?

이상이 있으면 **마감 해제 후 부과 수정**을 먼저 진행하세요.

---

## 2단계: 세대공지사항 입력 (3분, 선택)

> **메뉴 경로**: 단지관리 > [단지정보](/dang/dang-info)

고지서 하단 또는 여백에 인쇄할 공지 문구를 입력합니다.

### 활용 예시

- 납부 기한 안내
- 장기수선충당금 변경 공지
- 관리사무소 휴무 안내
- 주차 관련 공지

::: tip 고지서 1줄당 최대 50byte
한글 기준으로 **1줄에 최대 25자(50byte)**까지 입력 가능합니다. 문구가 길면 줄이 잘리므로 간결하게 작성하세요.
:::

---

## 3단계: 고지서 인쇄 (5분)

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) > 출력의뢰(부과마감)

종이 고지서를 인쇄합니다.

### 절차

1. 부과마감 화면에서 **출력의뢰** 버튼 클릭
2. 출력할 **동·호** 범위 입력 (전체 또는 선택)
3. **고지서 미리보기**로 내용 확인

### 미리보기 확인 포인트

- [ ] 세대 정보(동·호수, 세대주명)가 올바른가?
- [ ] 항목별 금액이 정확한가?
- [ ] 납부 기한이 맞는가?
- [ ] 계좌번호 또는 가상계좌가 올바른가?
- [ ] 공지사항 문구가 정상 출력되는가?

4. 확인 후 **인쇄** 실행

::: warning 미리보기 확인 필수
발송 전 미리보기로 반드시 확인하세요. 발송 후에는 취소하기 어렵습니다.
:::

---

## 4단계: 전자고지서 발송 (2분)

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) > 전자고지서 (IMPO9011)

전자고지서 서비스를 신청한 세대에 고지서를 전자 발송합니다.

### 절차

1. 전자고지서 화면 진입
2. **부과 연월** 입력 후 **조회**
3. 발송 대상 세대 목록 확인
4. **발송** 버튼 클릭
5. 발송 완료 메시지 확인

::: tip 전자고지서 미신청 세대
전자고지서 발송 대상은 서비스를 신청한 세대만 포함됩니다. 미신청 세대는 종이 고지서(3단계) 또는 모바일 발송(5단계)으로 안내합니다.
:::

---

## 5단계: 모바일 발송 (2분)

카카오톡 또는 SMS 문자로 고지서를 발송합니다.

### 카카오톡 발송

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) > 전자고지서 > 카카오톡 발송

| 발송 방식 | 설명 |
|-----------|------|
| **단건 발송** | 특정 세대 1건씩 발송 |
| **대량 발송** | 카카오톡 동의 세대 전체 일괄 발송 |

### SMS 문자 발송

> **메뉴 경로**: 부과 > [부과마감](/impo/closing) > 모바일발송

관리비 납부 안내 문자를 SMS로 발송합니다.

- 발송 대상: 휴대폰 번호가 등록된 세대
- 발송 내용: 부과 연월, 청구 금액, 납부 기한, 납부 계좌 등

::: tip 카카오톡 우선 발송
카카오톡 발송이 가능한 세대는 카카오톡으로 먼저 발송하세요. SMS보다 정보를 더 상세히 전달할 수 있고 비용도 절감됩니다. 카카오톡 미동의 세대는 SMS로 대체 발송됩니다.
:::

---

## 6단계: 발송이력 확인 (1분)

발송 완료 후 발송이력을 확인합니다.

### 발송이력 확인 항목

| 확인 항목 | 설명 |
|-----------|------|
| **발송 건수** | 총 발송 세대 수 |
| **성공 건수** | 정상 수신 세대 수 |
| **실패 건수** | 발송 실패 세대 수 (번호 오류, 수신 거부 등) |
| **발송 일시** | 발송이 완료된 시각 |

발송 실패 건은 **번호 재확인 후 재발송**하거나, 종이 고지서를 우편으로 발송하세요.

---

## 자주 발생하는 문제

### 특정 세대 고지서 금액이 이상해요

**원인**: 부과 계산 오류, 별도금액 잘못 입력, 부과제외 세대 미등록 등입니다.
**해결**: 부과마감을 해제하고 [부과처리](/impo/process)에서 해당 세대 금액을 재확인하세요. 수정 후 다시 마감하고 고지서를 재발송합니다.

### 전자고지서 발송 대상 세대가 없어요

**원인**: 전자고지서 서비스 신청 세대가 등록되지 않은 상태입니다.
**해결**: [기초정보](/impo/basic)에서 전자고지서 수신 동의 세대가 등록되어 있는지 확인하세요. 미등록이라면 세대 신청 후 등록합니다.

### 카카오톡 발송이 실패해요

**원인**: 세대의 카카오톡 동의가 없거나 번호가 변경된 경우입니다.
**해결**: 카카오톡 미동의 세대는 SMS로 자동 대체 발송되는 경우가 많습니다. 번호 변경이 원인이라면 세대 정보에서 연락처를 갱신하세요.

### 고지서 공지사항 문구가 잘려요

**원인**: 1줄에 입력 가능한 최대 글자 수(한글 25자/50byte)를 초과했습니다.
**해결**: 문구를 줄이거나 두 줄로 나누어 입력하세요. 특수문자나 영문 혼용 시 byte 수가 달라질 수 있으니 미리보기로 반드시 확인하세요.

### 이미 발송한 고지서를 취소하고 싶어요

**원인**: 금액 오류 또는 공지사항 오입력 후 발송한 경우입니다.
**해결**: 전자고지서·모바일 발송은 **취소가 어렵습니다**. 오류가 있다면 수정 내용을 별도 문자로 안내하거나, 수정된 고지서를 재발송하세요. 향후에는 발송 전 **미리보기 확인**을 철저히 하세요.
