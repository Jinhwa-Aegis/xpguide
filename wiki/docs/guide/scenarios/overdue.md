# 미수금 관리

> **소요 시간**: 약 15분 | **난이도**: ★★☆ | **주기**: 매월 수납 마감 후

수납 기한이 지났는데도 납부하지 않은 세대를 확인하고, 독촉·법적 조치를 진행하는 프로세스입니다.

## 전체 흐름

```
1. 미납 현황 확인       (3분)  → 미납 세대·금액 파악
2. 미납 집계 분석       (3분)  → 장기/단기 미납 구분
3. 독촉장 발송         (5분)  → 문서 생성·발송
4. 미수금/회계 비교     (2분)  → 수납 모듈과 회계 대사
5. 장기 체납 조치       (2분)  → 법적 절차 검토
```

---

## 1단계: 미납 현황 확인 (3분)

> **메뉴 경로**: 수납 > [미납조회](/recp/overdue) > 미납대장 (RECP5070)

### 절차

1. 미납대장 화면 진입
2. 조회 조건 설정:
   - **조회 기간**: 수납 마감일 기준
   - **금액 범위**: 필요 시 최소 금액 이상만 필터
   - **미납 기간**: 1개월 / 3개월 / 6개월 이상 등
3. **조회** 실행
4. 미납 세대 목록 확인

### 조회 옵션

| 옵션 | 설명 |
|------|------|
| 최종일자만 | 가장 최근 미납 기준으로 조회 |
| 특정일자 | 특정 시점 기준 미납 상태 확인 |
| 잔액조회 | 현재 남은 미납 잔액 기준 |

---

## 2단계: 미납 집계 분석 (3분)

> **메뉴 경로**: 수납 > [미납조회](/recp/overdue) > 미납집계표 (RECP5090)

### 절차

1. 미납집계표 화면 진입
2. 조회 조건 설정
3. **조회** 실행
4. 미납 월수별 집계 확인

### 분석 포인트

- [ ] 1개월 미납 세대 수는? → 단순 지연 가능성, 다음 달 자동이체로 해소 가능
- [ ] 3개월 이상 미납 세대는? → 독촉장 발송 대상
- [ ] 6개월 이상 장기 미납은? → 법적 조치 검토 대상
- [ ] 미납 총액이 전월 대비 증감했는가?

---

## 3단계: 독촉장 발송 (5분)

> **메뉴 경로**: 수납 > [미납조회](/recp/overdue) > 독촉장/독촉대장 (RECP5080)

### 독촉장 종류

| 종류 | 용도 | 대상 |
|------|------|------|
| **간단 독촉장** | 1차 납부 안내 | 1~2개월 미납 |
| **경고 독촉장** | 법적 조치 예고 | 3개월 이상 미납 |
| **최고장** | 최종 경고 (계좌정보 포함) | 장기 미납 |
| **영수증** | 납부 확인용 | 납부 시 발급 |
| **납부확인서** | 공식 납부 증빙 | 요청 시 발급 |

### 절차

1. 독촉장/독촉대장 화면 진입
2. 미납 세대 선택 (복수 선택 가능)
3. 독촉장 양식 선택
4. **출력** 또는 **발송** 실행

::: tip 단계적 독촉
1개월 미납 → 간단 독촉장, 3개월 → 경고 독촉장, 6개월 → 최고장 순서로 단계적 조치를 권장합니다.
:::

---

## 4단계: 미수금/회계 비교 (2분)

> **메뉴 경로**: 수납 > [미납조회](/recp/overdue) > 미수금/회계비교 (RECP5050)

수납 모듈의 미수금과 회계 모듈의 미수금이 일치하는지 대사합니다.

### 절차

1. 미수금/회계비교 화면 진입
2. 기준일자 설정
3. **조회** 실행
4. 수납 미수금 vs 회계 미수금 비교

### 불일치 시 원인

| 원인 | 해결 방법 |
|------|-----------|
| 수납 전표 연계 누락 | [수납 전표 연계](/guide/scenarios/collection)를 실행 |
| 회계 전표 직접 수정 | 전표관리에서 수정 내역 확인 후 조정 |
| 수납 취소 후 전표 미반영 | 해당 전표를 삭제하거나 역분개 처리 |

---

## 5단계: 장기 체납 조치 (2분)

6개월 이상 장기 체납 세대에 대해서는 법적 절차를 검토합니다.

### 조치 단계

```
1. 문자/전화 독촉 (1~2개월)
2. 독촉장 우편 발송 (3개월)
3. 내용증명 발송 (4~5개월)
4. 지급명령 신청 (6개월 이상)
```

### 체크리스트

- [ ] 독촉 이력이 문서로 남아 있는가? (법적 증빙용)
- [ ] 전출 세대의 미수금은 이전 세대주에게 청구하고 있는가?
- [ ] 장기수선충당금 미수금과 일반 관리비 미수금을 구분하고 있는가?

::: warning 전출 세대 미수금
이사 간 세대의 미수금은 **새 입주자가 아닌 이전 세대주**에게 청구해야 합니다. [전출입 처리](/guide/scenarios/move-in-out) 시나리오를 참고하세요.
:::

## 부과/수납/미납 현황 한눈에 보기

> **메뉴 경로**: 수납 > [미납조회](/recp/overdue) > 부과/수납/미납현황 (RECP5040)

세대별·동별로 부과 총액, 수납 총액, 미납 잔액을 한 화면에서 확인할 수 있습니다.

## 자주 발생하는 문제

### 미납 금액이 실제와 달라요

**원인**: 수납 처리가 누락되었거나, 가수금이 적용되지 않은 경우
**해결**: [수납처리](/recp/process)에서 해당 세대의 수납 이력을 확인하고, 가수금(RECP1051)에 미적용 건이 있는지 확인하세요.

### 연체료가 자동 계산되지 않아요

**원인**: 연체율이 등록되지 않았거나, 연체 기준 설정이 안 된 경우
**해결**: [기초정보](/impo/basic) > 연체율등록(IMPO1050)에서 월별 연체율이 정상 등록되어 있는지 확인하세요.

### 미수금/회계 비교에서 차이가 나요

**원인**: 수납 전표 연계 누락, 회계 전표 직접 수정, 수납 취소 후 미반영 등
**해결**: 수납 전표 연계를 다시 실행하고, 전표관리에서 수정 이력(ACCT2054)을 확인하세요.
