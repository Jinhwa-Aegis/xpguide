# 시작하기

XpERP는 아파트 관리사무소의 전반적인 업무를 지원하는 통합 ERP 시스템입니다.

## XpERP 접속 방법

1. 웹 브라우저에서 XpERP 사이트에 접속합니다
2. 발급받은 아이디와 비밀번호로 로그인합니다
3. 메인 화면에서 필요한 메뉴를 선택합니다

## 시스템 구성

XpERP는 다음 14개 대메뉴로 구성되어 있습니다:

| 번호 | 메뉴 | 설명 |
|------|------|------|
| 1 | [Xp전자결재](/appr/) | 전자결재 문서 작성 및 승인 |
| 2 | [Xp문서함](/docs-menu/) | 문서 보관 및 관리 |
| 3 | [단지관리](/dang/) | 단지정보, 환경설정, 조견표 |
| 4 | [입주자](/occp/) | 입주/전출, 차량, 개인정보 관리 |
| 5 | [검침](/insp/) | 전기, 수도, 가스 등 계량기 검침 |
| 6 | [부과](/impo/) | 관리비 부과 처리 |
| 7 | [수납](/recp/) | 관리비 수납 및 미납 관리 |
| 8 | [회계](/acct/) | 전표, 장부, 결산, 예산 관리 |
| 9 | [인사/급여](/pay/) | 직원 인사 및 급여 관리 |
| 10 | [장기수선계획](/afms/) | 장기수선충당금 관리 |
| 11 | [프리미엄](/premium/) | 프리미엄 서비스 |
| 12 | [민원](/jobm/) | 입주민 민원 접수 및 처리 |
| 13 | [아파트뱅크](/bank/) | 은행 거래 및 이체 관리 |
| 14 | [아파트아이](/apti/) | 입주민 모바일 앱 서비스 |

## 권장 학습 순서

신규 사용자라면 다음 순서로 학습하는 것을 권장합니다:

1. **[용어 사전](/guide/glossary)** - 기본 용어 이해
2. **[월간 업무 흐름](/guide/workflow)** - 전체 업무 흐름 파악
3. **[단지관리](/dang/)** - 기본 설정 확인
4. **[입주자](/occp/)** - 세대 정보 관리
5. **[검침](/insp/) → [부과](/impo/) → [수납](/recp/)** - 핵심 월간 업무
6. **[회계](/acct/)** - 회계 처리

## 업무 시나리오 가이드

실제 업무를 처리할 때는 **[업무 시나리오 가이드](/guide/scenarios/)**를 활용하세요. 메뉴 설명이 아니라 "**이 업무를 어떻게 처리하는가**"에 초점을 맞추어 단계별로 따라 할 수 있습니다.

| 업무 | 시나리오 가이드 | 소요 시간 |
|------|----------------|-----------|
| 계량기 검침 | [검침 처리](/guide/scenarios/meter-reading) | 1~2시간 |
| 관리비 산정·부과 | [관리비 부과](/guide/scenarios/billing) | 30분 |
| 관리비 수납 | [수납 처리](/guide/scenarios/collection) | 20분 |
| 체납 관리 | [미수금 관리](/guide/scenarios/overdue) | 15분 |
| 월 마감 | [월말 마감](/guide/scenarios/month-closing) | 40분 |
| 지출 기록 | [지출 전표 입력](/guide/scenarios/expense-voucher) | 건당 5분 |
| 잔액 대사 | [통장 대사](/guide/scenarios/bank-reconciliation) | 20분 |
| 입주·퇴거 | [전출입 처리](/guide/scenarios/move-in-out) | 건당 15분 |
| 급여 지급 | [급여 처리](/guide/scenarios/payroll) | 30분 |
| 결산 | [연말 결산](/guide/scenarios/year-end) | 2시간 |

## 도움이 필요할 때

- 업무별 단계 가이드는 [업무 시나리오](/guide/scenarios/)를 참고하세요
- 각 메뉴 페이지에서 상세한 사용법을 확인하세요
- 용어가 낯설다면 [용어 사전](/guide/glossary)을 참고하세요
