# 부과

> **메뉴 경로**: 부과

## 개요
부과 메뉴는 검침 데이터와 각종 기준 정보를 바탕으로 세대별 관리비를 산정하고 부과하는 업무를 처리합니다.

| 부과제외세대등록 | 부과 제외 세대 설정 |

## 업무 흐름
1. [기초정보](/impo/basic) 확인 및 설정
2. [검침](/insp/) 데이터 확인
3. [부과처리](/impo/process) > 부과기초작업 실행
4. [부과처리](/impo/process) > 관리비부과처리 실행
5. [관리비부과조회](/impo/inquiry) > 고지서로 검증
6. [부과마감](/impo/closing) 처리
7. [수납](/recp/)으로 연계
