# 아파트뱅크

> **메뉴 경로**: 아파트뱅크

## 개요
아파트뱅크는 은행 거래 내역 조회, 전자증빙 관리, 이체 처리 등 금융 관련 업무를 지원합니다. 회계 시스템과 연동하여 자금 관리를 효율화합니다.

## 하위 메뉴

### 서비스안내
아파트뱅크 서비스 이용 안내 및 가입 정보를 제공합니다.

### [은행거래내역](/bank/transaction)
| 소메뉴 | 설명 |
|--------|------|
| 계좌거래내역 | 등록된 은행 계좌의 거래 내역을 조회합니다 |
| 잔액증명서발급 | 계좌 잔액증명서를 발급합니다 |
| 통장/회계잔액비교 | 통장 잔액과 회계 잔액을 비교합니다 |

### [증빙내역](/bank/evidence)
| 소메뉴 | 설명 |
|--------|------|
| 전자증빙_(세금)계산서 | 전자 세금계산서 및 계산서를 조회합니다 |
| 전자증빙_현금영수증 | 현금영수증 내역을 조회합니다 |
| 전자증빙_카드매입 | 카드 매입 내역을 조회합니다 |
| 전자증빙_이체확인증 | 이체 확인증을 조회합니다 |
| 전자증빙_이자소득원천징수 | 이자소득 원천징수 내역을 조회합니다 |
| 기타증빙내역 | 기타 증빙 자료를 조회합니다 |

### [이체](/bank/transfer)
| 소메뉴 | 설명 |
|--------|------|
| 이체안내(신청) | 이체 서비스 안내 및 신청 정보를 제공합니다 |
| 지출결의서작성 | 이체를 위한 지출결의서를 작성합니다 |
| 이체결과(이체확인증) | 이체 결과 및 이체확인증을 조회합니다 |
| KB3자인증 | KB 3자 인증을 처리합니다 |
| 온라인출금전표작성 | 온라인 출금 전표를 작성합니다 |
| 출금전표결재요청 | 작성된 출금 전표의 결재를 요청합니다 |
| 자금이체실행 | 승인된 이체를 실행합니다 |
| 이체결과리뉴얼 | 이체 결과를 조회합니다 |

### [설정](/bank/settings)
| 소메뉴 | 설명 |

### 대조알림설정
통장 잔액과 회계 잔액 불일치 시 알림 설정을 관리합니다.
