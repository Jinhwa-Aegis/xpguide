# 설정

> **메뉴 경로**: 아파트뱅크 > 설정

## 개요

아파트뱅크 서비스의 수납처리 방법 설정 및 수납처리 실행을 관리합니다. 마리아 입금내역 수납처리 기능을 통해 입금내역을 직접 처리하거나 자동 처리할 수 있으며, 수납처리 결과를 확인하고 미처리 항목을 관리합니다.

## 소메뉴 상세

### 수납처리 - 수납처리 메뉴
![수납처리 화면](/screenshots/BANK2010.png)

자동수납처리에서 처리된 내역의 수납처리 메뉴를 이용하여 수납처리할 수 있습니다.

**수납처리 방법**
1. 수납처리 메뉴를 탭을 이용합니다.
2. 방법일 V 체크를 표시합니다.
3. 수납처리 미처리인 경우에 상에서 조회하여 처리합니다.

**수납처리 탭 구성**
- **수납처리**: 자동 또는 직접 수납처리를 실행합니다.
- **수납결과조회**: 처리된 수납 내역을 조회합니다.
- **자동수납오류**: 자동수납처리 중 오류가 발생한 내역을 확인합니다.

**수납처리 입력 필드**
- **입금일자**: 입금이 발생한 날짜 범위 설정
- **금액주체**: 정상수납 / 지동처리 / 미처리 중 선택하여 필터링
- **정상수납**: 수납이 정상적으로 완료된 내역
- **담당자**: 수납 처리 담당자 정보

**동수별 수납처리**
- 동수로 입금내역에서 이 시스템에서 동수를 선택하고 수납처리를 할 수 있습니다.
- 수납처리를 하기 위해서는 입금금액, 부서명, 별도내용을 차례로 조회합니다.
- 금액으로 동수를 선택하면 입금금액에서 이 시스템에서 수납처리로 조회할 수 있습니다.
- 일반처리로 제공되는 수납처리이므로 이 시스템으로 직접 등록 처리도 됩니다.

### 수납처리 실행
![수납처리 실행 화면](/screenshots/BANK2010.png)

수납처리 방법 V 체크 후 동수 입금금액을 실행하여 수납처리를 진행합니다.

**수납처리 실행 순서**
1. 동의/일괄선택 체크 후 V 입금금액을 클릭합니다.
2. 입금금액, 부서명, 별도내용에 기기를 검색 후 조회합니다.
3. 수납처리 미처리인 경우에 상에서 조회하여 처리합니다.

**수납처리 결과 확인**
- 수납처리 완료 후 수납처리 탭을 다시 조회하면 처리 여부를 확인할 수 있습니다.
- 아파트뱅크 뱅킹서비스에서도 수납처리 결과를 확인할 수 있습니다.

**수납처리 화면 구성**
- 상단: 수납조건 입력 (입금일자, 금액주체, 정상수납 등)
- 중단: 수납내역 목록 (일별처리 탭과 일별내역 탭으로 구분)
- 하단: 아파트뱅크 조회 결과 ([아파트뱅크] 탭에서 확인)

> 수납처리 완료 후 아파트뱅크에서 조회 시 수납처리된 결과를 확인할 수 있습니다. 수납처리가 30일 내 입금 기간 내에 이루어져야 합니다.

### 수납처리 후 결과 확인
![수납처리 후 화면](/screenshots/BANK2020.png)

수납 처리 후 자동수납오류처리에서 처리된 수납 내역을 아파트뱅크에서 조회하여 변경된 계좌에 수납처리가 수가 됩니다.

**수납 처리 후 확인 방법**
1. 수납처리를 아파트뱅크 뱅킹서비스에서 조회 시 수납처리가 됩니다.
2. 아파트뱅크에서 조회 시 수납처리 결과, 계좌에서 변경계좌로 수납처리 수가 됩니다.

**아파트뱅크 조회 결과 컬럼**
- **날짜**: 수납 처리 기준 일자
- **건수**: 처리된 수납 건수
- **금액**: 처리된 수납 금액
- **이름**: 수납 처리된 납부자명
- **내용**: 수납 처리 내용 (관리비, 주차비 등)
- **수납여부**: 수납처리 / 수납처리 완료 / 미처리

**처리 후 오류 발생 시 대응**
- 아파트뱅크에서 수납결과가 오류로 표시되면 수납오류 항목을 확인합니다.
- 수납오류 원인을 파악하여 재처리하거나 수동으로 처리합니다.

### 기능 사용 설정
![기능 사용 설정 화면](/screenshots/BANK2010.png)

아파트뱅크의 각 기능(이체, 증빙 조회, 자동수납처리 등) 사용 여부를 설정합니다.

**설정 항목**
- **자동수납처리 사용 여부**: 입금내역을 자동으로 수납 처리할지 여부를 설정합니다.
- **이체 기능 사용 여부**: 아파트뱅크를 통한 이체 기능 사용 여부를 설정합니다.
- **증빙 조회 기능 사용 여부**: 전자증빙 조회 기능 사용 여부를 설정합니다.
- **대조 알림 사용 여부**: 통장 잔액과 회계 잔액 불일치 시 알림을 받을지 여부를 설정합니다.

> 기능 사용 설정 변경은 관리자 권한이 필요합니다.

---

### 이용현황
![이용현황 화면](/screenshots/BANK2020.png)

아파트뱅크 서비스 이용 현황 및 수납 처리 통계를 조회합니다.

**조회 내용**
- 월별 자동수납처리 건수 및 금액
- 이체 처리 건수 및 금액
- 미처리 수납 내역 현황
- 오류 발생 통계

---

### 대조 알림설정
- 통장 잔액과 회계 잔액 불일치 시 알림을 설정합니다.
- 불일치 허용 범위를 설정하여 소액 차이로 인한 불필요한 알림을 방지할 수 있습니다.

---

### 빠른계좌신청안내
- 빠른계좌 서비스 신청 안내 및 절차를 확인합니다.
- 빠른계좌 서비스 이용 가능 은행 목록과 신청 방법을 안내합니다.
- 빠른계좌 서비스를 사용하면 수납처리가 완료된 후 아파트뱅크에 자동으로 반영됩니다.
