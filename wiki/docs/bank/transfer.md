# 이체

> **메뉴 경로**: 아파트뱅크 > 이체

## 개요

아파트뱅크를 통한 자동수납처리 기능을 제공합니다. 일반 입금내역의 자동수납처리 기능으로 입주민 관리비 수납을 자동화하고, 처리 결과를 확인하며 미처리 내역을 관리합니다. 지출결의서 작성부터 결재 요청, 이체 실행, 결과 확인까지 이체 전 과정을 지원합니다.

## 일반 입금내역 자동수납처리 기능

아파트뱅크 연동을 통해 입금 내역을 자동으로 수납 처리하는 기능입니다.

### 자동수납처리 여부 선택
![이체 화면](/screenshots/BANK1010.png)

아파트뱅크 사용 시 자동으로 자동거래를 선택하는 메뉴입니다.

**설정 방법**
1. 회계 > 기초정보 > 계좌정보관리에서 해당 계좌 탭을 찾습니다.
2. 조회 후 분류목록에서 아파트뱅크 수납을 클릭합니다.
3. 조회 후 분류목록의 자동안내 → 사용반영으로 클릭하면 됩니다.

**사업용으로 자동수납처리 순서**
1. 직접 동수 하여 후 수납처리
2. 직접 동수 등록 동수 파악이 어플리케이션 동수 해체가 안 될 경우 이 동수를 확인 중 수납처리 해야 합니다.

**직접등록으로 자동수납처리 순서**
1. 직접 동수 하여 수납처리
2. 직접 해체 관련 입금금액 처리 자세히 확인 분리 처리

### 수납결과조회 및 정산입금
![이체 화면](/screenshots/BANK1200.png)

자동으로 수납처리의 결과를 조회하고 자동처리의 정산입금을 기간별로 처리됩니다.

**조회 방법**
1. 수납처리를 탭을 통해 확인합니다.
2. 일별내역 / 일별처리 탭 조회합니다.
3. 정상수납여부: 미처리인 경우는 아파트뱅크 뱅킹서버에서 사이어하여 사이어조회 후 기간 내 내역이 조회됩니다.

**수납결과 조회 탭 구성**
- **수납결과조회**: 수납처리가 완료된 내역을 확인합니다.
- **일별처리현황**: 일별로 수납 처리된 금액과 건수를 확인합니다.
- **정산입금**: 처리된 수납금액의 정산 현황을 확인합니다.

**조회 결과 컬럼**
- **수납일자**: 수납이 처리된 날짜
- **입금일자**: 실제 입금이 발생한 날짜
- **납부자**: 수납한 입주민 정보
- **금액**: 수납 금액
- **정상여부**: 정상수납 / 지동처리 / 미처리
- **처리구분**: 자동 또는 직접 처리 구분

> 처리된 이후 기간내 내역이 아파트뱅크에 조회되어 처리 현황을 확인할 수 있습니다. 수납처리 완료 후 아파트뱅크에서 수납처리가 30일 내 입금됩니다.

### 미처리수납 관리
![이체 화면](/screenshots/BANK1030.png)

자동수납처리에서 처리되지 못한 내역의 수납 여부를 파악하여 처리합니다.

**조회 방법**
1. 수납처리탭에서 조회 입력을 탐색합니다.
2. 방법일 V 체크 후 V 입금금액을 클릭합니다.
3. 수납처리 미처리인 경우는 아파트뱅크 뱅킹에서 사이어하여 조회 후 기간 내 내역이 조회됩니다.

**미처리 수납 유형**

- **수납동수 없음**: 입금자 정보가 등록된 세대와 일치하지 않는 경우
  - 처리 방법: 직접 동수를 찾아 수납처리하거나, 수납 무시 처리

- **금액 불일치**: 입금금액이 청구금액과 다른 경우
  - 처리 방법: 분할 수납 처리 또는 차액을 별도 항목으로 처리

- **납부자 미확인**: 입금자 이름이 시스템에 없는 경우
  - 처리 방법: 수납담당자가 확인 후 해당 세대로 수동 수납처리

**주의사항**
- 수납자 정보는 계좌에서 사용되는 입금 내역 처리 건수에 따라 조회가 달라집니다.
- 아파트뱅크 뱅킹서비스에서 조회 시 수납처리가 어렵다면 수납 처리를 내부적으로 관리합니다.

## 소메뉴 상세

### 이체안내(신청)
![이체안내 화면](/screenshots/BANK1010.png)

이체 서비스 안내 및 이용 신청 정보를 제공합니다. 아파트뱅크 이체 서비스의 신청 절차 및 이용 방법을 안내합니다.

---

### 지출결의서작성
![지출결의서작성 화면](/screenshots/BANK1200.png)

이체를 위한 지출결의서를 작성합니다.

**작성 방법**
1. 신규를 클릭하여 지출결의서 작성 화면을 엽니다.
2. 결의일자, 결의번호, 지급처, 지급금액, 지급사유 등 필수 항목을 입력합니다.
3. 출금 계좌와 입금 계좌 정보를 입력합니다.
4. 저장 후 결재 요청 단계로 진행합니다.

**주요 입력 필드**
- **결의일자**: 지출 결의 일자
- **지급처**: 이체 대상 업체 또는 개인
- **지급금액**: 이체할 금액
- **지급사유**: 이체 사유 상세 기술
- **출금계좌**: 이체 출금 계좌
- **입금계좌**: 이체 입금 계좌 (수취인 계좌 정보)

---

### 이체결과(이체확인증)
![이체결과 화면](/screenshots/BANK1030.png)

완료된 이체의 결과 및 이체확인증을 조회합니다.

**조회 방법**
1. 이체 기간, 계좌, 이체 상태 등 검색조건을 입력합니다.
2. 조회 버튼을 클릭하여 이체 결과를 확인합니다.
3. 이체확인증 출력이 필요한 경우 해당 이체 건을 선택하고 출력합니다.

---

### 온라인출금전표작성
![온라인출금전표 화면](/screenshots/BANK1070.png)

온라인 이체를 위한 출금 전표를 작성합니다. 지출결의서와 연계하여 회계 처리를 함께 진행합니다.

**작성 방법**
1. 작성할 지출결의서를 선택합니다.
2. 출금전표 작성 화면에서 계정과목, 금액, 적요 등을 입력합니다.
3. 저장 후 결재 요청 단계로 진행합니다.

---

### 출금전표 결재요청
![출금전표 결재요청 화면](/screenshots/BANK1080.png)

작성된 출금 전표의 결재를 요청합니다.

**결재 요청 방법**
1. 결재 요청할 출금전표를 선택합니다.
2. 결재라인을 확인하고 결재 요청 버튼을 클릭합니다.
3. 결재자에게 결재 요청 알림이 발송됩니다.

---

### 자금이체실행
![자금이체실행 화면](/screenshots/BANK1090.png)

결재 승인된 이체를 실행합니다.

**이체 실행 방법**
1. 결재 승인 완료된 출금전표를 선택합니다.
2. 이체 실행 버튼을 클릭합니다.
3. 공인인증서 인증 후 이체가 처리됩니다.
4. 이체 완료 후 이체결과(이체확인증) 메뉴에서 결과를 확인합니다.

> 이체 실행 전 출금 계좌 잔액을 반드시 확인하세요. 잔액 부족 시 이체가 실패합니다.

---

### 이체결과 리뉴얼
![이체결과 리뉴얼 화면](/screenshots/BANK1100.png)

이체 결과를 최신 화면으로 조회합니다. 자동수납처리에서 처리된 수납 내역을 아파트뱅크에서 조회하여 변경된 계좌에 수납처리가 수가 됩니다.

**조회 결과 컬럼**
- 수납결과 처리된 건수와 금액을 요약하여 표시합니다.
- 아파트뱅크에서 처리된 결과와 XpERP 내 처리 결과를 비교 확인합니다.

## 주의사항
- 이체 전 출금 계좌 잔액을 확인하세요.
- 이체 내역은 회계 전표와 연결하여 처리하세요.
- 지출결의서 결재 완료 후 자금이체를 실행하세요.
- 공인인증서가 필요한 은행의 경우 인증서가 시스템에 등록되어 있어야 합니다.
- 자동수납처리 설정 후 미처리 내역을 정기적으로 확인하여 누락 없이 처리하세요.
- 수납처리 완료 후 아파트뱅크에서 30일 이내 입금 기간을 반드시 확인하세요.
