# 은행거래내역

> **메뉴 경로**: 아파트뱅크 > 은행거래내역

## 개요

등록된 은행 계좌의 거래 내역을 조회하고, 잔액 확인 및 통장·회계 잔액 비교를 처리합니다. 아파트뱅크 서비스에 가입된 계좌의 거래 내역을 실시간으로 확인할 수 있으며, 회계 전표와 비교하여 장부 관리의 정확성을 높일 수 있습니다.

## 아파트뱅크 서비스 안내

아파트뱅크는 XpERP와 연동되는 인터넷뱅킹 서비스로, 은행별로 제공 기능이 다릅니다. 서비스 가입 전에 해당 은행의 지원 기능을 확인하세요.

**주요 지원 은행 및 서비스**

|------|--------|---------|------------|---------|-----------|-----------|
| 농협 | 빠른조회 | O | | O | | |

> 은행별 세부 지원 범위는 변경될 수 있으므로, 서비스 가입 시 아파트뱅크 담당자에게 최신 정보를 확인하세요.

## 계좌 등록 방법

아파트뱅크 서비스를 사용하려면 먼저 계좌를 등록해야 합니다.

**등록 순서**
1. 회계>기초정보>계좌정보관리에서 아파트뱅크 사용할 운영 칸을 이용하여 등록합니다.
2. 아파트뱅크 사업자코드를 사용하여 연결된 아파트뱅크 운영 칸을 등록합니다.
3. 은행에 전자인증공인인증서 발급을 신청합니다.

**신규 계좌 신청 팝업 입력 항목**
- **은행코드**: 해당 은행 선택
- **계좌번호**: 등록할 계좌 번호
- **계좌명**: 계좌의 명칭 (예: 관리비 수납계좌)
- **사업자등록번호**: 아파트 관리단 사업자등록번호
- **전자인증서 여부**: 공인인증서 사용 여부
- **아이디/비밀번호**: 인터넷뱅킹 접속 정보 (해당 은행의 경우)

> 전자인증공인인증서를 사용하는 은행의 경우 공인인증서가 시스템에 등록되어 있어야 합니다. 가기 이미 등록된 아이피 설정이 되었으면 아이피 변경으로 인하여 사용이 안 될 수 있습니다.

## 소메뉴 상세

### 계좌거래내역
![계좌거래내역 화면](/screenshots/BANK0010.png)

계좌별, 기간별 거래 내역을 조회합니다. 수납내역처리와 같은 금액이 사용되어야 하는 거래 내역을 확인하는 데 활용합니다.

**조회 방법**
1. 조회 기간을 입력합니다. 기본값은 당일 기준이며, 범위를 변경하여 조회할 수 있습니다.
2. 제좌를 선택합니다. 등록된 아파트뱅크 계좌 목록에서 조회할 계좌를 선택합니다.
3. 조회 버튼을 클릭합니다.

**조회 결과 컬럼**
- **거래일자**: 거래가 발생한 날짜
- **거래시간**: 거래 시간
- **적요**: 거래 내용 설명
- **출금금액**: 해당 거래의 출금액
- **입금금액**: 해당 거래의 입금액
- **잔액**: 해당 거래 후 계좌 잔액
- **수납처리여부**: 회계 수납 처리 여부 (처리된 경우 표시)

**수납내역처리 기능**
- 수납내역처리입력(적/지코드)을 통해 조회 시 수납 관련 내역을 조회할 수 있습니다.
- 수납 동수 또는 직접 등록하여 수납처리를 할 수 있습니다.
- 금액을 클릭하면 관련 전표를 확인하여 수납처리 이용을 도와줍니다.

> 수납내역처리를 사용하지 않을 경우에는 전에 수납처리가 안 되어있으면 계좌거래에서 수납 처리를 해야 합니다. 수납 동수 또는 수납의 직접 등록하여 처리할 수 있습니다.

### 수납내역 자동처리 기능
![계좌거래내역 화면](/screenshots/BANK0010.png)

아파트뱅크 사용 시 자동으로 자동거래를 선택하는 메뉴입니다.

**자동처리 기능 사용 순서**

- **사업용으로 자동수납처리의 순서**
  1. 직접 동수 하여 후 수납처리
  2. 직접 동수 등록 동수 파악이 어플리케이션 동수 해체가 안 될 경우 이 동수를 확인 중 수납처리 해야 합니다.

- **직접등록으로 자동수납처리의 순서**
  1. 직접 동수 하여 수납처리
  2. 직접 해체 관련 입금금액 처리 자세히 확인 분리 처리

**자동수납처리오류현황 탭**
- 자동처리 중 오류가 발생한 내역을 확인하고, 수납처리할 수 없는 경우 원인을 파악하여 수동 처리합니다.
- 오류 유형: 수납내역 불일치, 입금금액 불일치, 납부자 정보 미확인 등

**수납결과조회 및 정산입금**
- 자동으로 수납처리의 결과를 조회하고 기간별 입금 내역과 정산입금으로 처리됩니다.
- 처리된 이후 기간내 내역이 아파트뱅크에 조회되어 처리 현황을 확인할 수 있습니다.

> 수납처리 완료 후 아파트뱅크에서 수납처리 30일 내 입금 기간내 더 2023.09.01 기간 내 30일 동안 수납이 됩니다.

### 계좌조회로그 / 최근잔액조회
![계좌거래내역 화면](/screenshots/BANK0010.png)

**계좌조회로그**: 계좌거래내역에서 데이터 조회가 되지 않는 경우, 계좌조회로그 탭으로 이동하여 조회 기간을 입력 후 조회하여 내역을 확인합니다. 거래내역이 없거나 은행 시스템 문제, 비밀번호 오입력 등으로 인한 사유가 나타납니다.

**최근잔액조회**: 조회 시 계좌의 최종거래일자 기준으로 잔액이 조회됩니다.
- 상단의 조회기간으로 조회 되지 않습니다.
- 최종거래일에 따라 금액은 변경되므로 일자를 확인하시기 바랍니다.

**최근잔액 조회 결과 컬럼**
- **계좌코드**: 등록된 계좌 코드
- **은행**: 해당 은행명
- **계좌번호**: 계좌 번호
- **최종거래일**: 마지막 거래가 발생한 날짜
- **최종잔액**: 최종거래일 기준 잔액

## 주의사항
- 거래 내역 조회는 아파트뱅크 서비스 가입 계좌에 한합니다.
- 거래내역이 조회되지 않는 경우 계좌조회로그를 확인하여 원인을 파악하세요.
- 수납 동수 또는 직접 등록하여 수납처리 시 이미 처리된 내역과 중복되지 않도록 주의하세요.
- 최근잔액조회는 최종거래일 기준이므로 현재 잔액과 다를 수 있습니다.
- 계좌 등록 시 IP 변경이 있었다면 서비스가 중단될 수 있으므로 아파트뱅크 고객센터에 문의하세요.
